📖 必背：洛必达法则使用条件

当 lim f(x)/g(x) 为 "0/0" 或 "∞/∞" 型未定式，且 f'(x)、g'(x) 存在，g'(x) ≠ 0 时：

lim f(x)/g(x) = lim f'(x)/g'(x)
📝 例题 1
求 y = x³ + 2x² - 3x + 1 在 x = 1 处的切线方程
先求导 y' = 3x² + 4x - 3，代入 x = 1 得斜率 k = 4；再求 y(1) = 1，所以切线为 y - 1 = 4(x - 1)
📝 例题 2
求函数 f(x) = x³ - 3x + 2 的极值
f'(x) = 3x² - 3 = 0 得 x = ±1；f''(x) = 6x；f''(1) = 6 > 0 为极小值 f(1) = 0；f''(-1) = -6 < 0 为极大值 f(-1) = 4
📖 必背：隐函数求导三步法

题目给的是 F(x, y) = 0，没法直接写成 y = f(x) 时，按下面三步走：

  1. 等式两边同时对 x 求导，y 看成 x 的函数 → 出现 dy/dx 或 y'。
  2. 把含 y' 的项留在等号左边，其他项移到右边。
  3. 解出 y'，得到 dy/dx 的显式表达式。
例：x² + y² = 1 两边求导得 2x + 2y·y' = 0 → y' = −x/y。注意：y 出现在分母，不要写成 −x。
📖 必背：反函数求导公式
ddxf−1(y)=1f′(x)=1f′(f−1(y))

小白理解：正函数求导是「顺水」，反函数求导是「逆水」，二者速度互为倒数。记忆口诀：「反函数的导 = 1 ÷ 原函数在对应点的导」。

常见陷阱：题目问的是 f−1(x) 的导数，要把 f−1(x) 的导数表达成 1 / f'(f−1(x))，不能直接当成 1/f'(x)。

📝 例题 3（隐函数）
设 y 由方程 ey + xy = e 确定，求 dy/dx。
两边对 x 求导：ey·y' + y + x·y' = 0 → y'(ey + x) = −y → y' = −y / (ey + x)。
📖 必背：麦克劳林公式（x=0 处的泰勒展开）

如果 f(x) 在 x=0 处 n 阶可导，则：

f(x) = f(0) + f'(0)x + f''(0)x²/2! + f'''(0)x³/3! + ... + f(n)(0)xn/n! + o(xn)

记忆口诀：「n 阶导除以 n!，乘 x 的 n 次方」一直加到 n 阶。

为什么重要：泰勒是洛必达的「降维打击」。当分子分母都是复杂函数时，直接洛必达要展开 3~4 次求导，容易写错；改用泰勒，只需代入「f(n)(0)」即可，结果更稳。

📖 必背：\(\ln(1+x)\) 在 \(x\to0\) 时的泰勒展开（小白分阶版）

🎯 先记最终结论

\[ \ln(1+x)=x-\dfrac{x^{2}}{2}+\dfrac{x^{3}}{3}-\dfrac{x^{4}}{4}+\cdots +(-1)^{n-1}\dfrac{x^{n}}{n}+o(x^{n}),\quad x\to0 \]

记忆口诀：「正负交替，次数往上走，分母等于次数」。第一项正，第二项负，第三项再正。

📚 小白扫盲：为什么有时只写一项，有时必须写三项？

泰勒展开不是背得越长越好，而是保留到足以找到第一个不被抵消的项。题目要几阶，你就展开到几阶：

精度写法什么时候够用
一阶\(\ln(1+x)=x+o(x)\)只求等价无穷小，且一次项不会抵消
二阶\(\ln(1+x)=x-\dfrac{x^{2}}{2}+o(x^{2})\)题中出现 \(x-\ln(1+x)\)，一次项会抵消
三阶\(\ln(1+x)=x-\dfrac{x^{2}}{2}+\dfrac{x^{3}}{3}+o(x^{3})\)二次项也被减掉，或分母含 \(x^{3}\)
\[ x-\ln(1+x)=\dfrac{x^{2}}{2}-\dfrac{x^{3}}{3}+o(x^{3}) \sim\dfrac{x^{2}}{2} \]
\[ \ln(1+x)+\ln(1-x)=\ln(1-x^{2}) =-x^{2}-\dfrac{x^{4}}{2}+o(x^{4})\sim -x^{2} \]

生活例子🌰：把展开式看成一排多米诺骨牌。若前面的 \(x\) 项被题目减掉，就必须继续看下一块 \(x^{2}\)；若 \(x^{2}\) 也被减掉，就再看 \(x^{3}\)。

🧭 遇到题目时的分支导航

小白识别信号走哪条分支停在几阶
单独出现 \(\ln(1+x)\)，一次项不抵消直接用等价无穷小一阶：\(\ln(1+x)\sim x\)
出现 \(x-\ln(1+x)\)用泰勒，先检查一次项抵消至少二阶
分母是 \(x^{m}\)分子各部分统一展开通常展开到 \(x^{m}\)
出现 \(\ln(1+u(x))\)先展开外层，再代入 \(u(x)\)按 \(u\) 的阶数换算
出现 \(\ln(1+x)+\ln(1-x)\)优先合并为 \(\ln(1-x^{2})\)首项就是二阶
反复洛必达会求导三次以上改用泰勒，减少长链求导由分母阶数倒推

一句话决策口诀：「先看极限型，再看谁抵消；分母到几阶，分子跟几阶；能合并先合并，三次求导换泰勒。」

📝 典型题：已知极限求导数（极限、连续、泰勒三合一）
设 \(f(x)\) 在 \(x=0\) 处连续，且
\[ \lim_{x\to0}\dfrac{x f(x)-e^{2\sin x}+1} {\ln(1+x)+\ln(1-x)}=-3. \]
证明 \(f'(0)\) 存在，并求 \(f'(0)\)。

🧭 第一步：看出题人在考什么

出题人不是让你直接对 \(f(x)\) 求导，因为题目根本没给 \(f(x)\) 的表达式。他要考的是：

  • 有限极限搭积木：分母趋于 \(0\)、商趋于有限值，推出分子也趋于 \(0\)。
  • 连续性补函数值：先由极限信息求出 \(f(0)\)。
  • 凑导数定义：把原式拆出 \(\dfrac{f(x)-f(0)}{x}\)。
  • 泰勒消灭已知函数：处理 \(e^{2\sin x}\) 和两个对数。

小白识别信号：看到「\(f\) 连续 + 已知一个含 \(f(x)\) 的极限 + 求 \(f'(0)\)」，立刻想到：先求 \(f(0)\)，再凑导数定义。

📚 第二步：先画路线图，避免走错分支

  1. 化简分母，判断它是几阶无穷小。
  2. 利用「商有有限极限 + 分母趋于 \(0\)」推出分子趋于 \(0\)，求 \(f(0)\)。
  3. 把原极限的分母换成首项 \(-x^{2}\)，确定整个分子需要保留到二阶。
  4. 在分子中加减 \(2x\)，拆出 \(x[f(x)-2]\)，从而凑出导数定义。
  5. 单独算已知函数产生的二阶极限，最后解出 \(f'(0)\)。

🧭 解法一：先求 \(f(0)\)，再凑导数定义

步骤 ①：先处理分母，并记下“阶数账本”。

\[ \ln(1+x)+\ln(1-x)=\ln(1-x^{2})\sim -x^{2}. \]

账本写下：分母是二阶无穷小，所以后面所有已知函数只需展开到 \(x^{2}\)，不能只写一次项，也不必写到四次项。

步骤 ②：先求 \(f(0)\)，这是凑导数定义的钥匙。

记原分子为 \(N(x)\)，原分母为 \(D(x)\)。因为 \(D(x)\sim-x^{2}\)，所以 \(D(x)/x\to0\)。由题设可得：

\[ \lim_{x\to0}\dfrac{N(x)}{x} =\lim_{x\to0}\left[\dfrac{N(x)}{D(x)}\cdot\dfrac{D(x)}{x}\right] =(-3)\cdot0=0. \]

这一步比只说“分子趋于 \(0\)”更严谨，因为接下来需要把分子除以 \(x\)：

\[ f(x)-\dfrac{e^{2\sin x}-1}{x}\to0. \]

又因为 \(e^{2\sin x}-1\sim2\sin x\sim2x\)，并且题设给出 \(f\) 在 \(0\) 处连续，所以

\[ f(0)=\lim_{x\to0}f(x) =\lim_{x\to0}\dfrac{e^{2\sin x}-1}{x}=2. \]

步骤 ③：把原极限改写为分母 \(x^{2}\) 的形式。

\[ 3=\lim_{x\to0}\dfrac{x f(x)-e^{2\sin x}+1}{x^{2}}. \]

注意符号：原极限是 \(-3\)，而原分母等价于 \(-x^{2}\)，两个负号抵消后变成 \(3\)。

步骤 ④：加减 \(2x\)，凑出导数定义。

对 \(x\ne0\)，先做恒等变形，不提前假设导数极限存在：

\[ \dfrac{f(x)-f(0)}{x} =\dfrac{x f(x)-e^{2\sin x}+1}{x^{2}} -\dfrac{2x-e^{2\sin x}+1}{x^{2}}. \]

步骤 ⑤：只计算已知函数的二阶项。

\[ \sin x=x+o(x^{2}),\qquad e^{2\sin x}=1+2x+2x^{2}+o(x^{2}). \]
\[ \lim_{x\to0}\dfrac{2x-e^{2\sin x}+1}{x^{2}} =\lim_{x\to0}\dfrac{-2x^{2}+o(x^{2})}{x^{2}}=-2. \]

上面恒等式右侧的两个极限都存在，因此左侧极限也存在，这就真正证明了 \(f'(0)\) 存在，并且

\[ f'(0)=3-(-2)=5. \]

🧭 解法二：串列公式法（上下同阶原则 + 前后不抵消原则）

这条路为什么更短？不先单独求 \(f(0)\)，而是顺着分母的二阶精度，直接从原式中串出 \(\dfrac{f(x)-2}{x}\)，最后再用连续性确认 \(f(0)=2\)。

原则 ①：上下同阶。分母 \(\ln(1-x^{2})\sim-x^{2}\)，所以分子只保留到二阶。令 \(u=2\sin x\)，则

\[ e^{2\sin x}=1+2\sin x+2\sin^{2}x+o(x^{2}). \]

原则 ②：前后可能抵消时，不要过早只写等价无穷小。先保留 \(x f(x)-2\sin x\) 这个整体，把完全已知的二阶项 \(-2\sin^{2}x\) 拆出去：

\[ \begin{aligned} -3 &=\lim_{x\to0}\dfrac{x f(x)-2\sin x-2\sin^{2}x+o(x^{2})}{-x^{2}}\\ &=\lim_{x\to0}\dfrac{x f(x)-2\sin x}{-x^{2}} +\lim_{x\to0}\dfrac{-2\sin^{2}x}{-x^{2}}+0\\ &=\lim_{x\to0}\dfrac{x f(x)-2\sin x}{-x^{2}}+2. \end{aligned} \]

于是

\[ \lim_{x\to0}\dfrac{x f(x)-2\sin x}{x^{2}}=5. \]

继续“串”出目标。因为目标是导数定义，想要的是 \(f(x)-2\)，所以把 \(2\sin x\) 展开：

\[ 2\sin x=2x-\dfrac{x^{3}}{3}+o(x^{3}). \]
\[ \begin{aligned} 5 &=\lim_{x\to0}\dfrac{x f(x)-2x+\dfrac{x^{3}}{3}+o(x^{3})}{x^{2}}\\ &=\lim_{x\to0}\left[\dfrac{f(x)-2}{x}+\dfrac{x}{3}+o(x)\right]. \end{aligned} \]

后两项趋于 \(0\)，所以

\[ \lim_{x\to0}\dfrac{f(x)-2}{x}=5. \]

该极限有限，故 \(f(x)-2=x\cdot\dfrac{f(x)-2}{x}\to0\)，即 \(\lim_{x\to0}f(x)=2\)。又因 \(f\) 在 \(0\) 处连续，所以 \(f(0)=2\)。因此上式正是导数定义：

\[ f'(0)=\lim_{x\to0}\dfrac{f(x)-f(0)}{x}=5. \]
两种解法怎么选：解法一“先求函数值，再凑导数”更稳，适合小白和规范证明；解法二“按阶展开，一路串出目标”更短，适合已经能熟练控制展开阶数时使用。

🧭 思维导图：每个分叉到底怎么选

  • 先求 \(f(0)\) 还是直接串目标？解法一先求 \(f(0)\) 再凑导数，逻辑最直观；解法二先串出 \(\dfrac{f(x)-2}{x}\)，再用连续性确认 \(f(0)=2\)。小白优先走解法一。
  • 等价无穷小还是泰勒？求 \(f(0)\) 只要一阶，用等价；计算剩余极限要二阶抵消，用泰勒。
  • 洛必达还是泰勒？洛必达会碰到未知的 \(f'(x)\)，而题目正要证明导数存在，不能提前假设；因此必须用代数拆分和泰勒。
  • 展开到几阶？分母是 \(x^{2}\)，所以展开到二阶；这就是“分母定精度”。

⚠️ 小白踩坑预警与防遗漏策略

  • 坑①：只背 \(\ln(1+x)\sim x\)。两个一次项会抵消，必须看到 \(\ln(1+x)+\ln(1-x)\sim-x^{2}\)。
  • 坑②：没求 \(f(0)\) 就写导数定义。先在草稿上写目标模板 \(\dfrac{f(x)-f(0)}{x}\)，空缺的 \(f(0)\) 就是第一阶段任务。
  • 坑③：把原式直接洛必达。这会默认 \(f'(x)\) 已存在，属于循环论证。
  • 坑④：展开阶数不统一。建立“阶数账本”：分母 \(x^{2}\) → 每一项保留到二阶 → 最后统一丢进 \(o(x^{2})\)。
  • 坑⑤：漏负号。单独写一行「原分母 \(\sim-x^{2}\)」，再把 \(-3\times(-1)=3\) 写出来。
  • 坑⑥：长推理跳步。每行只做一个动作，并在右侧标注依据：连续性、等价无穷小、泰勒或导数定义。

🧠 怎样充分吸收这道题：四轮训练法

  1. 识别轮：遮住答案，只说出四个信号——连续、有限极限、求导数、分母二阶。
  2. 路线轮：不计算，只默写「判分母阶数 → 求 \(f(0)\) → 凑导数定义 → 算已知余项」。
  3. 书写轮：完整写一遍，每行标依据；写完用“函数值、阶数、符号、导数模板”四项清单自检。
  4. 变式轮：把 \(2\sin x\) 换成 \(ax\)，或把极限值 \(-3\) 换成 \(L\)，仍按同一路线做，确认自己掌握的是方法而不是答案。

📝 例题 4（泰勒）
求 limx→0 (ex² − 1 − x²) / (x4) 的极限。
把 ex² 展开到 x⁴ 项：ex² = 1 + x² + x⁴/2 + o(x⁴)。代入分子 = (1 + x² + x⁴/2) − 1 − x² = x⁴/2。除以分母 x⁴ 得 1/2。
📝 典型题：[2016 数二、三] 分段函数求导 + 变限积分 + 最值
设函数 \(f(x)=\displaystyle\int_{0}^{1}\left|t^{2}-x^{2}\right|\,dt\)，\((x>0)\)，求 \(f'(x)\)，并求 \(f(x)\) 的最小值。

🧭 第一步：看出题人在考什么（三大考点）

  • 考点 ① 变限积分 + 绝对值：被积函数里带 \(|t^{2}-x^{2}|\)，要按被积变量 \(t\) 把绝对值拆开，先用「\(t^{2}=x^{2}\)」找到分界点。
  • 考点 ② 分段函数求导：积分表达式会随 \(x\) 的大小分成两段，每段分别求导。
  • 考点 ③ 闭区间最值（带积分表达式）：在分段点上比函数值，端点也要比，最后用连续性保证端点处极限相等。

小白识别信号：看到「变限积分 + 绝对值 + 求最值」三个词凑在一起，立刻想到「拆绝对值 → 分类讨论 → 各段求导 → 比较极值和端点」四步走。

⚠️ 难点警示：90% 的小白会在第一步就死

  • 难点 A：分不清分段点是按 \(t\) 还是按 \(x\)。绝对值 \(|t^{2}-x^{2}|\) 内是「\(t\) 和 \(x\) 的混合」，分段要看被积变量 \(t\)，分界点是让绝对值等于 0 的 \(t\) 值，也就是 \(t=x\) 和 \(t=-x\)。
  • 难点 B：忘记「分段点反过来决定自变量 \(x\) 的分类」。在 \([0,1]\) 上有意义的分段点只有 \(t=x\)。但 \(x\) 本身可能比 1 大或比 0 小，所以 \(x\) 的分类要结合积分区间 \([0,1]\) 来定。
  • 难点 C：最值比较时漏掉端点 \(x\to0^{+}\)。\(f(0)\) 要用「极限定义」或「直接代」算，不能因为题目写 \(x>0\) 就丢掉 \(x=0\)。

📚 第二步：小白扫盲：变限积分里的「分段」到底是谁在分？

一句话：分段永远跟着被积变量 \(t\)，不要跟着外部变量 \(x\)。在本题里：

\[ \left|t^{2}-x^{2}\right|= \begin{cases} x^{2}-t^{2}, & 0\le t\le x\\[2pt] t^{2}-x^{2}, & x< t\le 1 \end{cases} \quad(\text{要求 }x\in[0,1]) \]

生活例子🌰：把 \([0,1]\) 想象成一条 1 米长的跑道，\(x\) 是跑道上某个「分界点」。左边 \(0\sim x\) 米这段，\(t^{2}\) 比 \(x^{2}\) 小，所以 \(|t^{2}-x^{2}|=x^{2}-t^{2}\)；右边 \(x\sim 1\) 米反过来。\(x\) 跑到 1 米之外（\(x\ge1\)），整个跑道都在「左边」，就只剩一种情况。

🧭 第三步：方法步骤（小白逐步版）

步骤 ① 找分段点（按 \(t\)）。令 \(|t^{2}-x^{2}|=0\Rightarrow t=\pm x\)，在 \(t\in[0,1]\) 上只剩 \(t=x\)。所以被积变量 \(t\) 在 0、\(x\)、1 这三点分段。

步骤 ② 决定自变量 \(x\) 的分类。分段点 \(t=x\) 落在积分区间里，必须有 \(0\le x\le 1\)。否则：

\(x\) 的范围积分分段方式含义
\(0<x\le 1\)拆成 \([0,x]+[x,1]\) 两段分界点落在跑道内
\(x\ge 1\)只有 \([0,1]\) 一段分界点跑出跑道外

步骤 ③ 分类积分，得到 \(f(x)\) 的显式表达式。

情况 A：\(0<x\le 1\)

\[ \begin{aligned} f(x) &=\int_{0}^{x}(x^{2}-t^{2})\,dt+\int_{x}^{1}(t^{2}-x^{2})\,dt\\ &=\left[x^{2}t-\dfrac{t^{3}}{3}\right]_{0}^{x} +\left[\dfrac{t^{3}}{3}-x^{2}t\right]_{x}^{1}\\[4pt] &=\left(x^{3}-\dfrac{x^{3}}{3}\right) +\left(\dfrac{1}{3}-x^{2}\right)-\left(\dfrac{x^{3}}{3}-x^{3}\right)\\[4pt] &=\dfrac{4}{3}x^{3}-x^{2}+\dfrac{1}{3}. \end{aligned} \]

情况 B：\(x\ge 1\)（整个 \([0,1]\) 上 \(t^{2}\le 1\le x^{2}\)）

\[ f(x)=\int_{0}^{1}(x^{2}-t^{2})\,dt =x^{2}-\dfrac{1}{3}. \]

步骤 ④ 分段求导（这就是分段函数求导的核心！）。

\[ f'(x)= \begin{cases} 4x^{2}-2x=2x(2x-1), & 0<x\le 1\\[4pt] 2x, & x>1 \end{cases} \]

小白易错：千万不要在 \(x=1\) 处强行合并；分段函数的导数在分段点可能不存在（或不连续），本题 \(x=1\) 处左导数为 \(2\)、右导数为 \(2\)，恰好连续，但平时要单独验证。

步骤 ⑤ 求最值（连续函数在 \((0,+\infty)\) 上的最小值）。

在 \((0,1]\) 上，\(f'(x)=0\Rightarrow x=\dfrac{1}{2}\)。与端点 \(x=0^{+}\)（取极限）、\(x=1\) 一起比：

点\(f(x)\)值
\(x=\dfrac{1}{2}\)（极值点）\(\dfrac{4}{3}\cdot\dfrac{1}{8}-\dfrac{1}{4}+\dfrac{1}{3}=\dfrac{1}{6}-\dfrac{1}{4}+\dfrac{1}{3}\)\(\dfrac{1}{4}\)
\(x\to 0^{+}\)（用情况 A 代入 \(x=0\)）\(\dfrac{1}{3}\)\(\dfrac{1}{3}\)
\(x=1\)（两段共用）\(\dfrac{2}{3}\)\(\dfrac{2}{3}\)
\(x>1\)（情况 B 单调增）\(\ge \dfrac{2}{3}\)无新低点

三者比较：\(\dfrac{1}{4}<\dfrac{1}{3}<\dfrac{2}{3}\)，因此最小值为 \(\dfrac{1}{4}\)，在 \(x=\dfrac{1}{2}\) 处取得。

🧭 思维导图：每个分叉怎么选

  • 分段点找谁？找被积变量 \(t\)，不是外部变量 \(x\)。
  • 自变量 \(x\) 怎么分类？把分段点「代」进积分区间，看它是否落在区间里。
  • 用洛必达还是变限积分求导？本题已经把绝对值拆掉、显式积分完了，直接用基本求导公式更快；不需要再洛必达。
  • 最值比较别漏什么？极值点 + 区间端点 + 跨段衔接点 + 极限端点 \(\to 0^{+}\)。

⚠️ 小白踩坑预警与防遗漏清单

  • 坑①：把 \(t\) 和 \(x\) 混在一起讨论。✅ 应对：先写一句「分段看 \(t\)，分类看 \(x\)」贴草稿顶部。
  • 坑②：漏掉 \(x\ge 1\) 的情况。✅ 应对：画一张数轴，标出 0、1，跑道 \([0,1]\) 在图上画出来，分界点超出跑道就要换分类。
  • 坑③：积分算错号。✅ 应对：代入上下限时先在等式右边写 \(\big[F(t)\big]_{a}^{b}=F(b)-F(a)\)，再代数字。
  • 坑④：最值只比极值点、忘了端点。✅ 应对：写「极值点 + 端点 + 跨段衔接点 + 极限端点」四件套清单。
  • 坑⑤：忘记验证 \(f(0)\)。✅ 应对：题目写 \(x>0\) 不代表不能算 \(f(0)\)；用 \(f(0)=\lim_{x\to0^{+}}f(x)\) 就能取。
  • 坑⑥：最后一句话漏写「在 \(x=1/2\) 处取得」。✅ 应对：答案两件套：最小值 + 取得点。

⚠️ 四者关系（必背，每年一考）

  • 驻点 不一定是极值点（水平切线也可能是鞍点）
  • 极值点 不一定是驻点（不可导点也可以是极值点）
  • 拐点和极值点 没有直接关系（一个关心一阶导、一个关心二阶导）
  • 同一个点 可以同时是驻点和拐点（但不常见）

🚗 物理直觉 + 图形化理解：把导数想成「开车」

🚗 一阶导=速度，二阶导=加速度（考研物理+数学双杀）

想象一辆小车在公路上行驶，位置随时间变化记为 \(s(t)\)。那它的导数链就是：

\[ \underbrace{s(t)}_{\text{位置}} \xrightarrow{\;\frac{d}{dt}\;} \underbrace{v(t)=s'(t)}_{\text{速度，一阶导}} \xrightarrow{\;\frac{d}{dt}\;} \underbrace{a(t)=s''(t)}_{\text{加速度，二阶导}} \]
数学物理含义对应图像概念
\(f(x)\) 位置 \(s(t)\) 车在哪 —
\(f'(x)\) 速度 \(v\) 位置变化有多快 \(f'(x)=0\) → 极值点（车瞬间停下、准备掉头）
\(f''(x)\) 加速度 \(a\) 速度变化有多快 \(f''(x)=0\) 且变号 → 拐点（加速↔减速的临界）

🎯 一句话对应表（背下来）：

  • 🚗 极值点 ↔ 速度 = 0（瞬间停下、准备掉头）
  • 🔄 拐点 ↔ 加速度 = 0（加速变减速的临界点）
  • 🏔️ 驻点 ≠ 极值点：速度 = 0 但没掉头（比如匀速冲过山顶 \(y=x^3\) 在 0 点）

生活例子🌰（开车上山再下山）：

  • 山脚→上山：速度从 0 增大，加速度为正（踩油门）
  • 山顶：速度最大、加速度 = 0 → 这就是数学里的 拐点
  • 下山：速度减小，加速度为负（松油门/轻刹车）
  • 如果把"位移-时间"画成图，山顶对应 拐点（不是极值点，因为物理上速度 ≠ 0）

📐 拐点在图上长啥样？（凹凸性判定）

  • \(f''(x) > 0\) → 图像 「开口朝上 ⌣」（上扬、像碗底/笑脸，数学称"凹"/convex）
  • \(f''(x) < 0\) → 图像 「开口朝下 ⌢」（下垂、像山顶/哭脸，数学称"凸"/concave）
  • 二阶导 变号的位置 → 拐点（曲线方向突变）

📊 四个经典图像对照（看完这四个图就彻底懂了）

① \(y=x^2\)：驻点+极值点（标准组合）
\(f'(0)=0\)（水平切线），\(f''(0)=2>0\)（碗底）

② \(y=x^3\)：驻点=拐点，但不是极值
\(f'(0)=0\)，但 \(f''(x)=6x\) 在 0 两侧变号 → 拐点

③ \(y=|x|\)：不可导点也是极值点
左导数=-1 ≠ 右导数=+1 → 尖点 → 但 f(0)<f(x) 仍是极小值

④ \(y=\sin x\)：一段曲线有 3 个拐点
\(f''(x)=-\sin x=0\) → \(x=-\pi,0,\pi\)，每处都凹凸切换

🧭 一图串四概念（终极版：物理 + 数学 + 图像三位一体）

🔗 三位一体速记表
🚗 物理层：位置 → 速度 → 加速度
📐 数学层：\(f(x)\) → \(f'(x)\) → \(f''(x)\)
📊 图像层：高度 → 斜率 → 凹凸

① 极值点：速度=0（车停了下） → 斜率=0
② 拐点：加速度=0（加减速切换） → 凹凸切换
③ 驻点：速度=0 但没掉头 → 斜率=0 但不极值
④ 不可导点：尖点/跳跃 → 斜率不存在

🎯 口诀：一阶为零找驻点，两侧变号得极值；二阶为零看变号，异号才是拐点到

⚠️ 小白踩坑预警

  • 坑①：把「\(f'(x_0)=0\)」直接当成「极值点」→ 错！驻点可能是拐点（如 \(x^3\) 在 0）
  • 坑②：把「\(f''(x_0)=0\)」直接当成「拐点」→ 错！必须两侧变号才是（如 \(x^4\) 在 0，\(f''\ge 0\) 不变号）
  • 坑③：忘了检查不可导点 → \(y=|x|\) 在 0 就是极值点（很多同学漏掉）
  • 坑④：极值点和拐点没有必然联系，别看到一个极值点就急着找拐点
  • 坑⑤：物理类比中，极值点对应"速度=0"（车停下），拐点对应"加速度=0"（山顶速度最大但 ≠ 0），别搞混！

多元微分与极值
偏导与全微分 · 二元无条件极值 · 拉格朗日条件极值
P1 → P2 的桥接
把 x 当常数就得到偏导数；把一元的「f''(x)>0 是极小值」推广成「AC−B²>0 且 A>0 是极小值」。本质思想没变，只是维度变了。
📖 必背：偏导与全微分
概念公式小白理解
对 x 偏导 ∂f/∂x把 y 当常数，只对 x 求导只看 x 这一条方向上 f 怎么变化
对 y 偏导 ∂f/∂y把 x 当常数，只对 y 求导只看 y 这一条方向上 f 怎么变化
全微分 df∂f/∂x · dx + ∂f/∂y · dyx 和 y 同时小变化时，f 的总增量

生活例子：f = 工资 = 底薪 + 时薪 × 工时。底薪、时薪、工时 3 个变量互不影响，∂工资/∂工时 = 时薪，∂工资/∂时薪 = 工时。

📝 例题 5（偏导）
设 f(x, y) = x² + xy + y³，求 ∂f/∂x、∂f/∂y。
∂f/∂x = 2x + y（y 当常数）；∂f/∂y = x + 3y²（x 当常数）。全微分 df = (2x+y)dx + (x+3y²)dy。
📖 必背：二元无条件极值
  1. 求 ∂f/∂x = 0，∂f/∂y = 0，解出所有驻点。
  2. 求二阶偏导：fxx、fyy、fxy。
  3. 计算判别式 AC − B²（A=fxx，C=fyy，B=fxy）：
    • AC − B² > 0 且 A > 0 → 极小值
    • AC − B² > 0 且 A < 0 → 极大值
    • AC − B² < 0 → 不是极值
📝 例题 6（条件极值）
在 x² + y² = 1 的条件下，求 f = xy 的最大值。
用拉格朗日乘数法：L = xy + λ(1 − x² − y²)。∂L/∂x = y − 2λx = 0，∂L/∂y = x − 2λy = 0。解得 x = ±y，代入约束条件得 x = ±√2/2，y = ±√2/2，fmax = 1/2。

📝 真题 1 | 2019 数一/二/三 | 解答题 · 12分
设函数 \(f(x)=\begin{cases}x^{2x}, & x>0 \\ x+2, & x\le 0\end{cases}\)，求 \(f(x)\) 的极值。

🎯 出题人考察点（这是高频考点！）

① 分段函数极值需分段讨论；② 幂指函数求导（对数求导法）；③ 不可导点也可能是极值点

🧭 详细解析（小白逐步版）

**分段函数极值**：① \(x>0\)：\(f(x)=e^{2x\ln x}\)，\(f\prime(x)=x^{2x}(2\ln x+2)\)，令 \(f\prime=0\) 得 \(\ln x=-1\)，即 \(x=1/e\)，\(f(1/e)=e^{-2/e}\)；② \(x=0\)：左右极限 1 和 2；③ \(x<0\)：\(f\prime(x)=1>0\) 单调增。综合得极小值 \(f(1/e)=e^{-2/e}\)。

💯 得分点（阅卷老师按点给分）

① \(x>0\) 求导 (4分)；② 极值点判断 (3分)；③ 不可导点讨论 (3分)；④ 最终答案 (2分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 忘了讨论 \(x=0\) 处；⚠️ 幂指函数直接套公式错；⚠️ 把极值点写成 \(x=1\) 而非 \(1/e\)

📝 真题 2 | 2021 数一 | 解答题 · 12分
设 \(f(x)\) 在 \([0,1]\) 上连续，在 \((0,1)\) 内二阶可导，且 \(\int_0^1 f(x)\,dx=0,\) \(f(0)+f(1)=0\)。证明：存在 \(\xi\in(0,1)\) 使 \(f\prime\prime(\xi)=f(\xi)\)。

🎯 出题人考察点（这是高频考点！）

① 辅助函数构造（核心难点）；② 罗尔定理的使用；③ 复合函数求导

🧭 详细解析（小白逐步版）

**辅助函数**：构造 \(F(x)=f\prime(x)\sin x - f(x)\cos x\)，求导 \(F\prime=f\prime\prime\sin x + f\prime\cos x - f\prime\cos x + f\sin x = (f\prime\prime+f)\sin x\)。验证 \(F(0)=-f(0),F(1)=f\prime(1)\sin 1\)。由 \(f(0)+f(1)=0\) 知 \(F(0)=-f(0)=f(1)\)，结合 \(F(1)=f\prime(1)\sin 1\)，需 \(f\prime(1)\sin 1=-f(0)\)，不一定成立。**更好构造**：\(G(x)=f\prime(x)\sin x + f(x)\cos x\)，\(G\prime=(f\prime\prime+f)\sin x\)。

💯 得分点（阅卷老师按点给分）

① 构造辅助函数 (5分)；② 验证端点函数值相等 (3分)；③ 应用罗尔定理 (3分)；④ 结论 (1分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 辅助函数构造错误是最常见丢分点；⚠️ 没验证端点条件；⚠️ 罗尔定理的三个条件缺一不可
