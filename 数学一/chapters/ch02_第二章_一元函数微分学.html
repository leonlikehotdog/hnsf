🎬 一句话回顾 ch01 + 预告 ch02

ch01 我们学了极限——但只是"地基"，没有真正"挖到宝"。本章 ch02 要把极限变成武器：导数。

导数本质上就是「一个函数的瞬时变化率」。它有一个超俗的名字叫"差商取极限"——但等你看完下面这些故事，你会觉得它其实就是：

车表 / 切线斜率 / 存款利息 / AI 模型学习率 / GPS 转向角度 —— 这些截然不同的东西背后，全部都指向同一个公式。这正是导数的奇妙之处。

⏳ 160 年导数之战 · ch02 主要工具的诞生年份
📖 导数 Derivative
1666 牛顿 v.s. 莱布尼茨：导数的「世纪内战」

🎯它是为了解决什么问题？

导数这玩意儿天生就"内战"。1666 年牛顿被苹果砸了（或者没砸——反正大家爱讲这个故事），为了算"炮弹瞬时速度"发明了 流数术（fluxion）。但他严守秘密，写出来只在朋友圈传。

1676 年德国人 莱布尼茨 也独立发明了导数，而且做了一件牛事 —— 他发明了 dy/dx、∫ 等符号。这些符号实在太好用了，沿用 350 年没变。

两个人互相指责对方抄袭，撕了一辈子。这就是历史上著名的「微积分发明权之争」。今天大家公认是 各自独立发明，但功劳簿上莱布尼茨占了符号的便宜 —— 因为你今天写的就是他的 dy/dx。

"同一座山，从两边爬上来，会面时才发现我们发明了同一件事。" —— 莱布尼茨 1676

📱今天有什么用？导数的核心思想一个字：「变」。汽车变快/变慢是导数，存款变多/变少是导数，AI 模型学得越来越聪明也是导数（梯度下降）。全世界任何一个"变化"的东西，背后都是这个公式：

f'(x) = limΔx→0 [f(x+Δx) − f(x)] ÷ Δx
📖 切线 Tangent Line
1629 费马：「我还没发明微积分，但极值我会做」

🎯它是为了解决什么问题？

法国律师 费马 (Pierre de Fermat)，本职工作是法官（对，是法官！）数学只是副业。他 1637 年思考：

"炮弹飞出去的最高点在哪里？窗户距离地面多少最远？"

他发明了一个"作弊"的办法 —— 把函数取差，然后令差为零，即能找到极值。这其实就是导数 = 0 的雏形，但当时还没有微积分。

🧗怎么理解"取差 = 0"？—— 用"找山底"游戏

想象你在山上找最低点。最低点有个神奇的特征：

「站在山底，往左挪一步 + 往右挪一步 → 高度几乎没变化」

如果挪一下高度变了，说明你站错了，山底在别处。费马的"作弊"就是：用这个性质来反推山底在哪。

📘用 f(x) = x² 实际演示一遍（最简单的"碗形山"）

下面是 x² 这个函数的"高度表"：

x -2 -1 0 1 2
f(x) = x² 4 1 0 1 4

看出来了：x=0 时 f=0，是整座山的最低点。问题是我们假装不知道，用费马的方法推出来。

第 1 步：设山底在某个 a 处。挪一点点（设为 h）去看：

挪一步后：f(a+h) = (a+h)² = a² + 2ah + h²

第 2 步：算"挪一步的高度差"：

f(a+h) − f(a) = 2ah + h²

第 3 步：费马的关键"作弊"——既然这是山底，那么挪一步高度差应该 = 0：

2ah + h² = 0 ⟹ h(2a + h) = 0

第 4 步：h 是"挪的一点点"，不是 0。所以只能是 2a + h = 0。

第 5 步：费马再次"作弊"——h 太小了，扔掉（他当时没"极限"这个概念，但直觉认为"无穷小可忽略"）：

扔掉 h → 2a = 0 → a = 0

✔ 结论：山底在 x=0 处。这跟我们查表看到的答案一模一样！

⚖️和"现代导数"的对照表（两者本质是一回事）：

步骤 费马的"作弊" 现代导数
1 取两个邻近的点 a 和 a+h 同
2 算差 f(a+h) − f(a) 同上
3 差 = 0 [差 ÷ h] = 0（取极限 h→0）
4 h 太小可扔掉 极限 h → 0 是严格定义的
5 2a = 0 → a=0 f'(x) = 2x = 0 → x=0

结论：费马当时已经掌握"差=0 找极值"的核心套路，只是没有"求导"和"极限"这两个工具，所以他只能靠"无穷小可忽略"的朴素直觉去"作弊"。

🚫为什么"差=0"必然能找到极值？—— 用反例 f(x) = x 看

考虑一条直线 f(x) = x，它没有极值。我们试一下费马的方法：

f(a+h) − f(a) = (a+h) − a = h ≠ 0

无论 a 是多少，挪一步的差永远是 h（不为 0），所以"差=0"这件事永远不成立。这说明"差 = 0"这种性质只有极值点才有。一旦发现"差 = 0"，就锁定了极值。

这就像侦探破案：直线说"我这儿没东西"，山底说"我这儿差等于 0"，所以找极值 = 找"差 = 0 的位置"。

📌 一句话总结："差等于 0"不是数学家在猜，而是极值点的定义本身——因为极值是"挪一步也变不了多少"的地方。费马把这件直觉的事第一个写成代数，2 个世纪后才被牛顿/莱布尼茨用具极限的导数严格化。
"古代法官里数学最强的人 — 比许多职业数学家强。" —— 西方数学史评论

📱今天有什么用？你手机地图给的最佳路径，淘宝"你可能喜欢"推荐，背后都在做：在所有可能里面挑最优。这恰好是求导令其为 0做的事。

💡惊不惊喜？费马最出名的不是"费马极值定理"，而是 "费马大定理"（xⁿ + yⁿ = zⁿ 没有正整数解，n ≥ 3）。他在一本书边注上写"我有一个绝妙的证明，但这里空白太小，写不下" —— 结果 357 年才被怀尔斯证明出来，是 1993 年的事。

📖 极值判定
1637 一个判断「山是山顶还是山谷」的统一规则

🎯它是为了解决什么问题？

导数 = 0 找到了"可能的山顶/山谷"。可下一个问题立刻来了：

"这个点到底是真的山顶，还是假山顶？"

比如 y = x³ 在 x = 0 处导数也是 0，但它是"平缓通过"，既不是山顶也不是山谷。这种"假山顶"在物理里对应"亚稳态" —— 看起来平衡，实际一推就倒。

判断方法数学家们有几个：

① 二阶导数：f''(x) > 0 → 山谷，< 0 → 山顶，= 0 → 看不清（要更高阶）

② 一阶导数正负变化：导数从 + 变 - → 山顶；导数从 - 变 + → 山谷

③ 看 f(x) 的图（最直接，但题不让看）

"对山坡的描述用一阶导 — 判断山的身份用二阶导。" —— 江湖人称「二阶判别」

📱今天有什么用？你导航告诉你"再走 50 米到达目的地"，中间一阶导告诉你"方向对不对"。判断"是上坡还是下坡"靠二阶导。

📖 洛必达法则
1696 法国贵族"买"出来的规则

🎯它是为了解决什么问题？

1690 年代的欧洲，0/0、∞/∞ 这些"卡在中间不知道等于几"的式子让数学家很头疼 —— 每个都要手写一长串变形。

法国有一位洛必达侯爵 (Marquis de l'Hôpital, 1661-1704)，家里有钱但自己数学算不动。他直接花了 300 里弗（相当于现在 5 万欧元）一年高薪雇瑞士数学家约翰·伯努利当私教。

伯努利每节课的笔记，侯爵整理出版成书 —— 《分析无穷小》(Analyse des Infiniment Petits, 1696)。书上第一次公开写出"分子分母分别求导再相除"的规则。

约翰·伯努利知道后非常不爽："这是我教给你的，你却署了你的名字！" 但因为合同有规定，伯努利只能忍气吞声。

"花 300 里弗一年的课，包下整个欧洲的微积分教学市场 — 还有比这更划算的吗？" —— 八卦：伯努利 1704 年吐槽

今天，每本微积分课本都把这个规则命名为"洛必达法则" —— 这是数学史上最贵的一笔私人课堂费。

📱今天有什么用？考研卷子上 0/0、∞/∞ 的极限题，洛必达一按就解。但是有个警告：

① 多次求导后越来越复杂 → 改用泰勒展开

② 分子分母同时含 x → 用等价无穷小

③ 0·∞、∞−∞、1^∞ 型 → 先变形再用洛必达

📖 麦克劳林 / 泰勒公式
1715 19 岁就发表论文的神童

🎯它是为了解决什么问题？

1750 年前后，计算器还很慢。天文学家欧拉每天要算上千次 sin、cos、ln。如果能把它们化成"几个 x 的幂相加"的形式，就能用查表代替计算器。

麦克劳林 (Colin Maclaurin, 1698-1746) 19 岁就在皇家学会发表了数学论文，是当时公认的神童。1742 年他写出：

f(x) ≈ f(0) + f'(0)·x + f''(0)·x²/2! + f'''(0)·x³/3! + ...

意思是：任何复杂的函数（只要 n 阶可导）都可以用一个多项式近似。在 x = 0 附近展开就是"麦克劳林公式"；在 x = a 附近展开就是"泰勒公式"。

💡为啥叫「麦克劳林」而不是「泰勒」？其实麦克劳林 1742 才正式写成书，但 1715 年 泰勒 (Brook Taylor) 已经写过类似的展开形式。所以一般说"泰勒展开"是泛指，"麦克劳林展开"特指 x=0 的版本。

"难算的函数 → 多项式 — 这一步是工程学的浪漫。" —— 麦克劳林《流数论》1742

📱今天有什么用？你手机里的计算器就是用泰勒展开工作的：算 sin(0.01)，它就把 x=0 附近的 sin 展开成 x − x³/6，截到 5 阶就够用。同样：

① 游戏画面渲染：用泰勒近似曲面让 GPU 跑得动

② 航天器轨道：把平方根函数在 1 附近展开，省 80% 计算量

③ AI 梯度下降：泰勒展开给出损失函数的"局部地形"图

📖 拉格朗日乘数法
1788 走钢丝时一根绳子的故事

🎯它是为了解决什么问题？

假设你走钢丝 —— 你知道自己的能量最少消耗、姿势最稳定；但你必须始终抓着这条绳。这就是条件极值：在某个限制条件下，找最优。

1788 年拉格朗日在他的《分析力学》第 2 章提出乘数法：

思路：把"约束条件 + λ（拉格朗日乘数）"加到目标函数里，然后照常求导令其 = 0。

在绳子限制下，最优姿势 = 走钢丝的人被绳子牵着 → 引入一个看不见的拉力 λ

λ 物理意义是"约束的反作用力大小"。你想省钱（目标），但必须满足预算（约束）→ 公式正好告诉你"这个约束到底卡得多死"。

"我没画图就解出了绳结的力学。" —— 拉格朗日《分析力学》1788

📱今天有什么用？

① 物流公司：用最少的车次把所有包裹送完 → 乘数法给答案

② 机器学习 SVM：找一个超平面，最大化分类间隔 → 乘数法

③ 你家理财：让你资产最大化的同时承担最小风险 → 乘数法给最优配比

📖 偏导数 / 全微分
1750 达朗贝尔：「函数有 2 个变量？」

🎯它是为了解决什么问题？

牛顿、莱布尼茨搞的导数只能处理 y = f(x) —— 一个 x 控制一个 y。

但现实中大量现象有多个变量同时变化：

• 气温 = 维度 + 高度 + 时间（3 个变量）

• 房子的价值 = 地段 + 面积 + 年代 + 楼层（4 个变量）

• 飞机起飞 = 速度 × 角度 × 风向 × 油量（≥4 个）

1750 年 达朗贝尔 (D'Alembert) 给出答案：把其他变量按住不动，只看一个变量的变化。这就是偏导数 ∂f/∂x。

后来 欧拉 在 1755 年给"按住一个、看另一个"的偏导数写下了通用的数学形式 ∂f/∂x。

"不能动一个就动全部 — 必须一个一个查。" —— 达朗贝尔《动力学》1758

📱今天有什么用？偏导数就是 AI 学到的全部技能。你让 AI 翻译，它的"参数"有几亿个 ——每个参数就是一个偏导的对象。AI 学习就是调整每个参数，让损失函数（一个超多元函数）的偏导数都接近 0。

📖 隐函数 / 反函数求导
1820s 不用解出 y 也能求导

🎯它是为了解决什么问题？

很多方程解不出 y。比如 \(x^2 + y^2 = 1\) 的解是 \(y = \pm \sqrt{1-x^2}\)，但如果遇到 \(xy^3 + \sin(y) = e^x\)，你一辈子也解不出 y 来。

可我还是想知道"在某个 x 时 y 的瞬时变化率"——怎么办？

解法：不用解出 y，直接对 x 求导。当 y 出现时，把它当 x 的函数处理（链式法则），最后解 y' 即可。

"有的函数你一生都写不出 y = f(x)，但你可以用 ∂/∂x 求它的 '性格'。" —— 现代微积分名言

📱今天有什么用？

① GPS：告诉你 "先直行 200 米再左转" —— 隐函数求导决定转角

② 心理学曲线：考试焦虑和成绩的隐函数，无法直接解，但能算"再焦虑 1 分会下降多少成绩"

③ 反函数求导：函数的反函数导数 = 1/原函数导数，所以你"从 A 走到 B"和"从 B 走回 A"的速度正好互为倒数

📖 必背：洛必达法则使用条件

当 lim f(x)/g(x) 为 "0/0" 或 "∞/∞" 型未定式，且 f'(x)、g'(x) 存在，g'(x) ≠ 0 时：

lim f(x)/g(x) = lim f'(x)/g'(x)
📝 例题 1
求 y = x³ + 2x² - 3x + 1 在 x = 1 处的切线方程
先求导 y' = 3x² + 4x - 3，代入 x = 1 得斜率 k = 4；再求 y(1) = 1，所以切线为 y - 1 = 4(x - 1)
📝 例题 2
求函数 f(x) = x³ - 3x + 2 的极值
f'(x) = 3x² - 3 = 0 得 x = ±1；f''(x) = 6x；f''(1) = 6 > 0 为极小值 f(1) = 0；f''(-1) = -6 < 0 为极大值 f(-1) = 4
📖 必背：隐函数求导三步法

题目给的是 F(x, y) = 0，没法直接写成 y = f(x) 时，按下面三步走：

  1. 等式两边同时对 x 求导，y 看成 x 的函数 → 出现 dy/dx 或 y'。
  2. 把含 y' 的项留在等号左边，其他项移到右边。
  3. 解出 y'，得到 dy/dx 的显式表达式。
例：x² + y² = 1 两边求导得 2x + 2y·y' = 0 → y' = −x/y。注意：y 出现在分母，不要写成 −x。
📖 必背：反函数求导公式
ddxf−1(y)=1f′(x)=1f′(f−1(y))

小白理解：正函数求导是「顺水」，反函数求导是「逆水」，二者速度互为倒数。记忆口诀：「反函数的导 = 1 ÷ 原函数在对应点的导」。

常见陷阱：题目问的是 f−1(x) 的导数，要把 f−1(x) 的导数表达成 1 / f'(f−1(x))，不能直接当成 1/f'(x)。

📝 例题 3（隐函数）
设 y 由方程 ey + xy = e 确定，求 dy/dx。
两边对 x 求导：ey·y' + y + x·y' = 0 → y'(ey + x) = −y → y' = −y / (ey + x)。
📖 必背：麦克劳林公式（x=0 处的泰勒展开）

如果 f(x) 在 x=0 处 n 阶可导，则：

f(x) = f(0) + f'(0)x + f''(0)x²/2! + f'''(0)x³/3! + ... + f(n)(0)xn/n! + o(xn)

记忆口诀：「n 阶导除以 n!，乘 x 的 n 次方」一直加到 n 阶。

为什么重要：泰勒是洛必达的「降维打击」。当分子分母都是复杂函数时，直接洛必达要展开 3~4 次求导，容易写错；改用泰勒，只需代入「f(n)(0)」即可，结果更稳。

📖 必背：\(\ln(1+x)\) 在 \(x\to0\) 时的泰勒展开（小白分阶版）

🎯 先记最终结论

\[ \ln(1+x)=x-\dfrac{x^{2}}{2}+\dfrac{x^{3}}{3}-\dfrac{x^{4}}{4}+\cdots +(-1)^{n-1}\dfrac{x^{n}}{n}+o(x^{n}),\quad x\to0 \]

记忆口诀：「正负交替，次数往上走，分母等于次数」。第一项正，第二项负，第三项再正。

📚 小白扫盲：为什么有时只写一项，有时必须写三项？

泰勒展开不是背得越长越好，而是保留到足以找到第一个不被抵消的项。题目要几阶，你就展开到几阶：

精度写法什么时候够用
一阶\(\ln(1+x)=x+o(x)\)只求等价无穷小，且一次项不会抵消
二阶\(\ln(1+x)=x-\dfrac{x^{2}}{2}+o(x^{2})\)题中出现 \(x-\ln(1+x)\)，一次项会抵消
三阶\(\ln(1+x)=x-\dfrac{x^{2}}{2}+\dfrac{x^{3}}{3}+o(x^{3})\)二次项也被减掉，或分母含 \(x^{3}\)
\[ x-\ln(1+x)=\dfrac{x^{2}}{2}-\dfrac{x^{3}}{3}+o(x^{3}) \sim\dfrac{x^{2}}{2} \]
\[ \ln(1+x)+\ln(1-x)=\ln(1-x^{2}) =-x^{2}-\dfrac{x^{4}}{2}+o(x^{4})\sim -x^{2} \]

生活例子🌰：把展开式看成一排多米诺骨牌。若前面的 \(x\) 项被题目减掉，就必须继续看下一块 \(x^{2}\)；若 \(x^{2}\) 也被减掉，就再看 \(x^{3}\)。

🧭 遇到题目时的分支导航

小白识别信号走哪条分支停在几阶
单独出现 \(\ln(1+x)\)，一次项不抵消直接用等价无穷小一阶：\(\ln(1+x)\sim x\)
出现 \(x-\ln(1+x)\)用泰勒，先检查一次项抵消至少二阶
分母是 \(x^{m}\)分子各部分统一展开通常展开到 \(x^{m}\)
出现 \(\ln(1+u(x))\)先展开外层，再代入 \(u(x)\)按 \(u\) 的阶数换算
出现 \(\ln(1+x)+\ln(1-x)\)优先合并为 \(\ln(1-x^{2})\)首项就是二阶
反复洛必达会求导三次以上改用泰勒，减少长链求导由分母阶数倒推

一句话决策口诀：「先看极限型，再看谁抵消；分母到几阶，分子跟几阶；能合并先合并，三次求导换泰勒。」

📝 典型题：已知极限求导数（极限、连续、泰勒三合一）
设 \(f(x)\) 在 \(x=0\) 处连续，且
\[ \lim_{x\to0}\dfrac{x f(x)-e^{2\sin x}+1} {\ln(1+x)+\ln(1-x)}=-3. \]
证明 \(f'(0)\) 存在，并求 \(f'(0)\)。

🧭 第一步：看出题人在考什么

出题人不是让你直接对 \(f(x)\) 求导，因为题目根本没给 \(f(x)\) 的表达式。他要考的是：

  • 有限极限搭积木：分母趋于 \(0\)、商趋于有限值，推出分子也趋于 \(0\)。
  • 连续性补函数值：先由极限信息求出 \(f(0)\)。
  • 凑导数定义：把原式拆出 \(\dfrac{f(x)-f(0)}{x}\)。
  • 泰勒消灭已知函数：处理 \(e^{2\sin x}\) 和两个对数。

小白识别信号：看到「\(f\) 连续 + 已知一个含 \(f(x)\) 的极限 + 求 \(f'(0)\)」，立刻想到：先求 \(f(0)\)，再凑导数定义。

📚 第二步：先画路线图，避免走错分支

  1. 化简分母，判断它是几阶无穷小。
  2. 利用「商有有限极限 + 分母趋于 \(0\)」推出分子趋于 \(0\)，求 \(f(0)\)。
  3. 把原极限的分母换成首项 \(-x^{2}\)，确定整个分子需要保留到二阶。
  4. 在分子中加减 \(2x\)，拆出 \(x[f(x)-2]\)，从而凑出导数定义。
  5. 单独算已知函数产生的二阶极限，最后解出 \(f'(0)\)。

🧭 解法一：先求 \(f(0)\)，再凑导数定义

步骤 ①：先处理分母，并记下“阶数账本”。

\[ \ln(1+x)+\ln(1-x)=\ln(1-x^{2})\sim -x^{2}. \]

账本写下：分母是二阶无穷小，所以后面所有已知函数只需展开到 \(x^{2}\)，不能只写一次项，也不必写到四次项。

步骤 ②：先求 \(f(0)\)，这是凑导数定义的钥匙。

记原分子为 \(N(x)\)，原分母为 \(D(x)\)。因为 \(D(x)\sim-x^{2}\)，所以 \(D(x)/x\to0\)。由题设可得：

\[ \lim_{x\to0}\dfrac{N(x)}{x} =\lim_{x\to0}\left[\dfrac{N(x)}{D(x)}\cdot\dfrac{D(x)}{x}\right] =(-3)\cdot0=0. \]

这一步比只说“分子趋于 \(0\)”更严谨，因为接下来需要把分子除以 \(x\)：

\[ f(x)-\dfrac{e^{2\sin x}-1}{x}\to0. \]

又因为 \(e^{2\sin x}-1\sim2\sin x\sim2x\)，并且题设给出 \(f\) 在 \(0\) 处连续，所以

\[ f(0)=\lim_{x\to0}f(x) =\lim_{x\to0}\dfrac{e^{2\sin x}-1}{x}=2. \]

步骤 ③：把原极限改写为分母 \(x^{2}\) 的形式。

\[ 3=\lim_{x\to0}\dfrac{x f(x)-e^{2\sin x}+1}{x^{2}}. \]

注意符号：原极限是 \(-3\)，而原分母等价于 \(-x^{2}\)，两个负号抵消后变成 \(3\)。

步骤 ④：加减 \(2x\)，凑出导数定义。

对 \(x\ne0\)，先做恒等变形，不提前假设导数极限存在：

\[ \dfrac{f(x)-f(0)}{x} =\dfrac{x f(x)-e^{2\sin x}+1}{x^{2}} -\dfrac{2x-e^{2\sin x}+1}{x^{2}}. \]

步骤 ⑤：只计算已知函数的二阶项。

\[ \sin x=x+o(x^{2}),\qquad e^{2\sin x}=1+2x+2x^{2}+o(x^{2}). \]
\[ \lim_{x\to0}\dfrac{2x-e^{2\sin x}+1}{x^{2}} =\lim_{x\to0}\dfrac{-2x^{2}+o(x^{2})}{x^{2}}=-2. \]

上面恒等式右侧的两个极限都存在，因此左侧极限也存在，这就真正证明了 \(f'(0)\) 存在，并且

\[ f'(0)=3-(-2)=5. \]

🧭 解法二：串列公式法（上下同阶原则 + 前后不抵消原则）

这条路为什么更短？不先单独求 \(f(0)\)，而是顺着分母的二阶精度，直接从原式中串出 \(\dfrac{f(x)-2}{x}\)，最后再用连续性确认 \(f(0)=2\)。

原则 ①：上下同阶。分母 \(\ln(1-x^{2})\sim-x^{2}\)，所以分子只保留到二阶。令 \(u=2\sin x\)，则

\[ e^{2\sin x}=1+2\sin x+2\sin^{2}x+o(x^{2}). \]

原则 ②：前后可能抵消时，不要过早只写等价无穷小。先保留 \(x f(x)-2\sin x\) 这个整体，把完全已知的二阶项 \(-2\sin^{2}x\) 拆出去：

\[ \begin{aligned} -3 &=\lim_{x\to0}\dfrac{x f(x)-2\sin x-2\sin^{2}x+o(x^{2})}{-x^{2}}\\ &=\lim_{x\to0}\dfrac{x f(x)-2\sin x}{-x^{2}} +\lim_{x\to0}\dfrac{-2\sin^{2}x}{-x^{2}}+0\\ &=\lim_{x\to0}\dfrac{x f(x)-2\sin x}{-x^{2}}+2. \end{aligned} \]

于是

\[ \lim_{x\to0}\dfrac{x f(x)-2\sin x}{x^{2}}=5. \]

继续“串”出目标。因为目标是导数定义，想要的是 \(f(x)-2\)，所以把 \(2\sin x\) 展开：

\[ 2\sin x=2x-\dfrac{x^{3}}{3}+o(x^{3}). \]
\[ \begin{aligned} 5 &=\lim_{x\to0}\dfrac{x f(x)-2x+\dfrac{x^{3}}{3}+o(x^{3})}{x^{2}}\\ &=\lim_{x\to0}\left[\dfrac{f(x)-2}{x}+\dfrac{x}{3}+o(x)\right]. \end{aligned} \]

后两项趋于 \(0\)，所以

\[ \lim_{x\to0}\dfrac{f(x)-2}{x}=5. \]

该极限有限，故 \(f(x)-2=x\cdot\dfrac{f(x)-2}{x}\to0\)，即 \(\lim_{x\to0}f(x)=2\)。又因 \(f\) 在 \(0\) 处连续，所以 \(f(0)=2\)。因此上式正是导数定义：

\[ f'(0)=\lim_{x\to0}\dfrac{f(x)-f(0)}{x}=5. \]
两种解法怎么选：解法一“先求函数值，再凑导数”更稳，适合小白和规范证明；解法二“按阶展开，一路串出目标”更短，适合已经能熟练控制展开阶数时使用。

🧭 思维导图：每个分叉到底怎么选

  • 先求 \(f(0)\) 还是直接串目标？解法一先求 \(f(0)\) 再凑导数，逻辑最直观；解法二先串出 \(\dfrac{f(x)-2}{x}\)，再用连续性确认 \(f(0)=2\)。小白优先走解法一。
  • 等价无穷小还是泰勒？求 \(f(0)\) 只要一阶，用等价；计算剩余极限要二阶抵消，用泰勒。
  • 洛必达还是泰勒？洛必达会碰到未知的 \(f'(x)\)，而题目正要证明导数存在，不能提前假设；因此必须用代数拆分和泰勒。
  • 展开到几阶？分母是 \(x^{2}\)，所以展开到二阶；这就是“分母定精度”。

⚠️ 小白踩坑预警与防遗漏策略

📝 典型题：[2016 数二、三] 分段函数求导 + 变限积分 + 最值
设函数 \(f(x)=\displaystyle\int_{0}^{1}\left|t^{2}-x^{2}\right|\,dt\)，\((x>0)\)，求 \(f'(x)\)，并求 \(f(x)\) 的最小值。

🧭 第一步：看出题人在考什么（三大考点）

  • 考点 ① 变限积分 + 绝对值：被积函数里带 \(|t^{2}-x^{2}|\)，要按被积变量 \(t\) 把绝对值拆开，先用「\(t^{2}=x^{2}\)」找到分界点。
  • 考点 ② 分段函数求导：积分表达式会随 \(x\) 的大小分成两段，每段分别求导。
  • 考点 ③ 闭区间最值（带积分表达式）：在分段点上比函数值，端点也要比，最后用连续性保证端点处极限相等。

小白识别信号：看到「变限积分 + 绝对值 + 求最值」三个词凑在一起，立刻想到「拆绝对值 → 分类讨论 → 各段求导 → 比较极值和端点」四步走。

⚠️ 难点警示：90% 的小白会在第一步就死

  • 难点 A：分不清分段点是按 \(t\) 还是按 \(x\)。绝对值 \(|t^{2}-x^{2}|\) 内是「\(t\) 和 \(x\) 的混合」，分段要看被积变量 \(t\)，分界点是让绝对值等于 0 的 \(t\) 值，也就是 \(t=x\) 和 \(t=-x\)。
  • 难点 B：忘记「分段点反过来决定自变量 \(x\) 的分类」。在 \([0,1]\) 上有意义的分段点只有 \(t=x\)。但 \(x\) 本身可能比 1 大或比 0 小，所以 \(x\) 的分类要结合积分区间 \([0,1]\) 来定。
  • 难点 C：最值比较时漏掉端点 \(x\to0^{+}\)。\(f(0)\) 要用「极限定义」或「直接代」算，不能因为题目写 \(x>0\) 就丢掉 \(x=0\)。

📚 第二步：小白扫盲：变限积分里的「分段」到底是谁在分？

一句话：分段永远跟着被积变量 \(t\)，不要跟着外部变量 \(x\)。在本题里：

\[ \left|t^{2}-x^{2}\right|= \begin{cases} x^{2}-t^{2}, & 0\le t\le x\\[2pt] t^{2}-x^{2}, & x< t\le 1 \end{cases} \quad(\text{要求 }x\in[0,1]) \]

生活例子🌰：把 \([0,1]\) 想象成一条 1 米长的跑道，\(x\) 是跑道上某个「分界点」。左边 \(0\sim x\) 米这段，\(t^{2}\) 比 \(x^{2}\) 小，所以 \(|t^{2}-x^{2}|=x^{2}-t^{2}\)；右边 \(x\sim 1\) 米反过来。\(x\) 跑到 1 米之外（\(x\ge1\)），整个跑道都在「左边」，就只剩一种情况。

🧭 第三步：方法步骤（小白逐步版）

步骤 ① 找分段点（按 \(t\)）。令 \(|t^{2}-x^{2}|=0\Rightarrow t=\pm x\)，在 \(t\in[0,1]\) 上只剩 \(t=x\)。所以被积变量 \(t\) 在 0、\(x\)、1 这三点分段。

步骤 ② 决定自变量 \(x\) 的分类。分段点 \(t=x\) 落在积分区间里，必须有 \(0\le x\le 1\)。否则：

\(x\) 的范围积分分段方式含义
\(0<x\le 1\)拆成 \([0,x]+[x,1]\) 两段分界点落在跑道内
\(x\ge 1\)只有 \([0,1]\) 一段分界点跑出跑道外

步骤 ③ 分类积分，得到 \(f(x)\) 的显式表达式。

情况 A：\(0<x\le 1\)

\[ \begin{aligned} f(x) &=\int_{0}^{x}(x^{2}-t^{2})\,dt+\int_{x}^{1}(t^{2}-x^{2})\,dt\\ &=\left[x^{2}t-\dfrac{t^{3}}{3}\right]_{0}^{x} +\left[\dfrac{t^{3}}{3}-x^{2}t\right]_{x}^{1}\\[4pt] &=\left(x^{3}-\dfrac{x^{3}}{3}\right) +\left(\dfrac{1}{3}-x^{2}\right)-\left(\dfrac{x^{3}}{3}-x^{3}\right)\\[4pt] &=\dfrac{4}{3}x^{3}-x^{2}+\dfrac{1}{3}. \end{aligned} \]

情况 B：\(x\ge 1\)（整个 \([0,1]\) 上 \(t^{2}\le 1\le x^{2}\)）

\[ f(x)=\int_{0}^{1}(x^{2}-t^{2})\,dt =x^{2}-\dfrac{1}{3}. \]

步骤 ④ 分段求导（这就是分段函数求导的核心！）。

\[ f'(x)= \begin{cases} 4x^{2}-2x=2x(2x-1), & 0<x\le 1\\[4pt] 2x, & x>1 \end{cases} \]

小白易错：千万不要在 \(x=1\) 处强行合并；分段函数的导数在分段点可能不存在（或不连续），本题 \(x=1\) 处左导数为 \(2\)、右导数为 \(2\)，恰好连续，但平时要单独验证。

步骤 ⑤ 求最值（连续函数在 \((0,+\infty)\) 上的最小值）。

在 \((0,1]\) 上，\(f'(x)=0\Rightarrow x=\dfrac{1}{2}\)。与端点 \(x=0^{+}\)（取极限）、\(x=1\) 一起比：

点\(f(x)\)值
\(x=\dfrac{1}{2}\)（极值点）\(\dfrac{4}{3}\cdot\dfrac{1}{8}-\dfrac{1}{4}+\dfrac{1}{3}=\dfrac{1}{6}-\dfrac{1}{4}+\dfrac{1}{3}\)\(\dfrac{1}{4}\)
\(x\to 0^{+}\)（用情况 A 代入 \(x=0\)）\(\dfrac{1}{3}\)\(\dfrac{1}{3}\)
\(x=1\)（两段共用）\(\dfrac{2}{3}\)\(\dfrac{2}{3}\)
\(x>1\)（情况 B 单调增）\(\ge \dfrac{2}{3}\)无新低点

三者比较：\(\dfrac{1}{4}<\dfrac{1}{3}<\dfrac{2}{3}\)，因此最小值为 \(\dfrac{1}{4}\)，在 \(x=\dfrac{1}{2}\) 处取得。

🧭 思维导图：每个分叉怎么选

  • 分段点找谁？找被积变量 \(t\)，不是外部变量 \(x\)。
  • 自变量 \(x\) 怎么分类？把分段点「代」进积分区间，看它是否落在区间里。
  • 用洛必达还是变限积分求导？本题已经把绝对值拆掉、显式积分完了，直接用基本求导公式更快；不需要再洛必达。
  • 最值比较别漏什么？极值点 + 区间端点 + 跨段衔接点 + 极限端点 \(\to 0^{+}\)。

🚗 一阶导=速度，二阶导=加速度（考研物理+数学双杀）

想象一辆小车在公路上行驶，位置随时间变化记为 \(s(t)\)。那它的导数链就是：

\[ \underbrace{s(t)}_{\text{位置}} \xrightarrow{\;\frac{d}{dt}\;} \underbrace{v(t)=s'(t)}_{\text{速度，一阶导}} \xrightarrow{\;\frac{d}{dt}\;} \underbrace{a(t)=s''(t)}_{\text{加速度，二阶导}} \]
数学物理含义对应图像概念
\(f(x)\) 位置 \(s(t)\) 车在哪 —
\(f'(x)\) 速度 \(v\) 位置变化有多快 \(f'(x)=0\) → 极值点（车瞬间停下、准备掉头）
\(f''(x)\) 加速度 \(a\) 速度变化有多快 \(f''(x)=0\) 且变号 → 拐点（加速↔减速的临界）

🎯 一句话对应表（背下来）：

  • 🚗 极值点 ↔ 速度 = 0（瞬间停下、准备掉头）
  • 🔄 拐点 ↔ 加速度 = 0（加速变减速的临界点）
  • 🏔️ 驻点 ≠ 极值点：速度 = 0 但没掉头（比如匀速冲过山顶 \(y=x^3\) 在 0 点）

生活例子🌰（开车上山再下山）：

  • 山脚→上山：速度从 0 增大，加速度为正（踩油门）
  • 山顶：速度最大、加速度 = 0 → 这就是数学里的 拐点
  • 下山：速度减小，加速度为负（松油门/轻刹车）
  • 如果把"位移-时间"画成图，山顶对应 拐点（不是极值点，因为物理上速度 ≠ 0）

📐 拐点在图上长啥样？（凹凸性判定）

  • 二阶导 变号的位置 → 拐点（曲线方向突变）

🎯 核心考点：抽象复合函数求高阶偏导

📝 例题 7 | 2019 数三 | 抽象复合函数求二阶偏导
设 f(u,v) 具有二阶连续偏导数，函数 g(x,y) = xy − f(x+y, x−y)，求 ∂²g/∂x²、∂²g/∂x∂y、∂²g/∂y²。

为什么要做这题？考研真题里"抽象复合函数 + 二阶偏导"几乎是每年都考的门票题——做不出它，多元微分这一章就白学了。做完它再去搞条件极值、拉格朗日乘数法，会顺得多。

🎯 这道题要考什么？

① 抽象复合函数链式法则（外层 g 是 (x,y)，里层 f 是 (u,v)，其中 u = x+y, v = x−y）；
② 求高阶偏导（对已经求出的一阶偏导再求偏导）；
③ 混合偏导相等定理（f 二阶连续偏导 ⇒ f"12 = f"21，用来化简结果）。

🧭 第一步：先认清楚"自变量 vs 中间变量"

外层 g(x, y)，里层 f(u, v)。其中 u = x+y, v = x−y。

为了书写方便，咱们记：f'1 = ∂f/∂u（对 f 的第一个变量求偏导），f'2 = ∂f/∂v（对第二个变量求偏导）。注意：f'1、f'2 仍然是 (u, v) 的函数，再求偏导时还得用链式法则。

🧭 第二步：求一阶偏导（链式法则登场）

对 x 求偏导：∂u/∂x = 1, ∂v/∂x = 1，所以：

∂g/∂x = y − [f'1·1 + f'2·1] = y − f'1 − f'2

对 y 求偏导：∂u/∂y = 1, ∂v/∂y = −1（v = x−y 对 y 求偏导为 −1，这是最常错的符号），所以：

∂g/∂y = x − [f'1·1 + f'2·(−1)] = x − f'1 + f'2

🧭 第三步：求二阶偏导（对一阶偏导再求偏导）

⚠️ 关键提醒：f'1、f'2 仍然是 (u, v) = (x+y, x−y) 的函数，不是常数！再求偏导时必须再用一次链式法则。

(1) 求 ∂²g/∂x²：对 (y − f'1 − f'2) 再对 x 求偏导。

∂(f'1)/∂x = f"11·1 + f"12·1 = f"11 + f"12

∂(f'2)/∂x = f"21·1 + f"22·1 = f"21 + f"22

所以：

∂²g/∂x² = −(f"11 + f"12 + f"21 + f"22) = −f"11 − 2f"12 − f"22

最后一步用了混合偏导相等定理：因为 f 二阶偏导连续，f"12 = f"21，所以两个交叉项合成 2f"12。

(2) 求 ∂²g/∂x∂y：对 (y − f'1 − f'2) 先对 y 求偏导。

∂(f'1)/∂y = f"11·1 + f"12·(−1) = f"11 − f"12

∂(f'2)/∂y = f"21·1 + f"22·(−1) = f"21 − f"22

所以：

∂²g/∂x∂y = 1 − (f"11 − f"12) − (f"21 − f"22) = 1 − f"11 + f"22

中间两项 f"12 和 −f"21 正好抵消（因为它们相等）。

(3) 求 ∂²g/∂y²：对 (x − f'1 + f'2) 再对 y 求偏导。

∂(f'1)/∂y = f"11 − f"12（和上面一样）

∂(f'2)/∂y = f"21 − f"22（和上面一样）

所以：

∂²g/∂y² = −(f"11 − f"12) + (f"21 − f"22) = −f"11 + 2f"12 − f"22
🧠 小白记忆类比（1 个理解 + 1 个记忆）：
① 理解类比：把 f(u, v) 想象成一家咖啡店——u 是拿铁销量，v 是摩卡销量。f'1 = "拿铁涨 1 杯带来的利润变化"，f'2 = "摩卡涨 1 杯带来的利润变化"。f"12 = "摩卡多卖 1 杯时，拿铁多卖 1 杯带来的额外利润变化"；f"21 = "拿铁多卖 1 杯时，摩卡多卖 1 杯带来的额外利润变化"。这两个物理意义完全相同（咖啡店就是一个对称的二维利润场），所以它们相等！这正是「混合偏导相等定理」的生活版本。
② 记忆类比：记住 "链子套链子"——求一阶偏导时套一层链式法则（把 x, y 的偏导传到 f），求二阶偏导时再套一层（把 x, y 的偏导传到 f'1, f'2）。一共要套 2 层「链子」，缺一层就漏一半结果。

🎯 出题人视角：这道题是"分层链式法则 + 混合偏导相等定理"的缝合题。他想筛三种能力：① 能不能正确分辨"自变量 vs 中间变量"（不辨清就乱套）；② 能不能在求二阶偏导时再次想到链式法则（最常漏）；③ 记不记得混合偏导相等定理可以化简（少了就白丢 1~2 分）。三大能力缺一不可，整道题就是一个"三关卡"。
📖 必背：抽象复合函数求偏导 · 三步曲

三步走：

  1. 认路：明确谁是自变量 (x, y)，谁是中间变量 (u, v)，并写出 ∂u/∂x、∂u/∂y、∂v/∂x、∂v/∂y 这 4 个数。
  2. 求一阶偏导：用链式法则 ∂f/∂x = f'1·∂u/∂x + f'2·∂v/∂x。
  3. 求二阶偏导：对一阶偏导再求偏导，链式法则每求一次都要重新用（f'1, f'2 仍是 (u,v) 的函数）。

抽象复合函数求导 · 漏项预防针
① 先写中间变量：u = x+y, v = x−y，把 ∂u/∂x, ∂u/∂y, ∂v/∂x, ∂v/∂y 这 4 个数（全是 ±1）写下来再动手；
② f'1, f'2 仍是 (u,v) 的函数，再求偏导时链式法则又要重新用一遍；
③ 题目说"二阶连续偏导"就要用 f"12 = f"21 化简，是送分项，不化简就丢 1~2 分。
链式法则 · 一图抵十句
∂f/∂x = f'1·∂u/∂x + f'2·∂v/∂x（沿 x 方向）
∂f/∂y = f'1·∂u/∂y + f'2·∂v/∂y（沿 y 方向）
∂(f'1)/∂x = f"11·∂u/∂x + f"12·∂v/∂x（再链一次）
∂(f'1)/∂y = f"11·∂u/∂y + f"12·∂v/∂y（再链一次）
记法："下标对下标，链子套链子"
这道题最容易踩的 4 个坑
  • 坑 1：忘了 f'1 还是 (u,v) 的函数 → 求二阶偏导时直接当成常数 → 漏掉链式法则，结果少一半。
  • 坑 2：把 ∂v/∂y 算错 → v = x−y，所以 ∂v/∂y = −1（很多同学写成 +1）。
  • 坑 3：不化简混合偏导 → 题目给了"二阶连续偏导"，就是让你用 f"12 = f"21，不化简等于白给分数。
  • 坑 4：求导顺序乱 → 因为 f 二阶连续偏导，求导顺序不影响结果，建议固定"先 x 后 y"的顺序，避免漏项。

① 写清"中间变量"（1 分）：u = x+y, v = x−y；
② 写出一阶偏导 ∂g/∂x, ∂g/∂y（3 分）：必须展示链式法则过程；
③ 写出 ∂²g/∂x²（3 分）：再次链式法则 + 混合偏导相等化简；
④ 写出 ∂²g/∂x∂y、∂²g/∂y²（4 分）：同上，化简到最简形式；
⑤ 明确写出"由二阶偏导连续 ⇒ f"12 = f"21"（1 分）：这是结论性陈述，不能省略。
📖 必背：二元无条件极值
  1. 求 ∂f/∂x = 0，∂f/∂y = 0，解出所有驻点。
  2. 求二阶偏导：fxx、fyy、fxy。
  3. 计算判别式 AC − B²（A=fxx，C=fyy，B=fxy）：
    • AC − B² > 0 且 A > 0 → 极小值
    • AC − B² > 0 且 A < 0 → 极大值
    • AC − B² < 0 → 不是极值
    • AC − B² = 0 → 判别式失效，必须额外分析（沿特殊路径 / Taylor / 定义法）
🎓 AC − B² 是谁创造的？为什么偏偏是它？

👨‍🏫发明人：Ludwig Otto Hesse（路德维希·奥托·黑塞）

19 世纪德国数学家（1811–1874），哥尼斯堡大学教授。他系统研究了多元函数的二阶偏导数矩阵，后人把它命名为 Hessian 矩阵。对于二元函数 f(x, y)：

H = \(\begin{pmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{pmatrix} = \begin{pmatrix} A & B \\ B & C \end{pmatrix}\)，其行列式 \(\det(H) = AC - B^2\)

所以"AC − B²"就是 Hessian 矩阵的行列式，也叫"黑塞判别式"（Hessian Discriminant）。

🧪为什么偏偏是它？（配方推导）

在驻点 (x₀, y₀) 处用泰勒展开（ch01 学过的工具），f 近似为：

f(x₀+h, y₀+k) ≈ f(x₀, y₀) + \(\tfrac{1}{2}\big[\underbrace{Ah^2 + 2Bhk + Ck^2}_{\text{关键二次型}Q(h,k)}\big]\)

驻点处一阶偏导 = 0 → 一阶项消失 → 留下的就是二次型 Q。Q 决定驻点是山谷/山顶/马鞍。

把 Q 对 h 配方（关键一步！）：

Q(h,k) = A\(\!\left(h + \dfrac{Bk}{A}\right)^{\!2}\) + \(\dfrac{AC - B^2}{A}\,k^2\)

配方后冒出两个平方项，符号各管各的：

  • 第一项 A(h+Bk/A)² —— 平方，符号由 A 决定
  • 第二项 (AC-B²)/A · k² —— 平方，符号由 AC − B² 决定（k² 永远 ≥ 0）

两类符号的"配合"决定了地形：

AC − B²AQ 的符号几何形状结论
> 0> 0两项均 ≥ 0 → Q ≥ 0🕳️ 山谷极小值
> 0< 0两项均 ≤ 0 → Q ≤ 0⛰️ 山顶极大值
< 0任意两项异号 → Q 可正可负🐴 马鞍不是极值
= 0任意配方失效❓ 退化判别式失效，要额外分析

🔗推广到 n 元（了解即可）

n 元函数的 Hessian 是 n×n 对称矩阵，Sylvester 判据给出：

  • 顺序主子式全 > 0 → 正定 → 极小
  • 顺序主子式符号交替 → 负定 → 极大
  • 有主子式 ≤ 0 → 不定号 → 鞍点

二元情形下，2 个顺序主子式是 A 和 AC−B²，正好退化成我们这个判别式。AC−B² 其实是 Sylvester 判据在 n=2 时的特例。

🎯口诀速记

"驻点处展二次型，配方分两半；
一半看 A 决定上下，另一半看 AC−B² 决定方向；
同号是极值，异号是马鞍，等于零要换判。"

AC − B² 配方公式（核心公式卡）
在驻点 (x₀, y₀) 处展开 f：\(f \approx f_0 + \tfrac{1}{2}(Ah^2 + 2Bhk + Ck^2)\)
对 h 配方：\(Q(h,k) = A\!\left(h + \dfrac{Bk}{A}\right)^{\!2} + \dfrac{AC-B^2}{A}\,k^2\)
结论：两个平方项 → 符号组合决定地形。
记法："下标对下标，配方看两端"
AC − B² 的 2 个真题坑（2022 必背）
  • 坑 1：AC − B² = 0 不是"不是极值"！
    2022 数三 (0,0) 处 AC−B² = 0 → 判别式失效。
    小白直接判"不是极值" → 丢 1~2 分。
    正确做法：沿 y=0 等路径分析（f(x,0)=x⁵ 有正有负 → 确认不是极值）。
  • 坑 2：AC − B² < 0 不是"极值"！
    2022 数二 (-e⁻¹, (2k+1)π) 处 AC−B² = -e⁻² < 0 → 鞍点，不是极值。
    小白易混"AC−B² 为负 = 极大值" → 错！
    AC−B² 符号本身不能单独决定，要看和 A 的组合。
4 类驻点判别（小白救命卡）
① 山谷：AC−B² > 0 + A > 0 → 极小值，记得代回去求 f 值；
② 山顶：AC−B² > 0 + A < 0 → 极大值；
③ 马鞍：AC−B² < 0 → 不是极值（一个方向上山，另一个下山）；
④ 退化：AC−B² = 0 → 判别式失效，必须额外分析（沿特殊路径 / Taylor 展开 / 定义法）。

📝 真题 1 | 2019 数一/二/三 | 解答题 · 12分
设函数 \(f(x)=\begin{cases}x^{2x}, & x>0 \\ x+2, & x\le 0\end{cases}\)，求 \(f(x)\) 的极值。

🎯 出题人考察点（这是高频考点！）

① 分段函数极值需分段讨论；② 幂指函数求导（对数求导法）；③ 不可导点也可能是极值点

🧭 详细解析（小白逐步版）

**分段函数极值**：① \(x>0\)：\(f(x)=e^{2x\ln x}\)，\(f\prime(x)=x^{2x}(2\ln x+2)\)，令 \(f\prime=0\) 得 \(\ln x=-1\)，即 \(x=1/e\)，\(f(1/e)=e^{-2/e}\)；② \(x=0\)：左右极限 1 和 2；③ \(x<0\)：\(f\prime(x)=1>0\) 单调增。综合得极小值 \(f(1/e)=e^{-2/e}\)。

💯 得分点（阅卷老师按点给分）

① \(x>0\) 求导 (4分)；② 极值点判断 (3分)；③ 不可导点讨论 (3分)；④ 最终答案 (2分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 忘了讨论 \(x=0\) 处；⚠️ 幂指函数直接套公式错；⚠️ 把极值点写成 \(x=1\) 而非 \(1/e\)

📝 真题 2 | 2021 数一 | 解答题 · 12分
设 \(f(x)\) 在 \([0,1]\) 上连续，在 \((0,1)\) 内二阶可导，且 \(\int_0^1 f(x)\,dx=0,\) \(f(0)+f(1)=0\)。证明：存在 \(\xi\in(0,1)\) 使 \(f\prime\prime(\xi)=f(\xi)\)。

🎯 出题人考察点（这是高频考点！）

① 辅助函数构造（核心难点）；② 罗尔定理的使用；③ 复合函数求导

🧭 详细解析（小白逐步版）

**辅助函数**：构造 \(F(x)=f\prime(x)\sin x - f(x)\cos x\)，求导 \(F\prime=f\prime\prime\sin x + f\prime\cos x - f\prime\cos x + f\sin x = (f\prime\prime+f)\sin x\)。验证 \(F(0)=-f(0),F(1)=f\prime(1)\sin 1\)。由 \(f(0)+f(1)=0\) 知 \(F(0)=-f(0)=f(1)\)，结合 \(F(1)=f\prime(1)\sin 1\)，需 \(f\prime(1)\sin 1=-f(0)\)，不一定成立。**更好构造**：\(G(x)=f\prime(x)\sin x + f(x)\cos x\)，\(G\prime=(f\prime\prime+f)\sin x\)。

💯 得分点（阅卷老师按点给分）

① 构造辅助函数 (5分)；② 验证端点函数值相等 (3分)；③ 应用罗尔定理 (3分)；④ 结论 (1分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 辅助函数构造错误是最常见丢分点；⚠️ 没验证端点条件；⚠️ 罗尔定理的三个条件缺一不可

📝 真题 3 | 2022 数二 | 解答题 · 12分
求函数 \(f(x,y)=x e^{\cos y}+\dfrac{x^2}{2}\) 的极值。

为什么要做这题？"x 配 x²"+"三角指数"是 2022 之后真题最常考的二元极值组合 —— 一个等式解 x，一个等式挑出三角零点（sin y = 0），套路固定，背下来 5 分钟拿满 12 分。

🎯 出题人考察点（必背三件套）

① 偏导数正确求（含链式法则 \(\frac{d}{dy}e^{\cos y}=-e^{\cos y}\sin y\)）；② 解方程组而不是解单个方程；③ AC−B² 判别 + 求极值（注意 cos y 的周期性）。

🧭 第一步：求一阶偏导（小白挨个敲）

把 f 拆成两块：第一块 \(x\cdot e^{\cos y}\)，第二块 \(\dfrac{x^2}{2}\)，分别对 x、y 求偏导再相加。

\(f_x = e^{\cos y}+x\)（y 当常数，\(e^{\cos y}\) 整体看成一个"系数"）
\(f_y = x\cdot e^{\cos y}\cdot (-\sin y) = -x e^{\cos y}\sin y\)（x 当常数，对 y 求导要套链式：\(\frac{d}{dy}e^{\cos y}=-e^{\cos y}\sin y\)）

🧭 第二步：解驻点方程组（核心难点）

令 \(f_x=0\)：\(x=-e^{\cos y}\)（因为 \(e^{\cos y}>0\)，x 必为负数）

令 \(f_y=0\)：\(-x e^{\cos y}\sin y=0\)，因为 \(e^{\cos y}>0\)，必有 \(\sin y=0\)

所以 \(y=k\pi\)，于是 \(\cos y = \cos(k\pi)=(-1)^k\)。

🎨 sin/cos 在 y = kπ 处的取值图（小白速记卡）

速记口诀："sin 撞零，cos 翻 ±1" —— k 每走过 1，cos 就翻一次号。

把 \(y=k\pi\) 代回第一个等式：\(x=-e^{(-1)^k}\)。分类讨论：

  • k 是偶数（\(y=0,\pm 2\pi,\pm 4\pi,\dots\)）：\(\cos y=1\)，\(x=-e\)，驻点 \((-e, 2k\pi)\)
  • k 是奇数（\(y=\pm\pi,\pm 3\pi,\dots\)）：\(\cos y=-1\)，\(x=-e^{-1}\)，驻点 \((-e^{-1}, (2k+1)\pi)\)

⚠️ 小白易错：以为"驻点只有 1 个"。其实 y 有无穷多解（cos y 周期性），但判别式结果按 k 分类相同，只需算两类代表点。

🧭 第三步：求二阶偏导

这一题二阶偏导里 A=1 是固定的，C 看着吓人但驻点上 sin y=0 → 大幅化简。

\(A = f_{xx} = 1\)
\(B = f_{xy} = -e^{\cos y}\sin y\)（驻点上 = 0）
\(C = f_{yy} = -x e^{\cos y}(\cos y-\sin^2 y)\)（驻点上 \(\sin y=0\)，化简为 \(-x e^{\cos y}\cos y\)）

🧭 第四步：AC−B² 判别 + 求极值

驻点 1：\((-e, 2k\pi)\)（k 为偶数，\(\cos y=1\)）

  • \(A=1\)，\(B=0\)，\(C = -(-e)\cdot e^1 \cdot 1 = e^2\)
  • \(AC-B^2 = e^2 > 0\) 且 \(A>0\) → 极小值
  • 极小值 = \(f(-e, 2k\pi) = (-e)\cdot e^{1}+\dfrac{(-e)^2}{2}=-e^2+\dfrac{e^2}{2}=\boxed{-\dfrac{e^2}{2}}\)

驻点 2：\((-e^{-1}, (2k+1)\pi)\)（k 为奇数，\(\cos y=-1\)）

  • \(A=1\)，\(B=0\)，\(C = -(-e^{-1})\cdot e^{-1}\cdot(-1) = -e^{-2}\)
  • \(AC-B^2 = 1\cdot(-e^{-2}) = -e^{-2} < 0\) → 不是极值（鞍点）

🎯 最终答案：驻点 \((-e, 2k\pi)\)，\(k\in\mathbb{Z}\) 是极小值点，极小值 \(=-\dfrac{e^2}{2}\)；驻点 \((-e^{-1}, (2k+1)\pi)\) 不是极值；无极大值。

💯 得分点（阅卷老师按点给分）

① 一阶偏导正确（特别是 f_y 的链式）(4分)；② 解方程组得到驻点（注意 cos y 周期性、k 分类）(3分)；③ 二阶偏导正确 (2分)；④ AC−B² 判别 + 排除非极值 (2分)；⑤ 求出极值 (1分)。

⚠️ 小白踩坑预警（错一个扣 2~3 分）

⚠️ f_y 的链式求导漏掉 \(-\sin y\)（最常错！）；⚠️ 只解出 y = 0 这一个，漏掉 y = π 等；⚠️ 没按 k 分类讨论；⚠️ 把鞍点 \((-e^{-1}, (2k+1)\pi)\) 当成极值点（这是 2022 数二最容易丢分的地方）。

🔗 知识点桥接

这题是 P2「判别式 AC−B²」的经典应用。还记得 P2 必背的判别规则吗？AC−B² > 0 + A > 0 → 极小值；AC−B² < 0 → 鞍点，不是极值（这条小白最爱忘！）。本题 A 恒等于 1，所以只需看 C 的正负。

🧠 小白记忆类比：把 f 想象成"沙漠里的温度场"——
x = -e^(cos y) 是"温度等高线在每个 y 处必须满足的位置"，sin y = 0 是"z 轴方向上的冷热分界线"。每个 kπ 都是一条等温线，所以驻点有无穷多个（cos y 周期性），但每个驻点的"地形"不同：(-e, 2kπ) 是山谷（极小），(-e⁻¹, (2k+1)π) 是马鞍（不是极值）。

🎯 出题人视角：这道题是 2022 数二的"送分题"——套路固定（链式 + 周期性 + AC−B²），出题人想筛：① 链式求导的扎实度（链子套对没？）；② 周期性带来的多解意识（漏掉 y=π 就丢 3 分）；③ AC−B² 判别的反向（排除鞍点）；他不想要的是"只会套公式、看到 sin y = 0 就懵、看到 AC−B² < 0 也判极值的小白"。
📝 真题 4 | 2022 数三 | 解答题 · 12分
求函数 \(f(x,y)=(y-x^2)(y-x^3)\) 的极值。

为什么要做这题？这是 2022 数三的"压轴级"二元极值 —— AC−B² = 0 的边界情形（判别式失效）+ 多元多项式因式分解 + 解联立方程组的代数功底。三关卡全过才拿满分。

🎯 出题人考察点（三关卡）

① 多元函数求偏导（带 x²、x³ 项）；② 解二元方程组（不是解单个方程！要联立）；③ AC−B² = 0 时的边界情形处理（这关最坑，判别式失效）。

🧭 第一步：展开 + 求一阶偏导（小白要慢）

先把 f 展开（避免求偏导时漏项）：

\(f = y^2 - x^2 y - x^3 y + x^5\)

再分别对 x、y 求偏导：

\(f_x = -2xy - 3x^2 y + 5x^4 = 5x^4 - 2xy(1+\frac{3x}{2})\)（简化：\(= 5x^4 - 2xy - 3x^2y\)）
\(f_y = 2y - x^2 - x^3\)

🧭 第二步：解驻点方程组（核心难点！）

从最简单的 \(f_y=0\) 入手：

\(2y = x^2 + x^3 = x^2(1+x) \Rightarrow y = \dfrac{x^2(1+x)}{2}\)

把这个 y 代回 \(f_x=0\)：

\(f_x = -2x(y-x^3) - 3x^2(y-x^2) = -x[2(y-x^3) + 3x(y-x^2)]\)

分两种情况：

  • 情况 1：x = 0 → 代入 y 公式得 \(y=0\)，驻点 \((0, 0)\)
  • 情况 2：\(2(y-x^3) + 3x(y-x^2) = 0\) → 展开：\(2y + 3xy = 5x^3\)，即 \(y(2+3x) = 5x^3\)，所以 \(y = \dfrac{5x^3}{2+3x}\)

联立两种情况下的 y：

\(\dfrac{5x^3}{2+3x} = \dfrac{x^2(1+x)}{2}\)，两边乘 \(2(2+3x)\) 并除以 \(x^2\)（x≠0）：\(10x = (1+x)(2+3x)\)

展开：\(10x = 2 + 5x + 3x^2\)，整理：\(3x^2 - 5x + 2 = 0\)，因式分解：\((3x-2)(x-1) = 0\)

解得 \(x = 1\) 或 \(x = \dfrac{2}{3}\)。代回 y 公式：

  • \(x=1\)：\(y = \dfrac{1 \cdot 2}{2} = 1\)，驻点 \((1, 1)\)
  • \(x=\dfrac{2}{3}\)：\(y = \dfrac{(4/9) \cdot (5/3)}{2} = \dfrac{10}{27}\)，驻点 \((\dfrac{2}{3}, \dfrac{10}{27})\)

共 3 个驻点：\((0, 0)\)、\((1, 1)\)、\((\dfrac{2}{3}, \dfrac{10}{27})\)。

🧭 第三步：求二阶偏导（机械活，但要稳）

\(A = f_{xx} = 20x^3 - 2y - 6xy\)（对 f_x = 5x⁴ - 2xy - 3x²y 求 x 偏导）
\(B = f_{xy} = -2x - 3x^2\)（对 f_x 求 y 偏导）
\(C = f_{yy} = 2\)

🧭 第四步：AC−B² 判别（三个驻点逐一过堂）

驻点 1：(0, 0)

  • \(A = 0, B = 0, C = 2\)
  • \(AC - B^2 = 0 - 0 = 0\)
  • ❗ 判别式 = 0，AC−B² 失效！需要额外判断（最常考也最容易出错）

怎么破？用"沿特殊路径分析"。

  • 沿 \(y = 0\)：\(f(x, 0) = -x^2 \cdot (-x^3) = x^5\)。当 x > 0 时 f > 0；当 x < 0 时 f < 0 → 有正有负，不是极值
  • 沿 \(y = x^2\)：\(f(x, x^2) = 0\)
  • 沿 \(y = x^3\)：\(f(x, x^3) = 0\)

判定：(0, 0) 不是极值点。

驻点 2：(1, 1)

  • \(A = 20 - 2 - 6 = 12, B = -2 - 3 = -5, C = 2\)
  • \(AC - B^2 = 24 - 25 = -1 < 0\)
  • 判定：不是极值（AC−B² < 0 是鞍点）

驻点 3：\((\dfrac{2}{3}, \dfrac{10}{27})\)

  • \(A = 20 \cdot \dfrac{8}{27} - 2 \cdot \dfrac{10}{27} - 6 \cdot \dfrac{2}{3} \cdot \dfrac{10}{27} = \dfrac{160 - 20 - 40}{27} = \dfrac{100}{27}\)
  • \(B = -\dfrac{4}{3} - \dfrac{4}{3} = -\dfrac{8}{3}\)
  • \(C = 2\)
  • \(AC - B^2 = \dfrac{200}{27} - \dfrac{64}{9} = \dfrac{200}{27} - \dfrac{192}{27} = \dfrac{8}{27} > 0\)
  • \(A > 0\) → 极小值
  • 极小值 = \(f(\dfrac{2}{3}, \dfrac{10}{27}) = (\dfrac{10}{27} - \dfrac{4}{9})(\dfrac{10}{27} - \dfrac{8}{27}) = (-\dfrac{2}{27}) \cdot \dfrac{2}{27} = -\dfrac{4}{729}\)

解二元驻点方程组的"两段式"心法
① 先解 f_y = 0：因为 f_y 一般更简单（只有一个 y），能显式解出 y = g(x)；
② 再代入 f_x = 0：把 y = g(x) 代进去，f_x 就变成"只含 x 的方程"；
③ 别漏 x = 0 这种隐式解：如果 f_x 有公因子 x，x = 0 是一条独立分支，漏掉就丢 3~4 分。
二元极值 · 4 大常见丢分点
  • 丢分点 1：AC−B² = 0 直接判"不是极值" → 错！必须额外判断（沿路径分析）。2022 数三真题坑过 80% 的小白。
  • 丢分点 2：周期性函数漏解 → sin y = 0 漏掉 y = π；cos y = 0 漏掉 y = π/2。漏一个扣 2~3 分。
  • 丢分点 3：联立方程漏掉 x = 0 支 → f_x 有公因子 x 时，x = 0 是隐式解，必须单独列。
  • 丢分点 4：二阶偏导 C 算错 → 求 f_yy 时漏掉链式项（特别是 sin y · cos y 这种乘积项）。
2022 两道极值题的"3 件套武器"
① 武器 1 · 链式法则：偏导时遇到复合函数（如 e^(cos y)、(y-x³)），必须套链式；

③ 武器 3 · AC−B² 判别 + 边界兜底：三种情形各走各路，AC−B²=0 必须额外判断。
