🎬 一句话回顾 ch02 + 预告 ch05

ch02 我们学了"一维的导数"——f'(x) 告诉你函数沿 x 方向怎么变。但现实世界从来不是一维的：

温度场 / 电磁场 / 流体速度场 / 股票组合 / 神经网络梯度 / GPS 三维坐标 —— 这些都是"输入多个变量，输出一个值"的多维函数。

本章 ch05 要解决的核心问题："多维世界里，每一维的变化率怎么算？怎么合成？怎么判断极值？"

三个答案就是本章三大块：

  • 🥇 偏导数（每维单独算）—— 欧拉 1734
  • 🥈 全微分（多维一起算）—— 柯西 1820
  • 🥉 极值判别 AC−B²（多维找最值）—— 黑塞 1850
⏳ 200 年多元微积分编年史 · ch05 主要工具的诞生年份

速记："欧拉提概念 → 拉格朗日搞乘数法 → 勒让德起名 → 柯西定形式 → 雅可比铺矩阵 → 黑塞封判别"

🌍 一元到二元的"维度跃迁"：为什么多元微积分更"难"？

从 ch02 的一元导数 f'(x)，到本章的偏导数 ∂f/∂x，看上去只多了一个 ∂ 符号，但背后难度飙升：

维度求导数求极值难度
1 维f'(x) 一个就够f'(x)=0 一个方程⭐
2 维∂f/∂x + ∂f/∂y 两个联立两个方程 + AC−B² 判别⭐⭐⭐
n 维n 个偏导 + 链式法则Hessian 矩阵 + Sylvester 判据⭐⭐⭐⭐⭐

🧠为什么"维度 +1"难度 ×3？

  • 每多一维，偏导数就多 1 个 → 方程组规模涨
  • 每多一维，极值判定从"导数=0"变成"判别矩阵" → AC−B² 之类的判别法
  • 每多一维，可微性条件从"导数连续"变成"所有方向极限都吻合" → 一元能导就能微，多元不行！

🎯本章核心要诀：

"降维思想：面对 f(x,y) 不知道怎么办？—— 把一个变量当成常数，就退化成 ch02 的一元函数！
例：对 y 求偏导时把 x 当常数 → 整个问题就变成 ch02 那种"一个变量"的微积分 → 所有一元工具都能用。"

🎯 核心考点（小白路线图）

  • 🧮 偏导数：每维单独算（"按住其他变量，对一个变量求导"）
  • 📐 全微分：多维一起算（"把所有变化打包成一个 df"）
  • 🔗 链式法则：复合函数求导（"路径分叉"）
  • 🕵️ 隐函数：F(x,y)=0 求 dy/dx（"等高线斜率"）
  • ⛰️ 无条件极值：求 AC−B² 判别（"找山顶/山谷"）
  • 🔒 条件极值：拉格朗日乘数法（"带锁链的山顶"）
🏔️ 一句话直觉：偏导数 = 山坡的"坡度"

把多元函数 z = f(x, y) 想象成一座山的海拔：

  • x 方向 = "东边" → 偏导 ∂f/∂x = 面朝东的坡度
  • y 方向 = "北边" → 偏导 ∂f/∂y = 面朝北的坡度
  • 山顶 / 山谷 = 两个方向的坡度都是 0 → 极值点

⛰️ 生活例子：你去爬山，手机 GPS 显示当前海拔 z = f(x, y)。你想知道"如果往东走 1 米，海拔会变多少？"——答：∂z/∂x 米。"往北走 1 米呢？"——答：∂z/∂y 米。

🎯 小白速记："按住别的变量 → 对一个变量求导 → 这就是偏导数。" 这一句话就是偏导的全部！

📖 必背：求偏导法则（3 个递进例子）

核心口诀："求谁把谁当主元，其他变量全部当常数。"

这条口诀 + ch02 的一元求导公式 → 你已经会求所有偏导！

🌰 例 1：多项式（最简单）

求 z = x²y + xy² 的偏导

∂z/∂x = 2xy + y²   （y 当常数，对 x 求导：x²→2x，xy²→y²）
∂z/∂y = x² + 2xy   （x 当常数，对 y 求导：x²y→x²，xy²→2xy）

🌰 例 2：带三角/指数（链式）

求 z = e^(xy)·sin(x+y) 的偏导

∂z/∂x = ye^(xy)·sin(x+y) + e^(xy)·cos(x+y)（乘积法则 + 链式）
= e^(xy)·[y·sin(x+y) + cos(x+y)]
∂z/∂y = x·e^(xy)·sin(x+y) + e^(xy)·cos(x+y)
= e^(xy)·[x·sin(x+y) + cos(x+y)]

🔍 易错：别忘了 e^(xy) 对 x 求导要乘 y（链式），cos(x+y) 内部求导是 1（不是 2x）。

🌰 例 3：抽象函数（链式法则登场）

设 z = f(xy, x+y)，求 ∂z/∂x, ∂z/∂y（这是 2019 真题 1 的简化版）

记 u = xy, v = x+y。z 通过 u, v 间接依赖 x, y。

∂z/∂x = (∂f/∂u)·(∂u/∂x) + (∂f/∂v)·(∂v/∂x) = y·f'1 + f'2
∂z/∂y = (∂f/∂u)·(∂u/∂y) + (∂f/∂v)·(∂v/∂y) = x·f'1 + f'2

📝 记号：f'1 = ∂f/∂u，f'2 = ∂f/∂v。f'1 还是 (u,v) 的函数 → 还要再套一次链式（这就是二阶偏导的来源）。

🧠 核心心法：降维思想（小白救命稻草）

面对 f(x, y, z, ...) 不知道怎么求导？—— 把其他变量全当成常数，剩下的问题就是 ch02 的一元微积分！

这就是"降维"：把 N 维问题降成 1 维，然后用 ch02 学的所有工具（链式、乘积、商、复合函数）逐个击破。

N 维函数降维手法变 1 维后用什么
∂f/∂xy, z 全当常数ch02 一切工具
∂f/∂yx, z 全当常数ch02 一切工具
∂²f/∂x²对 ∂f/∂x 再按 x 降维ch02 二阶导
∂²f/∂x∂y先对 x 降维，再对 y 降维ch02 链式法则

"降维思想：再复杂的多变量问题，本质就是'按住一个 → 算一个'的循环。所有多元考点，90% 都是 ch02 的工具在多元场景下的重复使用。"

🔗 链式法则："路径图" 心法

复合函数求导是小白最懵的地方——变量太多、关系太乱。教你一招：画"路径图"，每条路径 = 一次偏导 × 偏导。

🌰 例题：z = f(u, v), u = g(x, y), v = h(x, y)，求 ∂z/∂x

画出"路径图"：

z
↑    ↑
f1   f2   （每条箭头 = 一次偏导）
↑    ↑
u    v
↑    ↑
∂u/∂x   ∂v/∂x   （这些"中转"也可能是 x, y 的函数）
↑    ↑
x    y

规则：从 z 出发，沿每条路径走到 x，把每条路径上的偏导全部乘起来，最后所有路径相加。

∂z/∂x = f1·(∂u/∂x) + f2·(∂v/∂x)   （2 条路径相加）
∂z/∂y = f1·(∂u/∂y) + f2·(∂v/∂y)

⚠️ 小白易错：① 漏掉某条路径（漏一项 = 错）；② 把 f1, f2 当成常数（它们还是 u, v 的函数，求二阶偏导时还要再套一次链式）。

📝 例题 1（基础）· z = x²y + xy² 求偏导
求 z = x²y + xy² 的偏导 ∂z/∂x 和 ∂z/∂y

🎯 考点：多项式偏导（"按一个变量、其余当常数"）

🧭 解析：

∂z/∂x：把 y 当常数，对 x 求导：x² → 2x, xy² → y² → 2xy + y²

∂z/∂y：把 x 当常数，对 y 求导：x²y → x², xy² → 2xy → x² + 2xy

📝 例题 2（链式）· z = e^(xy) 求偏导
求 z = e^(xy) 的 ∂z/∂x 和 ∂z/∂y

🎯 考点：链式法则（最易错的复合函数）

🧭 解析：

设 u = xy。z = e^u。

∂z/∂x = (dz/du)·(du/dx) = e^(xy) · y = y·e^(xy)

∂z/∂y = (dz/du)·(du/dy) = e^(xy) · x = x·e^(xy)

💡 小白速记：看到 e^(...) / sin(...) / ln(...) 这种"函数套函数"，立刻想"链式"：外层求导 × 内层求导。

📝 例题 3（隐函数）· F(x,y) = x² + y² - 1 = 0 求 dy/dx
设 x² + y² = 1（单位圆），求 dy/dx

🎯 考点：隐函数求导（不能显式解出 y 的情况）

🧭 解析（小白逐步）：

第 1 步：把 y 当成 x 的函数，整体对 x 求导：

2x + 2y·(dy/dx) = 0

第 2 步：把 dy/dx 单独留在一边，其他移到另一边：

2y·(dy/dx) = -2x → dy/dx = -x/y

🧐 几何意义：在单位圆上，dy/dx = -x/y 就是该点的切线斜率。验证：在 (1, 0) 处切线斜率 = -∞（垂直）；在 (0, 1) 处切线斜率 = 0（水平）—— 完全对！

💡 小白速记："y 是 x 的函数" → 求导时不要把 dy/dx 当 0 → 链式法则：dy/dx 保留。→ 解出 dy/dx。

📖 小白扫盲：偏导数 · 全微分 · 可微性（一图分清）

🎯 三者关系：偏导存在 → 偏导连续 ⟹ 可微 ⟹ 连续且偏导存在

概念定义判定方法常见误解
偏导数
\(\dfrac{\partial f}{\partial x}\)		 固定 \(y\)，对 \(x\) 的导数 用定义：\(\lim_{h\to0} \dfrac{f(x+h,y)-f(x,y)}{h}\) 「偏导存在」≠「可微」（如 \(f(x,y)=\dfrac{xy}{x^2+y^2}\) 补充定义在 0 处）
全微分
\(\mathrm{d}f\)		 \(\dfrac{\partial f}{\partial x}\mathrm{d}x + \dfrac{\partial f}{\partial y}\mathrm{d}y\) 可微时全微分才存在；全微分形式 = 各偏导乘对应微分之和 「写出全微分」≠「可微」— 写全微分的表达式不要求可微，但全微分概念意义上的"存在"要求可微
可微性 \(f(x,y)-f(0,0)=f_x(0,0)x+f_y(0,0)y+o(\sqrt{x^2+y^2})\) ① 偏导连续 ⟹ 可微（充分条件）
② 直接用可微定义验证增量能否线性逼近		 「偏导存在」≠「可微」；「偏导连续」⟹ 可微（但可微 ≠ 偏导连续）

⚠️ 一元到二元的直观差异

  • 一元：可导 ⟺ 可微（等价）
  • 二元：偏导存在 不 ⟺ 可微（偏导存在是必要条件，不是充分条件）
  • 记忆方法：「二元的可微 = 一元的可导 × 2，但要求比一元苛刻得多」

🧠 判定流程

问：二元函数在 \((0,0)\) 处是否可微？
步骤 ①：检查连续性 — 不连续一定不可微
步骤 ②：求两个偏导 — 偏导不存在一定不可微（但存在也不一定可微）
步骤 ③：计算 \(\lim_{(x,y)\to(0,0)} \dfrac{f(x,y)-f(0,0)-f_x(0,0)x-f_y(0,0)y}{\sqrt{x^2+y^2}}\) — 若为 0 则可微

🆘 d vs ∂ 全攻略（小白救命卡）

看到公式里又 d 又 ∂ 别晕——其实只记一句话："d 整体动，∂ 一边看"。

📊 d vs ∂ 对比（4 维分类）

你在算什么用 ∂ 还是 d？例子
f 对 x 的变化率（其他变量冻结）∂ （片面）∂f/∂x, ∂f/∂y, ∂²f/∂x²
f 的总变化（所有变量都动）d （整体）df = (∂f/∂x)dx + (∂f/∂y)dy
u 对 x 的变化率（u 是 x, y 的函数）∂ （片面）∂u/∂x, ∂u/∂y
u 的总变化d （整体）du = (∂u/∂x)dx + (∂u/∂y)dy

💡 小白速记：∂ 算"局部变化率"（按一个变量）；d 算"实际变化量"（所有变量都动）。

🚗 (∂z/∂x)·dx 的物理意义：开车类比

核心公式：dz = (∂z/∂x)·dx + (∂z/∂y)·dy = 2x·dx + 2y·dy

这一项 (∂z/∂x)·dx 什么意思？—— 开车类比一目了然：

🚗 "速度 × 时间 = 路程" 类比

开车数学含义
速度 60 km/h∂z/∂x = 2x"x 变 1 单位时 z 变多少"（变化率）
开了 0.5 小时dx = 0.5"x 实际变了多少"（实际变化）
60 × 0.5 = 30 km(∂z/∂x)·dx"这一路对 z 变化的贡献量"

🧪 用 z = x² + y² 在 (3, 4) 验证

假设 x 动 dx = 0.1，y 动 dy = 0.05：

路径∂z/∂x · dx∂z/∂y · dy相加
数值2·3·0.1 = 0.62·4·0.05 = 0.41.0

直接算：(3.1)² + (4.05)² - 25 = 1.0125 ≈ 1.0 ✓（误差 0.0125 = "曲面弯曲" 引起的）

🎯 小白结论：(∂z/∂x)·dx = "速度 × 时间" = "这一路走的路" = "对 z 变化的贡献量"。两路贡献相加 = z 的总变化。

🌡️ 温度场类比：为什么"两路相加"

问：为什么"链式 = 两路相加"？不是人为规定，而是 "f 依赖两路" + "两路又依赖 x" 的自然推论。

🌡️ 想象场景：你去看天气

变量物理含义偏导符号
z = 温度你最终要的—
u = 当地气压影响温度的"路 1"∂f/∂u
v = 当地湿度影响温度的"路 2"∂f/∂v
x = 经度, y = 纬度你走的方向∂u/∂x, ∂v/∂x

问：往东走 1 公里（dx=1），温度变多少？

温度 (z)
↑           ↑
∂f/∂u      ∂f/∂v
↑           ↑
气压 (u)     湿度 (v)
↑           ↑
∂u/∂x      ∂v/∂x
↑           ↑
经度 (x)    纬度 (y)

总温度变化 = 路径1贡献 + 路径2贡献：

∂z/∂x = (∂f/∂u)·(∂u/∂x) + (∂f/∂v)·(∂v/∂x)

两路独立并行汇入 z → 各自乘起来 → 全部相加。

🎯 小白结论："两路相加"不是人为规定，是全微分的线性性质。f 同时依赖 u 和 v → df = (∂f/∂u)du + (∂f/∂v)dv（天然两路）→ 链式 = 把 du, dv 代入 → 还是两路相加。

🧪 增量法证明（让"两路相加"无懈可击）

用严格的"加一项减一项"恒等式证明链式法则，每一步都是"两路分开近似再合并"。

📝 证明 4 步走

第 1 步：从 Δz 出发

Δz = f(u+Δu, v+Δv) − f(u, v)

第 2 步：加一项减一项（雕虫小技，但关键！）

= [f(u+Δu, v+Δv) − f(u, v+Δv)]   ↑ 路径 1：只让 u 变
  + [f(u, v+Δv) − f(u, v)]   ↑ 路径 2：只让 v 变

第 3 步：每一项用偏导数定义近似

≈ (∂f/∂u)·Δu + (∂f/∂v)·Δv

📝 "偏导数 = 小变化率"，对 f 在每一维上单独近似。

第 4 步：把 Δu, Δv 用 Δx, Δy 表示（再套一次全微分）

Δu ≈ (∂u/∂x)Δx + (∂u/∂y)Δy
Δv ≈ (∂v/∂x)Δx + (∂v/∂y)Δy

代入，按 Δx, Δy 合并：

Δz ≈ [(∂f/∂u)(∂u/∂x) + (∂f/∂v)(∂v/∂x)]·Δx + [(∂f/∂u)(∂u/∂y) + (∂f/∂v)(∂v/∂y)]·Δy

和 dz 的定义对比 → 系数匹配：

∂z/∂x = (∂f/∂u)(∂u/∂x) + (∂f/∂v)(∂v/∂x)    ✓
∂z/∂y = (∂f/∂u)(∂u/∂y) + (∂f/∂v)(∂v/∂y)    ✓

🎯 关键洞察：第 2 步的"加一项减一项"看起来是雕虫小技，但这正是"两路分别近似、最后相加"的核心。少做这一步，"链式 = 相加"就推不出来。

🔗 一气呵成：链式法则 = 全微分代入

现在把 d vs ∂ + 全微分 + 链式 串起来看，3 步出链式：

第 1 步：z = f(u, v) 的全微分（df 是"z 变了多少"，写成"u 这路 + v 这路"）

dz = df = (∂f/∂u)·du + (∂f/∂v)·dv

第 2 步：u, v 又是 x, y 的函数，把 du, dv 也写成全微分

du = (∂u/∂x)·dx + (∂u/∂y)·dy
dv = (∂v/∂x)·dx + (∂v/∂y)·dy

第 3 步：代入，按 dx, dy 整理（系数匹配）

dz = (∂f/∂u)·[(∂u/∂x)dx + (∂u/∂y)dy] + (∂f/∂v)·[(∂v/∂x)dx + (∂v/∂y)dy]
　 = [(∂f/∂u)(∂u/∂x) + (∂f/∂v)(∂v/∂x)]·dx + [(∂f/∂u)(∂u/∂y) + (∂f/∂v)(∂v/∂y)]·dy

对比 dz = (∂z/∂x)·dx + (∂z/∂y)·dy，系数必须相等 → 得到链式法则。

🎯 小白口诀："链式 = 全微分的嵌套代入"。df 是"两路信号汇入"（∂f/∂u + ∂f/∂v），du, dv 又是"上游两路汇入"（∂u/∂x + ∂u/∂y）→ 代进去自然变成"上游×下游 = 多路乘积相加"。

📝 真题 1 | 2019 数一 | 解答题 · 10分
设 \(z=f(xy, x+y)\)，求 \(\dfrac{\partial^{2}z}{\partial x\partial y}\)。

🎯 出题人考察点（必背三件套）

① 多元复合函数链式法则；② 二阶偏导计算（再套一次链式！）；③ 偏导符号记法 f1, f2（下标 = 对哪个中间变量求导）

🧭 第一步：画"路径图"找全所有依赖关系

设中间变量 u = xy, v = x+y。z 通过 u, v 间接依赖 x, y。

z
↑    ↑
f1   f2
↑    ↑
u=xy   v=x+y
↑    ↑    ↑
y    1    1

📝 记号速查：f1 = ∂f/∂u, f2 = ∂f/∂v, f11 = ∂²f/∂u², f12 = ∂²f/∂u∂v, f22 = ∂²f/∂v²。

🧭 第二步：求一阶偏导（走路径图）

从 z 走到 y：路径 z→u→y + z→v→y（"路径相加"）

∂z/∂y = f1·(∂u/∂y) + f2·(∂v/∂y) = x·f1 + f2

🧭 第三步：求二阶偏导（再走一次路径图！）

∂z/∂y 还是 (u, v) 的函数（因为 f1, f2 是 u, v 的函数）→ 要 再套一次链式，对 x 求导：

从 ∂z/∂y 走到 x：
x·f1 这一项 → 走 3 条路径：x→u→f1（用乘法法则 + 链式）
f2 这一项 → 走 2 条路径：x→v→f2（链式）

逐项展开：

∂(x·f1)/∂x = 1·f1 + x·∂(f1)/∂x   （乘法法则）
     = f1 + x·(f11·(∂u/∂x) + f12·(∂v/∂x))   （f1 还要再链式）
     = f1 + x·(f11·y + f12·1) = f1 + xy·f11 + x·f12

∂(f2)/∂x = f21·(∂u/∂x) + f22·(∂v/∂x)   （f2 还要再链式）
     = f21·y + f22·1 = y·f21 + f22

把两段加起来：

∂²z/∂x∂y = f1 + xy·f11 + x·f12 + y·f21 + f22
        = f1 + xy·f11 + (x+y)·f12 + f22   （合并 f12 项）

🎯 最终答案：\(\dfrac{\partial^{2}z}{\partial x\partial y} = f_1' + xy f_{11}'' + (x+y) f_{12}'' + f_{22}''\)

💯 得分点（阅卷老师按点给分）

① 一阶偏导 (3分)；② 识别 f1, f2 仍要再链式 (3分)；③ 二阶偏导完整展开 (3分)；④ 整理合并 (1分)。

⚠️ 小白三大踩坑（错一个扣 3~4 分）

⚠️ 漏掉项：只算了 f1·x 链式展开，漏了 f1 本身（"1 × f1"）。

⚠️ 二阶没再链式：把 f1, f2 当成 x, y 的函数算（其实它们是 u, v 的函数）→ 二阶偏导算错。

⚠️ 路径漏一条：从 z 到 (x, y) 一共 4 条路径（z→u→x, z→v→x, z→u→y, z→v→y），漏一条就少一项。

📝 真题 2 | 2021 数一 | 解答题 · 12分
求 \(f(x,y)=x^{3}-y^{3}+3x^{2}+3y^{2}-9x-27y\) 的极值。

🎯 出题人考察点（看上去是极值题，其实是"无解"题）

① 多元极值的必要条件（偏导 = 0）；② 配方判断二次函数有解/无解；③ 极值点的充分性判定。

🧭 第一步：求一阶偏导

对 x 求偏导（y 当常数）：

∂f/∂x = 3x² + 6x - 9 = 3(x-1)(x+3)

对 y 求偏导（x 当常数）：

∂f/∂y = -3y² + 6y - 27

🧭 第二步：解驻点方程组

从最简单的 ∂f/∂y = 0 开始（一眼看出能不能解）：

-3y² + 6y - 27 = 0 → 配方：-3(y² - 2y + 9) = 0 → -3(y-1)² - 24 = 0

左边是 (y-1)² + 8 × -3，永远 ≤ -24 < 0，无实数解！

⚠️ 小白易错：可能想"配方能解开吧"——但 (y-1)² ≥ 0，乘 -3 ≤ 0，再加 -24 永远 < 0，所以 ∂f/∂y = 0 无实数根。

🧭 第三步：得出"无极值"结论

驻点方程组 ∂f/∂x = 0 且 ∂f/∂y = 0 无解（∂f/∂y 永远不为 0）→ 无驻点 → 无极值。

🎯 最终答案：f(x, y) 在 R² 上无驻点，无极值。

🧐 补充：为什么这题是"看上去极值实际无极值"？f(x,y) 在 (3, 3) 附近一直是单调的（∂f/∂x 走过零点、∂f/∂y 一直负）—— f 在该区域 y 方向一直下降。所以"无山顶也无山谷"——真正的结论是无极值，不是无最大值。

💯 得分点（阅卷老师按点给分）

① 正确求偏导 (3分)；② 配方 ∂f/∂y 看出恒负 (4分)；③ 得出无驻点 + 无极值 (4分)；④ 解释清楚为什么无解 (1分)。

⚠️ 小白踩坑预警（错一个扣 3~4 分）

⚠️ 配方不仔细：漏掉 (y-1)² 前的 -3 符号，错误地"解出 y = 1"。

⚠️ 用错判别式：看到"求极值"就上 AC−B²，但没有驻点根本用不上判别式。

⚠️ 漏掉 ∂f/∂x 的解：∂f/∂x = 0 有解 (x = 1, -3)，但 ∂f/∂y = 0 无解，联立方程组无解。

看不见 z，也能找到它的山顶
为什么要学它？普通极值题把函数直接写成 z=f(x,y)，这道题却把 z 藏在方程里，故意让你“看不见函数”。咱们不必硬解 z，只要按三步走：把方程记成 F=0 → 隐式求偏导找驻点 → 二阶偏导判断山顶或山谷。
还记得 ch02 里“方程两边同时对 x 求导”吗？这里做的是同一件事，只不过现在要分别沿 x、y 两个方向各做一次。
📖 必背：二元隐函数极值题的工具箱

若方程 F(x,y,z)=0（F 是约束方程）确定了 z=z(x,y)（z 是 x、y 的函数），且 Fz≠0（F 对 z 的偏导不为 0），那么：

\(z_x=-\dfrac{F_x}{F_z}\)，  \(z_y=-\dfrac{F_y}{F_z}\)
符号小白翻译本题用途
\(F_x,F_y,F_z\)把 x、y、z 暂时看成三个独立按钮，只按其中一个按钮求偏导写出隐函数的一阶偏导
\(z_x=z_y=0\)沿 x、y 两个方向的坡度都为 0寻找驻点
\(A=z_{xx},B=z_{xy},C=z_{yy}\)驻点附近三个二阶偏导用 \(AC-B^2\) 判断极值

二阶判别口诀：若 \(AC-B^2>0\)，再看 A：A<0 是极大值，A>0 是极小值；若 \(AC-B^2<0\)，则不是极值。

📝 真题 3 | 2016 数二 | 二元隐函数求导 + 极值
已知 \(z=z(x,y)\) 由方程 \((x^2+y^2)z+\ln z+2(x+y+1)=0\) 所确定，求 \(z(x,y)\) 的极值。

🎯 先读题：为什么不能直接对 z 求普通导数？

因为 z 同时跟 x、y 变化，它不是只依赖一个变量的一元函数。题目还没有把 z 单独写出来，所以要把整个等式看成一根“约束绳”，x 或 y 一动，z 就必须跟着调整，保证等式仍等于 0。

第一步：建立 F，并先检查公式能不能用

令

\(F(x,y,z)=(x^2+y^2)z+\ln z+2(x+y+1)\)

由于式子里有 \(\ln z\)（z 的自然对数），所以定义域要求 z>0。把 x、y 固定，只对 z 求偏导：

\(F_z=x^2+y^2+\dfrac{1}{z}>0\)

这里 \(x^2+y^2\ge 0\)，而 \(z>0\Rightarrow 1/z>0\)，所以分母 \(F_z\) 永远不为 0。也就是说，方程确实能在附近把 z 看成 x、y 的函数，隐函数求导公式可以放心使用。

第二步：求两个一阶偏导，找到“平坡点”

先把 x、y、z 暂时看成互相独立的三个量，分别计算 F 的偏导：

\(F_x=2xz+2\)，  \(F_y=2yz+2\)，  \(F_z=x^2+y^2+\dfrac1z\)

因此 z 对 x、y 的偏导分别为：

\(z_x=-\dfrac{2xz+2}{x^2+y^2+1/z}\)，   \(z_y=-\dfrac{2yz+2}{x^2+y^2+1/z}\)

极值点像山顶或山谷：朝 x 方向看是平的，朝 y 方向看也必须是平的，所以令 \(z_x=0\)、\(z_y=0\)。由于分母已经证明不为 0，只需让两个分子等于 0：

\(2xz+2=0\)，\(2yz+2=0\)
\(\Rightarrow xz+1=0\)，\(yz+1=0\)
\(\Rightarrow x=y=-\dfrac1z\)

第三步：代回原方程，求出驻点

“偏导等于 0”只给出了 x、y、z 的关系，还没给出具体坐标，所以必须代回原约束方程：

\(x=y=-\dfrac1z\)
\(\Rightarrow \dfrac{2}{z}+\ln z+2\left(-\dfrac2z+1\right)=0\)
\(\Rightarrow \ln z+2-\dfrac2z=0\)

设 \(\varphi(z)=\ln z+2-2/z\)（φ 是关于 z 的一元函数）。在 \(z>0\) 时：

\(\varphi'(z)=\dfrac1z+\dfrac{2}{z^2}>0\)，并且 \(\varphi(1)=0\)

φ 在 \(z>0\) 上严格递增，所以它至多有一个零点；又因为 \(z=1\) 正好让 φ 等于 0，因此唯一解是 \(z=1\)。再由 \(x=y=-1/z\)，得到唯一驻点：

\((x,y)=(-1,-1)\)，对应 \(z=1\)

第四步：用二阶偏导判断它是山顶还是山谷

在驻点处 \(z_x=z_y=0\)。对关系 \(F_x+F_z z_x=0\) 再求导时，所有带 \(z_x\)、\(z_y\) 的项都会变成 0，于是：

\(z_{xx}=-\dfrac{F_{xx}}{F_z}\)，   \(z_{xy}=-\dfrac{F_{xy}}{F_z}\)，   \(z_{yy}=-\dfrac{F_{yy}}{F_z}\)

本题有 \(F_{xx}=2z\)、\(F_{xy}=0\)、\(F_{yy}=2z\)。在 \((-1,-1,1)\) 处，\(F_z=1+1+1=3\)，因此：

\(A=z_{xx}=-\dfrac23\)，  \(B=z_{xy}=0\)，  \(C=z_{yy}=-\dfrac23\)
\(AC-B^2=\dfrac49>0\)，且 \(A<0\)

判别式为正且 A 为负，说明曲面在这个点朝两个方向都向下弯，所以这里是极大值点。

🎯 最终答案：z(x,y) 在 \((x,y)=(-1,-1)\) 处取得极大值 \(z_{\max}=1\)，没有极小值。

🧠 理解类比：把方程 F=0 想成三个人共同维持平衡的跷跷板。x 动一下，z 必须自动补偿；公式 \(z_x=-F_x/F_z\) 中的负号，就是“这边上去，那边要下来”。
🧠 记忆类比：把 z=f(x,y) 想成一座山。\(z_x=0\) 只说明东西方向是平的，\(z_y=0\) 只说明南北方向是平的；两者同时为 0，才有资格接受“山顶还是山谷”的二阶判定。
🎯 出题人视角：题目故意不让你解出 z，是在筛查三种能力：能否把方程改写成 F=0；能否用隐函数公式求两个偏导；能否记得“找到驻点之后还要判别”。只做到 \(z_x=z_y=0\) 只能拿过程分，不能直接宣布极大值。
一题两条路：局部判别与整体配方
① 标准考研路线：隐式求偏导 → 解驻点 → 算 \(A,B,C\) → 用 \(AC-B^2\) 判极大值。优点是步骤标准、按点给分。
② 一眼看穿路线：把原式配成 \(z[(x+1/z)^2+(y+1/z)^2]+\ln z+2-2/z=0\)。第一项不小于 0，而 \(\ln z+2-2/z\) 严格递增且在 z=1 时等于 0，所以必有 \(z\le1\)；等号只在 \((x,y,z)=(-1,-1,1)\) 成立。这说明 1 不只是局部极大值，还是全局最大值。
隐函数极值题的四个坑
  • 忘记定义域：看到 \(\ln z\) 必须先写 \(z>0\)。
  • 把 z 当常数到底：计算 \(F_x\) 时暂时固定 z；但对原方程整体求导时，z 是 \(z(x,y)\)，会产生 \(z_x\) 或 \(z_y\)。
  • 只令分子为 0，却不查分母：必须先说明 \(F_z=x^2+y^2+1/z>0\)。
  • 找到驻点就停笔：驻点只是“候选人”，还要用二阶偏导判极大、极小或非极值。
隐函数极值题的卷面四步
① 建模：令 \(F(x,y,z)=0\)，写出 \(z>0\) 与 \(F_z\ne0\)；
② 一阶：写 \(z_x=-F_x/F_z\)、\(z_y=-F_y/F_z\)，再令二者为 0；
③ 解点：把驻点条件代回原方程，解出 \((x,y,z)\)；
④ 判型：计算 \(A=z_{xx},B=z_{xy},C=z_{yy}\)，写清 \(AC-B^2\) 与 A 的符号，最后单独写结论。
得分节奏：公式与一阶偏导 → 驻点 → 二阶判别 → 极值结论，四段缺一不可。
固定 2 米铁丝，怎样分才最省面积
为什么要学它？现实中的预算、材料和时间通常都是固定的，难点不是“能不能做”，而是“怎么分配最划算”。这道题把 2 米铁丝当作固定预算，把三个图形的面积当作总成本，正是拉格朗日乘数法最典型的资源分配模型。
还记得 ch02 里一元函数求最值要令导数等于 0 吗？这里也是找“导数为 0”，只是多了三个变量和一条总长度约束，所以要请 λ 帮咱们把约束拴在目标函数上。
📖 必背：这道题进场前的工具箱
工具公式小白翻译
圆周长 \(l=2\pi r\)，面积 \(S=\pi r^2\)先用周长求半径，再算面积
正方形周长 \(l=4a\)，面积 \(S=a^2\)边长等于周长除以 4
正三角形周长 \(l=3a\)，面积 \(S=\sqrt3a^2/4\)边长等于周长除以 3

📝 真题 4 | 2018 数一、二、三 | 拉格朗日乘数法
将长为 2 m 的铁丝分成三段，依次围成圆、正方形、正三角形。三个图形面积之和是否存在最小值？若存在，求出最小值。

🎯 先抓矛盾：不是平均分，而是看“面积增长速度”

三个图形虽然共用 2 m 铁丝，但相同长度围成不同图形时，面积并不相同。所以直接把铁丝平均分成三份没有依据；咱们要找的是一组长度，使得再从任意一段挪一点铁丝给另一段，都不会让总面积继续下降。

第一步：设三段长度，并写出约束

设围成圆、正方形、正三角形的铁丝长度依次为 \(x,y,z\)（单位都是 m）。因为整根铁丝长 2 m，所以：

\(x+y+z=2\)，且 \(x>0,\ y>0,\ z>0\)

这里 x、y、z 表示的是三个图形的周长，不是半径或边长，这是建模时最容易看错的地方。

第二步：把三个周长翻译成三个面积

圆：周长是 x，所以半径 \(r=x/(2\pi)\)。代入圆面积公式：

\(S_1=\pi\left(\dfrac{x}{2\pi}\right)^2=\dfrac{x^2}{4\pi}\)

正方形：周长是 y，所以边长 \(a=y/4\)。因此：

\(S_2=\left(\dfrac y4\right)^2=\dfrac{y^2}{16}\)

正三角形：周长是 z，所以边长 \(b=z/3\)。因此：

\(S_3=\dfrac{\sqrt3}{4}\left(\dfrac z3\right)^2=\dfrac{\sqrt3}{36}z^2\)

三个面积相加，得到本题要最小化的目标函数：

\(S(x,y,z)=\dfrac{x^2}{4\pi}+\dfrac{y^2}{16}+\dfrac{\sqrt3}{36}z^2\)

第三步：构造拉格朗日函数

目标是让 S 最小，但 x、y、z 不能随便取，它们必须满足 \(x+y+z-2=0\)。于是把约束乘上 λ（拉格朗日乘子）并加到目标函数中：

\(L(x,y,z,\lambda)=\dfrac{x^2}{4\pi}+\dfrac{y^2}{16}+\dfrac{\sqrt3}{36}z^2+\lambda(x+y+z-2)\)

λ 不代表新的铁丝长度，它只是一个“协调员”，负责保证求极值的全过程不离开总长度为 2 的约束。

第四步：四个偏导同时等于 0

对 x、y、z、λ 分别求偏导，得到：

\(L_x=\dfrac{x}{2\pi}+\lambda=0\)
\(L_y=\dfrac{y}{8}+\lambda=0\)
\(L_z=\dfrac{\sqrt3}{18}z+\lambda=0\)
\(L_\lambda=x+y+z-2=0\)

前三式都等于同一个 \(-\lambda\)。令 \(k=-\lambda\)，就得到：

\(\dfrac{x}{2\pi}=\dfrac{y}{8}=\dfrac{\sqrt3z}{18}=k\)
\(\Rightarrow x=2\pi k,\quad y=8k,\quad z=6\sqrt3k\)

再把它们代入总长度约束：

\(2\pi k+8k+6\sqrt3k=2\)
\(\Rightarrow 2k(\pi+4+3\sqrt3)=2\)
\(\Rightarrow k=\dfrac1{\pi+4+3\sqrt3}\)

因此三段铁丝应该分成：

\(x=\dfrac{2\pi}{\pi+4+3\sqrt3}\ \text{m}\)（围圆）
\(y=\dfrac{8}{\pi+4+3\sqrt3}\ \text{m}\)（围正方形）
\(z=\dfrac{6\sqrt3}{\pi+4+3\sqrt3}\ \text{m}\)（围正三角形）

第五步：算最小值，并回答“是否存在”

为了不把同一个长式子写很多遍，记 \(D=\pi+4+3\sqrt3\)。把 \(x=2\pi/D\)、\(y=8/D\)、\(z=6\sqrt3/D\) 代入总面积：

\(S=\dfrac{\pi}{D^2}+\dfrac4{D^2}+\dfrac{3\sqrt3}{D^2}\)
\(=\dfrac{\pi+4+3\sqrt3}{D^2}=\dfrac1D\)

还要说明为什么它真的是最小值，而不只是一个候选点。把三角形面积项改写为 \(\sqrt3z^2/36=z^2/(12\sqrt3)\)，由柯西不等式：

\(S\ge\dfrac{(x+y+z)^2}{4\pi+16+12\sqrt3}\)
\(=\dfrac4{4(\pi+4+3\sqrt3)}=\dfrac1{\pi+4+3\sqrt3}\)

上面求出的 x、y、z 恰好能让等号成立，而且三段长度都大于 0，所以这个下界能够真正取到，最小值确实存在。

🎯 最终答案：三个图形的面积总和存在最小值；当三段长度之比为 \(x:y:z=\pi:4:3\sqrt3\) 时取得，最小值为 \(S_{\min}=\dfrac1{\pi+4+3\sqrt3}\ \text{m}^2\)。

🧠 理解类比：把 2 m 铁丝想成 2 元预算，圆、正方形、正三角形是三个“花钱速度不同”的项目。最优分配不是每个项目分一样多，而是让三个项目“再多给 1 cm 所增加的面积”相同，这正是三个偏导都等于 \(-\lambda\) 的含义。
🧠 记忆类比：把 λ 想成拴住三段铁丝的一根总绳。x、y、z 可以互相挪长度，但总绳始终锁死在 2 m；写拉格朗日函数时，就是“目标函数 + λ×(总量−2)”。
🎯 出题人视角：这道题表面考几何公式，真正筛查的是“把生活语言翻译成目标函数与约束”的建模能力。最容易拉开分差的不是求偏导，而是能否正确认出 x、y、z 是周长，并在最后证明候选值确实是最小值。
铁丝分配题的通用流水线
① 设长度：每个去向设一个变量；
② 写约束：所有长度相加 = 总长度；
③ 翻译目标：先由周长求半径或边长，再写面积和；
④ 乘 λ：构造“目标函数 + λ×约束”；
⑤ 联立求解：所有偏导与约束同时等于 0；
⑥ 证明最小：用正定二次函数、柯西不等式或题目给定范围说明最小值能够取到。
铁丝分段题的五个易错点
  • 把周长当边长：x、y、z 是三段铁丝长度，也就是三个图形的周长。
  • 圆面积漏掉 π：\(r=x/(2\pi)\)，所以面积是 \(x^2/(4\pi)\)，不是 \(x^2/(4\pi^2)\)。
  • 三角形系数算错：边长是 \(z/3\)，面积是 \(\sqrt3z^2/36\)。
  • 想当然平均分：三个面积公式的系数不同，最优分配不可能直接写成 \(2/3,2/3,2/3\)。
  • 只求驻点不判最小：必须补一句“下界能够取到”或说明目标函数严格凸。
条件极值应用题的卷面五段
第 1 段：设变量并写 \(x+y+z=2\)；
第 2 段：分别写三个面积，再得到目标函数 S；
第 3 段：构造 L，完整列出四个偏导方程；
第 4 段：解出三段长度并代回计算面积；
第 5 段：说明最小值存在，带单位写最终结论。
阅卷重点：建模、拉格朗日方程组、长度比例、最小值与存在性，五项都要落在卷面上。
