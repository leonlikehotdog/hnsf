第十三章：特征值与特征向量 | 6分
⭐⭐⭐⭐⭐

🎯 核心考点

  • 特征值、特征向量的计算
  • 相似对角化
📖 必背：特征方程
|A - λE| = 0

解此方程得特征值 λ，然后解 (A - λE)x = 0 得特征向量

📖 必背：特征值性质
  • 所有特征值之和 = 矩阵的迹（主对角线元素之和）
  • 所有特征值之积 = 行列式 |A|
  • 若 λ 是 A 的特征值，则 λk 是 Ak 的特征值
📝 例题
求矩阵 A = [2 1; 1 2] 的特征值和特征向量
特征方程 |A-λE| = |2-λ 1; 1 2-λ| = (2-λ)² - 1 = 0
解得 λ₁ = 3，λ₂ = 1。
λ=3 时：(A-3E)x = 0 ⟹ [-1 1; 1 -1]x=0 ⟹ x₁=x₂，取 ξ₁=(1,1)T
λ=1 时：(A-E)x = 0 ⟹ [1 1; 1 1]x=0 ⟹ x₁=-x₂，取 ξ₂=(1,-1)T
所以特征值为 3 和 1，对应特征向量为 (1,1)T 和 (1,-1)T

📝 真题 1 | 2023 数一/二 | 解答题 · 12分
设 \(A=\begin{pmatrix}1&-1&1\\2&4&-2\\-3&-3&5\end{pmatrix}\)，求 \(A\) 的特征值和特征向量。

🎯 出题人考察点（这是高频考点！）

① 特征多项式计算；② 特征向量求解；③ 行最简形化简

🧭 详细解析（小白逐步版）

特征多项式 \(|\lambda E-A|=(\lambda-2)(\lambda-3)(\lambda-5)\)。特征值 \(\lambda_1=2,\lambda_2=3,\lambda_3=5\)。对应特征向量分别 \((1,-2,3)^T,(-1,1,0)^T,(1,-2,1)^T\)。

① 特征多项式 (5分)；② 解特征值 (2分)；③ 特征向量 (5分)
