基础概念与重要极限
等价无穷小公式表 + 两个重要极限 + 1 个标准证明
📖 必背：等价无穷小公式表（x→0时）
公式等价于
sin x\(\sim x\)
tan x\(\sim x\)
arcsin x\(\sim x\)
arctan x\(\sim x\)
ln(1+x)\(\sim x\)
ex - 1\(\sim x\)
1 - cos x\(\sim x^{2}/2\)
(1+x)α - 1\(\sim \alpha x\)
📖 必背：两个重要极限
① limx→0 sin x / x = 1
② limx→∞ (1 + 1/x)x = e
📐 证明：limx→∞ (1 + 1/x)x = e

思路：幂指函数取对数变乘法，洛必达搞定。

① 令 y = (1 + 1/x)x，取自然对数：
ln y = x · ln(1 + 1/x) = ln(1 + 1/x) ÷ (1/x)
② 当 x → ∞ 时，1/x → 0，所以分子 ln(1+1/x) → ln 1 = 0，分母 1/x → 0，0/0 型未定式，用洛必达：
分子求导：d/dx[ln(1 + 1/x)] = 1/(1 + 1/x) · (−1/x²) = −1/(x² + x)
分母求导：d/dx[1/x] = −1/x²
所以 lim ln y = lim [−1/(x²+x)] / [−1/x²] = lim x²/(x²+x) = lim 1/(1 + 1/x) = 1
③ 因为 ln y → 1，所以：
y = eln y → e1 = e ∎

小白记忆：「幂指函数，先取对；化成 0/0，洛必达收；对数极限是 1，e 的 1 次方就是 e」。

跨章节工具箱（含变限积分 / 积分中值）
打 ch03 积分基础的底，是后面所有 0/0 型极限的前置工具
📖 高频题型：含变限积分的 0/0 型极限ch03 积分

识别信号：极限的分子或分母含有形如 ∫axf(t)dt 的积分，并且代入极限点后得到 0/0 型。

处理口诀：先判断型，再看结构；能等价就等价，能换元就换元，最后再用洛必达。

方法适用场景小白理解
洛必达法则分子、分母均趋于0，且求导后明显变简单把“积分比值”变成“导数比值”
积分中值定理被积函数连续，积分区间较简单把积分近似看成“区间长度 × 区间内某点的函数值”
换元或等价无穷小被积函数中同时含 x、t，或能提取 x 的幂次先把积分区间固定，再直接比较最高阶
📖 必背：先分清 x、t、u 是谁ch03 积分
字母身份小白理解
x外部变量，也是最后要趋近 0 的变量x 决定积分上限，整道题最终对 x 求极限或求导
t积分变量（哑变量）t 只在积分号里面活动；积分算完后，t 就消失了
u换元后新引入的积分变量用 u 替换原来的 t，积分结束后 u 也会消失

关键：∫0xf(t)dt 是关于 x 的函数。虽然积分里面写的是 t，但上限 x 在变化，所以整个积分值也会跟着 x 变化。

📖 必背：变限积分求导（微积分基本定理）ch03 积分
ddx∫axf(t)dt=f(x)

为什么？上限 x 增加一点点 Δx，积分面积大约增加“高度 f(x) × 宽度 Δx”，所以变化率就是 f(x)。

ddx∫u(x)v(x)f(t)dt=f(v)v′−f(u)u′

记忆口诀：上限代入乘上限导数，减去下限代入乘下限导数。

注意：若被积函数本身也含 x，例如 ∫0x√(x-t)etdt，就不能只把上限 x 代进去。对小白来说，优先整体换元，把根号中的 x-t 消掉，再求导最稳。

题型一：含变限积分的极限
整体代换 + 洛必达 / 积分中值定理 / 等价无穷小快速套用
📝 视频题型：整体代换 + 洛必达
求
limx→0+∫0xx−tetdtx3

第一步：代入判断类型

x→0+ 时，积分区间 [0,x] 缩成一个点，所以分子→0；分母 x3/2→0。这是 0/0 型，可以考虑洛必达。

第二步：为什么不能马上洛必达

分子中 x 出现了两次：既是积分上限，又藏在 √(x-t) 中。直接求导要使用含参数积分的复杂公式，而且把 t=x 代入根号时还会碰到奇异形式。视频因此先做整体代换。

第三步：看到 x-t，就令 u=x-t

令 u=x-t，则 t=x-u，dt=-du。上下限也必须跟着变：

  • 原来 t=0 时，u=x；
  • 原来 t=x 时，u=0。
∫0xx−tetdt=∫x0uex−u(−du)=ex∫0xue−udu

这一步的目标：把 √(x-t) 变成 √u，使积分内部不再混着 x 和 t；同时 ex-u=exe-u，可以把与 u 无关的 ex提到积分号外。

第四步：拆掉趋于非零常数的因子

limx→0+ex∫0xue−udux32=1·limx→0+∫0xue−udux32

因为 ex→1，它不是导致 0/0 的核心，可以先放在一边。真正需要处理的是后面的积分与 x3/2之比。

第五步：洛必达，变限积分直接求导

limx→0+xe−x32x=limx→0+e−x32=23
最终答案是 2/3。破题链条只有五步：代入判型 → 发现积分内外都含 x → 用 u=x-t 整体换元 → 去掉趋于1的 ex → 对变限积分使用洛必达。
📝 同一道题的第二种解法：积分中值定理

先理解它在做什么

原积分里有两个相乘的部分：

∫0→xx−t保留并积分·et取某个代表值dt

当 x 很小时，t 只能在很短的区间 [0,x] 内活动，因此 et 在这段区间内变化很小。积分中值定理告诉我们：可以在区间内找到某个点 ξ，使整个积分的效果等于“把 et 固定为 eξ，再对剩余部分积分”。

📖 必背：积分中值定理（加权形式）ch03 积分

若 f(t) 连续，g(t) ≥ 0 且可积，则存在 ξ∈[a,b]，使：

∫abf(t)g(t)dt=f(ξ)∫abg(t)dt

小白理解：f(t) 在区间里会变化，但一定能找到一个“平均代表点”ξ，使 f(ξ) 代表它在这次加权积分中的整体效果。ξ 不需要求出来，只需知道它在积分区间内。

第一步：谁提出来，谁留下

本题令 f(t)=et，g(t)=√(x-t)。在固定一个 x 后，√(x-t)≥0，因此可用积分中值定理：

∫0xx−tetdt=eξ∫0xx−tdt,0≤ξ≤x

为什么提 et，不提根号？因为留下来的 √(x-t) 很容易直接积分；如果反过来留下 et，反而不能简化问题。

第二步：计算剩下的简单积分

令 u=x-t，则 du=-dt：

∫0xx−tdt=∫0xudu=23x32

于是原积分就等于：

∫0xx−tetdt=eξ·23x32

第三步：代回原极限并约分

limx→0+eξ·23x32x32=limx→0+23eξ

第四步：为什么 ξ→0

积分中值定理只告诉我们 0≤ξ≤x。现在 x→0+，ξ 被夹在 0 和 x 之间，根据夹逼定理，ξ→0。因此 eξ→e0=1。

limx→0+23eξ=23·1=23
第二种解法的破题思路：看到“容易积分的非负函数 × 在小区间内变化不大的连续函数”，就尝试积分中值定理。把难处理但变化平缓的 et 变成 eξ，留下容易积分的 √(x-t)，最后用 0≤ξ≤x 推出 ξ→0。注意：ξ 只是定理保证存在的代表点，不用也无法具体求出。
limx→0 (sin 3x) / (2x)
sin 3x ~ 3x → 原式 = 3x / 2x = 3/2。直接套等价无穷小即可，连洛必达都不用。
limx→0 (1 − cos x) / x²
1 − cos x ~ x²/2 → 原式 = (x²/2) / x² = 1/2。记住「1−cos x 是二阶无穷小」是关键。
极限计算的 5 种武器
① 等价无穷小：分子分母能替换时优先用，最快；
② 洛必达：0/0 或 ∞/∞ 时的万能钥匙；
③ 泰勒展开：分子分母都复杂时，避免求导；
④ 夹逼定理：看到「在区间里变化不大」的函数；
⑤ 积分中值定理：连续函数 × 非负函数的乘积积分。

题型二：∞ − ∞ 型未定式
通分 / 泰勒展开 / 洛必达三种武器并用
🎬 视频新题型：∞ − ∞ 型未定式极限（2021 数一）
求
limx→01+∫0xet2dtex−1−1sinx

🧭 破题第一步：先判型

把 x=0 直接代入分子分母看：

  • 分子：\(\displaystyle 1 + \int_{0}^{0} e^{t^{2}}\,dt = 1 + 0 = 1\)
  • 分母：\(\displaystyle e^{0} - 1 = 0\)，所以前半段是 \(\displaystyle \frac{1}{0} \to \infty\)。
  • 后半段：\(\displaystyle \frac{1}{\sin x} \to \frac{1}{0} \to \infty\)。

整体是 \(\infty - \infty\) 型，公式表里没有现成结论，必须先变形。视频里给出 5 种常用方法，我们从最稳的「通分」入手。

🧭 破题第二步：方法一 —— 通分 + 洛必达

「∞ − ∞」最经典的思路是 通分变成 \(\dfrac{0}{0}\) 或 \(\dfrac{\infty}{\infty}\)，再用洛必达。本题把两个分式写成公分母 \(\displaystyle (e^{x}-1)\cdot\sin x\)：

原式=limx→0(1+∫et2dt)·sinx−(ex−1)(ex−1)·sinx

分子分母同时趋于 0（因为 sin x → 0，ex−1 → 0，积分 → 0），化成了 0/0，可以洛必达。

这里有个小坑：分子里既有 sin x 也有积分，求导后会出现 4 项，别怕，一步一步写：

  • 1·sin x 求导 → cos x
  • sin x · ∫0xet²dt 求导 → cos x · 积分 + sin x · ex²
  • − (ex−1) 求导 → − ex

把分子合成（用 ex−1 ~ x，sin x ~ x 把它化为 2x² 形式），最终结果是 1/2。

🧭 破题第三步：方法二 —— 泰勒展开（避免大段求导）

这道题涉及 变限积分、ex−1、sin x 三个不同函数，通分后洛必达要展开 4 项，很容易写错加减号。泰勒展开是更稳的偷懒法：

📖 必背：常用泰勒展开（x→0）
sin x = x − x³/6 + O(x⁵)
ex = 1 + x + x²/2 + O(x³)
et² = 1 + t² + O(t⁴)

技巧：展开到和分母同阶即可。本题分母是 x·x = x²，所以分子展开到 x² 项就停。

把每一项用泰勒展开代回去：

  • ∫0xet²dt = ∫0x(1 + t² + ...) dt = x + x³/3 + ...
  • ex − 1 = x + x²/2 + ...
  • sin x = x − x³/6 + ...

代回原式，分子最高阶 x² 项的系数是 (1 − 1/2) = 1/2，除以分母 x²，得到 1/2。

✅ 三种解题方法汇总（视频里给出的 5 种思路里最推荐 3 种）：
① 通分 + 洛必达：通用，但容易在「积分求导 × 三角函数」交叉项中写错符号；
② 通分 + 泰勒展开：能跳过求导，但需要准确背出三大展开；
③ 提公因子 + 洛必达：把 sin x 提取出来，使分子只剩两项，比方法①少一半书写量。
小白建议优先掌握方法②，配上下面的「防粗心 7 条军规」最稳。

防粗心 7 条军规
针对泰勒太长、加减号写错、积分上下限反等问题
🛡️ 防粗心 7 条军规（针对泰勒太长、加减号写错、积分上下限反等问题）
  1. 动笔前先圈关键词：看到 \(\displaystyle e^{x}-1\)、\(\displaystyle 1-\cos x\)、\(\displaystyle \ln(1+x)\) 这种能展开的，先在旁边写「≈ ?x + ?x²」，提醒自己别只想到一次项。
  2. 分母是 0 时立即用等价无穷小替换，不要硬算洛必达。例如 \(\displaystyle e^{x}-1 \approx x\)，\(\displaystyle \sin x \approx x\)，整个分母立刻变 x²。
  3. 分子分母同阶才有效。如果分母是 x³，分子却只展开到 x²，最后一项会被吞掉，答案偏大或偏小就是这个原因。
  4. 展开前先口算「目标阶数」：例如分母是 x²，目标就是 x² 项；展开到 x³ 就是浪费时间还容易写错。
  5. 所有 ± 号都画框：尤其是「负负得正」「− 1 − 1 = −2」这种，建议把负号写大一点（如 −1）。
  6. 积分上下限变换后写「原上限\(\to\)新上限」的小表，防止换元后上下限反了。
  7. 做完一步立刻反代验证：把算出来的极限代回原始式，例如 x=0.01，看分子分母是不是真的都趋于 0，且比值接近答案。

📌 形成肌肉记忆的方法：每次做错一道题，在题目旁边写下「错在第 ? 条军规」，一周后回看，你会发现 80% 的计算错误都集中在 1~3 条上。

题型三：1^∞ 型未定式
万能公式法 / 拍底法 + 泰勒展开，12 分大题常客

🎯 核心考点 12 分大题 1^∞ 型 万能公式 拍底法 泰勒展开

📖 必背：1^∞ 型极限的「长相识别」与「总体思路」

\(1^{\infty}\) 长什么样？想象你买理财：今天 1 块钱的本金，连续利滚利很多很多年，本金一直保持「接近 1」，但年数「趋于无穷」，这就是 \(1^{\infty}\)。它既不是确定的 1，也不是确定的 \(\infty\)，是一个卡在中间的「未定式」。

用数学语言表达，只要满足下面 两条 同时成立，题目就是 1^∞ 型：

条件人话小白例子
底数 \(\displaystyle f(x) \to 1\) 那个长得像「数字」的玩意儿，趋近于 1 \(\displaystyle 1.01\)、\(\displaystyle 0.99\)、\(\displaystyle 1.00001 \to 1\) 都算「接近 1」
指数 \(\displaystyle g(x) \to \infty\) 那个右上角小小的家伙，趋近于无穷大 \(\displaystyle 100\)、\(\displaystyle x^{4}\) 这种 → 越来越大

总体思路（最重要的一句）：所有 1^∞ 型极限，结果一定是 \(\displaystyle e^{\text{某个数}}\)。咱们只要求出「e 上方那个数」就行。下面两种方法殊途同归，都能把这个数算出来。

🧠 小白记忆类比：把 1^∞ 比作「本金几乎不动、但时间疯长」的复利，本金是「f(x)」、年数是「g(x)」。时间一长，1.01 这种基础也会膨胀一点点，所以最终答案一定带 e（自然对数的底 = 复利的极限形态）。
📖 必背：1^∞ 极限的两种通用工具（法一 法二）

法一：万能公式法（最省事，绝大多数 \(1^{\infty}\) 首选）

limx→x₀ f(x)g(x) （1^∞ 型）
= e limx→x₀ g(x) · ( f(x) − 1 )

公式长什么样？左边那个复杂的 f^g，右边只剩「g 乘 (f-1)」—— 几乎所有 1^∞ 都能丢到 e 里面去求。它的灵魂只有一句话：当 f 接近 1 时，f^g ≈ e^{g·(f-1)}，所以极限 = e^{那个 g·(f-1) 的极限}。

法二：拍底法（万能公式用不了时再上，对数登场）

A = eln A   ,   AB = eB · ln A

公式的灵魂：任何「某个东西的某次方」都能拍到底——拍成 e 的「B × ln A」次方。原本要算 f^g，现在只需要算 lim B·ln A。当底数不是干净的 1-ε 形式时，拍底法比万能公式更灵活。

两种方法的关系：

  • 万能公式是「ln(1+u) ≈ u」+ 「拍底」的结晶。如果底数很干净、容易展成「1 + 小量」，优先法一。
  • 如果底数 f(x) 很复杂、里面还套了 log、arctan 之类，直接法二，再用 ln(1+u) ≈ u 把 ln f 化简成 f−1，最终仍然回到「g·(f−1)」。
📝 视频题型：[2016 数二、三] 未定式 1^∞ 型求极限
求
limx→0(cos2x+2xsinx)1x4

🧭 第一步：先判型 → 1^∞

把 x=0 直接代入底数和指数看：

  • 底数：cos 2x + 2x sin x → cos 0 + 0 = 1；
  • 指数：1/x⁴，当 x→0 时分母越来越小 → \(\infty\)。

完美符合 1^∞ 型，套用上方的两种方法都可以。我们先走万能公式（法一），再走拍底法（法二）。

🧭 法一：万能公式法（推荐）

套万能公式，记成一句话「e 上方等于 g 乘 (f−1)」：

原式 = e limx→0 ( 1/x⁴ ) · ( cos 2x + 2x sin x − 1 )

现在的任务：求 lim \((cos 2x + 2x sin x − 1) / x⁴\)。别一上来就洛必达求四次导！分母是 x⁴，分子展开必须到 x⁴ 阶，否则抓不到答案。直接用泰勒展开：

  • cos 2x = 1 − (2x)²/2! + (2x)⁴/4! + O(x⁶) = 1 − 2x² + (2/3)x⁴ + O(x⁶)
  • 2x sin x = 2x · ( x − x³/3! + O(x⁵) ) = 2x² − (1/3)x⁴ + O(x⁶)

两式相加，再减去开头那个 1：

cos 2x + 2x sin x − 1
= (1 − 2x² + (2/3)x⁴) + (2x² − (1/3)x⁴) − 1 + O(x⁶)
= (2/3 − 1/3)x⁴ + O(x⁶) = (1/3)x⁴ + O(x⁶)

注意看 —2x² 和 +2x² 完美对消，只剩 x⁴ 项。这就是为什么要展到 4 阶：分母是 x⁴，分子常数项消掉之后，4 阶项刚好决定胜负。代入：

limx→0 (1/x⁴) · ( (1/3)x⁴ + O(x⁶) ) = 1/3

所以原式 = e1/3。方法一结束。

🧭 法二：拍底法（对比验证）

换一条路。拍底口诀「A^B = e^{B · ln A}」：

原式 = e limx→0 (1/x⁴) · ln(cos 2x + 2x sin x)

现在的任务：求 \(\displaystyle \lim_{x \to 0} \frac{\ln(\cos 2x + 2x\sin x)}{x^{4}}\)。

关键观察：x→0 时 cos 2x + 2x sin x → 1，所以 ln(⋯) 里的东西接近 1，套 \(\displaystyle \ln(1+u) \approx u\)（当 \(\displaystyle u \to 0\)）：

ln(cos 2x + 2x sin x) ≈ cos 2x + 2x sin x − 1

这就把问题又变成了法一同样的 g·(f−1)！再用一次泰勒展开：

limx→0 (cos 2x + 2x sin x − 1) / x⁴ = 1/3

所以原式 = e1/3。两种方法殊途同归。

🧭 第三步：为什么 ln(1+u) ≈ u 这个替换能省这么多事

原始极限里如果直接对 ln(cos 2x + 2x sin x) 跑洛必达，要先对 cos 2x + 2x sin x 求导（出现 −2sin 2x + 2sin x + 2x cos x），再对 1/x⁴ 求导，要洛必达多次。但只要记住「ln(1+u) ≈ u」，就能直接把 ln 替换掉，一步回到 x⁴ 形式，再泰勒展开即可。这是从 ch01 第三个必背军规「能用等价就用等价」在 1^∞ 上的应用。

🧠 小白记忆类比：把 1^∞ 比作「登山」—— 底部基础稳稳接近 1（平地），但要爬到无穷高（指数）。在山脚看，每一步都是「平地 + 一小段坡」，爬升慢慢累加；这个慢慢累加的过程就是 \(\displaystyle g\cdot(f-1)\)，最终汇聚到山顶 = \(\displaystyle e^{\,g\cdot(f-1)}\)。两种方法（法一用万能公式、法二用拍底 + 等价）就像坐缆车和徒步登顶，山顶一样，路径不同。
1^∞ 极限的万能三步走
看到 1^∞ 别慌，照这三步走就行：
① 判型：把 x₀ 代入底数 → 1，指数 → ∞，才用得上这两招；
② 套公式或拍底：底数干净 → 万能公式；底数套了 log/arctan → 拍底法；
③ 化简指数：本质都是求 lim g·(f−1)，这一步用泰勒展开（展到与 g 同阶）或洛必达。
小白口诀：判型 \(\to\) 套招 \(\to\) 展阶。
万能公式法 vs 拍底法 选用对照
① 万能公式（首选）：底数是多项式 / 三角函数 / 简单复合，直接套 e^{ lim g·(f−1) }，省字；
② 拍底法（备胎）：底数里套了 log / arctan / 指数函数，先拍底再用 ln(1+u)≈u 化简，更灵活；
③ 指数内部最终都是求 lim g·(f−1)，这一步统一用泰勒展开（多变量版），展到与 g 同阶；
④ 何时改用洛必达：分子分母都有 log、arctan、sec 这种不好展开的函数时，洛必达更稳，但要小心复合求导；
⑤ 判断套路是否真的 \(1^{\infty}\)：很多看似 \(f \to 1\)、\(g \to \infty\) 的题其实底数减 1 是 \(x^{2}\) 量级，先算清楚阶数再决定要不要展到哪一阶。
1^∞ 极限 6 大踩坑点
  • 判型判错：底数最终是 1 但指数不是 ∞，或反过来 — — 别硬套万能公式，先把 x₀ 代一次确认。
  • 展开阶数不够：分母 x⁴ 只展到 x² → 抓不到答案，结果永远是 1（错）。
  • 展开阶数太多：展到 x⁶/x⁸ 写不完，还容易加错；和分母同阶就停手。
  • ln(1+u) ≈ u 滥用：只有当括号里「→ 1」时才能换，不能 ln(2+x) ≈ 2+x。
  • cos 2x 用错公式：cos(2x) ≠ 2cos²x − 1 展开；这里用 cos 2x = 1 − (2x)²/2! + (2x)⁴/4! + … 更直接。
  • 忘了拍底后还要算 lim：原式 = e^{B·ln A} 后，e 上方必须真的算出极限，不能留在 lim 之外。

📖 工具箱前置①：拉格朗日中值定理（从零复习）ch02 导数

先忘掉公式，听听这个故事：

你从家到公司，开车走了 10 公里，花了 20 分钟。你的平均速度是 30 km/h。拉格朗日中值定理说：这 20 分钟里，至少有一个瞬间你的车速表恰好指在 30 km/h 上。不多不少，正好等于平均速度。

若 f(x) 在 [a,b] 上连续，在 (a,b) 内可导
则存在 ξ ∈ (a,b)，使 f(b) − f(a) = f'(ξ)·(b−a)

拆开来看：

  • f(b) − f(a) = 从 a 到 b 的总变化量（相当于 10 公里）
  • b − a = 走过的区间长度（相当于 20 分钟）
  • f'(ξ) = 中间某个 ξ 点的瞬时变化率（相当于车速表某刻的读数）

重要的事说三遍：你不需要知道 ξ 具体是多少！你只需要知道 ξ 在 a 和 b 之间。比如在 [0, x_n] 上，ξ 就在 0 和 x_n 之间 → 如果 x_n > 0，那 ξ 就在 (0, x_n) 里，是个正数。

拉格朗日最常见的用法（今天这道题就用这个）：

当碰到 \(\displaystyle \frac{e^{b}-e^{a}}{b-a}\) 这种"差÷长"结构时，直接看成 \(\displaystyle e^{\xi}\)。因为对 \(\displaystyle f(t)=e^{t}\) 用拉格朗日，正好等于 \(\displaystyle e^{\xi}\)。

🧠 小白记忆类比①（理解用途）：拉格朗日就像"交通调度员"——你想算两地的平均速度，调度员说"不用算全程，我帮你找一个中间时刻的瞬时速度来代表平均值"。
🧠 小白记忆类比②（记忆公式）：f(b)−f(a) = f'(ξ)·(b−a) → "总变化 = 某点变化率 × 全程长度"，就像"总流量 = 平均流速 × 水管长度"。

📖 工具箱前置④：用定积分定义求数列极限（小白从零版）

🎯 出题人意图拆解

出题人每年都在这儿等着你：数列极限里 \(n\) 项求和、\(n\) 项连乘，这种「\(n\) 越大、求和项越多」的结构，99% 是在考你能不能认出定积分。为什么？因为「无穷多个无穷小量求和」在数学上就叫做定积分，这是定积分最本质的来源。

📚 小白扫盲：定积分是个啥？

一句话定义：定积分 \(\int_{a}^{b} f(x)\,dx\) = 在区间 \([a,b]\) 上无数个「细长矩形」面积累加起来。

生活例子🌰：想象你在一块弯曲的菜地（曲边梯形）上种菜，要算面积。你没办法直接套矩形公式，怎么办？

  • 第一步：把菜地横着切成 \(n\) 条，每条宽度 \(\dfrac{b-a}{n}\)
  • 第二步：每条近似看成一个小矩形，高取条上某个点的函数值
  • 第三步：把 \(n\) 个小矩形面积加起来
  • 第四步：让 \(n \to \infty\)，条越来越细，矩形越来越接近真实面积 → 这个极限就是定积分

严格的数学定义（考法四的核心）：

\[ \int_{0}^{1} f(x)\,dx \;=\; \lim_{n \to \infty} \sum_{i=1}^{n} \frac{1}{n} \cdot f\!\left(\frac{i}{n}\right) \]

小白拆解公式：

  • 左边 \(\int_{0}^{1} f(x)\,dx\)：在 \([0,1]\) 上求 \(f(x)\) 的定积分（精确值）
  • 右边 \(\lim_{n\to\infty}\sum\frac{1}{n}f(\frac{i}{n})\)：把 \([0,1]\) 等分成 \(n\) 份，每份宽 \(\frac{1}{n}\)，在第 \(i\) 个点 \(\frac{i}{n}\) 取函数值，累加再取极限
  • 本质：极限版的「割圆术」（刘徽算圆周率的方法）

⚠️ 小白识别信号（看到这种题就要想到定积分）

数列求和的式子里，同时满足以下 3 条，就能化成定积分：

  1. 有 \(\lim\limits_{n \to \infty}\)（极限符号）
  2. 有 \(\sum\limits_{i=1}^{n}\)（求和号，求和项数随 \(n\) 增长）
  3. 求和项里能拆出一个 \(\dfrac{1}{n}\)（这是「矩形宽度」）+ 一个只关于 \(\frac{i}{n}\) 的函数（这是「矩形高度」）

三步走口诀「提取因子 → 凑标准型 → 令 x」：

  • ① 提取因子：把 \(\frac{1}{n}\) 单独拎出来（这是矩形宽）
  • ② 凑标准型：剩下的部分只关于 \(\frac{i}{n}\)，记为 \(f\!\left(\frac{i}{n}\right)\)
  • ③ 令 \(x = \frac{i}{n}\)，把求和变成 \(\int_{0}^{1} f(x)\,dx\)，直接套积分公式
📝 [2017 数一、二、三] 用定积分定义求数列极限（答案 1/4）
求 \(\displaystyle\lim_{n \to \infty} \sum_{i=1}^{n} \dfrac{i}{n^{2}} \ln\!\left(1 + \dfrac{i}{n}\right)\)

🧭 第一步：看出题人在考什么

这道题 2017 年数一、数二、数三都考了（说明是当年公共大题）。出题人的意图清晰：他不要你硬算，他要你一眼看出「这是定积分的极限定义」。只要识别出来，剩下的就是套积分公式，3 分钟做完。

为什么出题人偏爱这道题？因为它考察三件事，每件都是高频考点：

  • ✅ 能不能识别「\(n\) 项求和 → 定积分」的转换（考眼力）
  • ✅ 会不会用 \(\ln(1+x)\) 的积分（含对数函数的积分，是 ch03 重点）
  • ✅ 会不会分部积分法（考法四里那道 \(\int x\ln(1+x)\,dx\) 是分部积分的典型）

🧭 第二步：三步套定积分公式

① 提取因子：把 \(\frac{1}{n}\) 拎出来

\[ \text{原式} = \lim_{n \to \infty} \sum_{i=1}^{n} \left( \dfrac{i}{n} \right) \cdot \dfrac{1}{n} \cdot \ln\!\left(1 + \dfrac{i}{n}\right) \]

② 凑标准型：令 \(\dfrac{i}{n}\) 为整体，得到 \(f\!\left(\dfrac{i}{n}\right) = \dfrac{i}{n} \cdot \ln\!\left(1 + \dfrac{i}{n}\right)\)

\[ \text{原式} = \lim_{n \to \infty} \sum_{i=1}^{n} \dfrac{1}{n} \cdot f\!\left(\dfrac{i}{n}\right) \]

③ 令 \(x = \frac{i}{n}\)：当 \(i\) 从 1 跑到 \(n\)，\(x\) 从 \(\frac{1}{n}\) 跑到 \(1\)。\(n\to\infty\) 时下界 \(\to 0\)。所以：

\[ \text{原式} = \int_{0}^{1} f(x)\,dx = \int_{0}^{1} x \ln(1 + x)\,dx \]

🧭 第三步：算这个定积分（小白版每一步详细拆）

⚠️ 重要警告：如果你忘了 \(\int x\ln(1+x)\,dx\) 怎么求，看下面这 4 步！

步骤 ①：识别「乘积」形式 → 用分部积分

被积函数是 \(x \cdot \ln(1+x)\)——一个幂函数 × 一个对数函数。这种形式必须用分部积分法，没有第二条路。

步骤 ②：分部积分公式（回忆 5 秒）

\[ \int u\,dv = u\,v - \int v\,du \]

口诀「反对幂三指」：遇到「指数×三角」「幂×三角」「幂×对数」「幂×指数」时，按这个顺序选靠前的当 \(u\)（先求导）。

  • 反：反三角函数（arcsin, arctan…）→ 作 \(u\)
  • 对：对数函数（ln, log）→ 作 \(u\)
  • 幂：幂函数（\(x, x^2\)…）→ 作 \(u\)
  • 三：三角函数（sin, cos）→ 作 \(u\)
  • 指：指数函数（\(e^x, a^x\)）→ 作 \(u\)

我们的被积函数是「幂（\(x\)）× 对（\(\ln(1+x)\)）」，按口诀对在前，所以：

  • 取 \(u = \ln(1+x)\)，则 \(du = \dfrac{1}{1+x}\,dx\)（对数求导）
  • 取 \(dv = x\,dx\)，则 \(v = \dfrac{x^{2}}{2}\)（幂函数积分）

步骤 ③：套公式

\[ \int_{0}^{1} x \ln(1+x)\,dx = \left[\dfrac{x^{2}}{2} \ln(1+x)\right]_{0}^{1} - \int_{0}^{1} \dfrac{x^{2}}{2} \cdot \dfrac{1}{1+x}\,dx \]

代入上下限：\(\left[\dfrac{x^{2}}{2}\ln(1+x)\right]_{0}^{1} = \dfrac{1}{2}\ln 2 - 0 = \dfrac{\ln 2}{2}\)。

步骤 ④：拆剩余积分（多项式长除法）

现在要算 \(\int_{0}^{1} \dfrac{x^{2}}{1+x}\,dx\)。这是一个有理分式积分，分子次数 ≥ 分母次数，要先做多项式除法：

\[ \dfrac{x^{2}}{1+x} = \dfrac{(x^{2}-1)+1}{1+x} = (x-1) + \dfrac{1}{1+x} \]

生活例子🌰：这就像你要把「8 米布分给 3 个人」，每人 2 米还剩 2 米，再单独处理剩下的。所以 \(\dfrac{x^{2}}{1+x}\) 拆成「整除部分 \((x-1)\)」+「真分式部分 \(\dfrac{1}{1+x}\)」。

逐项积分：

\[ \int_{0}^{1} (x-1)\,dx + \int_{0}^{1} \dfrac{1}{1+x}\,dx = \left[\dfrac{x^{2}}{2} - x\right]_{0}^{1} + \left[\ln(1+x)\right]_{0}^{1} \]

代入：\((1/2 - 1) - 0 + \ln 2 - 0 = -\dfrac{1}{2} + \ln 2\)。

步骤 ⑤：合并最终结果

\[ \int_{0}^{1} x\ln(1+x)\,dx = \dfrac{\ln 2}{2} - \dfrac{1}{2}\left(-\dfrac{1}{2} + \ln 2\right) = \dfrac{\ln 2}{2} + \dfrac{1}{4} - \dfrac{\ln 2}{2} = \dfrac{1}{4} \]
🎯 最终答案：\(\displaystyle\lim_{n\to\infty}\sum_{i=1}^{n}\dfrac{i}{n^{2}}\ln\!\left(1+\dfrac{i}{n}\right) = \dfrac{1}{4}\)

🧭 第四步：整套方法的思维导图

  • 识别：看到 \(\lim\sum\) + 求和项里含 \(\frac{1}{n}\) → 想到定积分
  • 变形：提取 \(\frac{1}{n}\)，凑 \(\sum\frac{1}{n}f(\frac{i}{n})\) 形式
  • 转化：令 \(x = \frac{i}{n}\)，写成 \(\int_0^1 f(x)\,dx\)
  • 计算：被积函数含「幂 × 对」→ 分部积分，「反对幂三指」选对数为 \(u\)
  • 化简：积分结果含真分式 → 多项式除法拆成整除 + 真分式

⚠️ 小白踩坑预警（这 3 个地方错了就一分没有）

  • 坑① 上下限写反：原式是 \(\lim\sum_{i=1}^{n}\)，下限是 \(\frac{1}{n}\)，上限是 \(\frac{n}{n}=1\)。\(n\to\infty\) 时下限 \(\to 0\)，不能写成 \(\int_1^1\)。
  • 坑② 忘记提取 \(\frac{1}{n}\)：如果漏了 \(\frac{1}{n}\)，就凑不出 \(\sum\frac{1}{n}f\) 形式，下一步转化就不成立。
  • 坑③ 分部积分选错 \(u\)：「幂 × 对」应该选「对」作 \(u\)。如果你选了幂作 \(u\)，会得到更复杂的 \(\int v\,du\)，算不出来。

  • 和 ch03 关系：这个方法其实是 ch03「定积分」的反向应用——定积分是「求面积」，这里是「用极限定义反推面积」。所以要掌握这个方法，ch03 的积分公式和分部积分必须熟练。
  • 和生活关系：这就像你给菜地围篱笆。先量长度（每段宽 \(\frac{1}{n}\)），再算每段大概多长（高 \(f(\frac{i}{n})\)），加起来就是近似总长。\(n\) 越大越精确。
📝 视频题型：[2018 数一、二、三] 利用单调有界证明数列极限
设数列 {xn} 满足 x1 > 0，
xn · exn+1 = exn − 1   (n = 1,2,…)
证明 {xn} 收敛，并求 limn→∞ xn

🧭 第一步：先看出题人想考你什么

出题人递给你这个递推式的时候，脑子里在想三件事：

  • 他想看你能不能从 \(x_{n}\, e^{x_{n+1}} = e^{x_{n}} - 1\) 变形出 \(e^{x_{n+1}} = \dfrac{e^{x_{n}} - 1}{x_{n}}\) — 这是一个"差 ÷ 长"的结构。看到这个结构你就要反应出"拉格朗日"四个字。
  • 他想看你会不会用 \(e^{x} - 1 \ge x\) 来证有界性。
  • 他想看你会不会用单调有界原理做收尾。（还有一个隐含的）

翻译成人话：出题人给了你一个递推式，他期待你用「拉格朗日」或「\(e^{x} - 1 \ge x\)」来判断数列单调性，用「数学归纳法 + 不等式」来证有界，最后用「单调有界原理」说收敛，再对递推式取极限求答案。

🧭 第二步：证下界（数学归纳法）

先把递推式变形，让 \(e^{x_{n+1}}\) 单独在一边：

由 xn exn+1 = exn − 1  ⇒  exn+1 = (exn − 1) / xn

n=1 时：

  • 已知 x₁ > 0，代入 n=1：ex₂ = (ex₁ − 1) / x₁
  • 由经典不等式 e^x − 1 ≥ x（x>0 时严格大于），得 ex₁ − 1 > x₁
  • 所以 ex₂ = (ex₁ − 1) / x₁ > x₁ / x₁ = 1 → ex₂ > 1 → x₂ > 0 ✓

假设 n=k 时 xk > 0 成立 → 验证 n=k+1：

  • exk+1 = (exk − 1) / xk
  • 由 exk − 1 > xk（xk > 0），得 exk+1 > 1 → xk+1 > 0 ✓

由数学归纳法，对所有 n ∈ N，有 xn > 0。数列有下界 0。✓

这一步在考什么：数学归纳法的标准格式 + 经典不等式 e^x−1≥x 的灵活运用。出题人不需要你证明这个不等式（那是高中水平），但需要你会用。

🧭 第三步：证单调——方法① 拉格朗日（考场首选，最短）

注意，上一步我们得到了：exn+1 = (exn − 1) / xn

右边那个 \(\dfrac{e^{x_{n}} - 1}{x_{n}}\) 是什么？

是 f(xn) − f(0) 除以 (xn − 0) —— 其中 f(t) = et！

对 f(t) = et 在 [0, xn] 上用拉格朗日中值定理：

(exn − e0) / (xn − 0) = eξn，其中 ξn ∈ (0, xn)

所以：

exn+1 = eξn，其中 ξn ∈ (0, xn)

因为 ξn < xn，且 et 严格递增（导数 et > 0），所以：

eξn < exn  ⇒  exn+1 < exn  ⇒  xn+1 < xn

所以 {xn} 严格单调递减。✓

注意速度：拉格朗日法只用了 2 行 —— 识别结构 → 套定理 → 利用 et 递增判大小。这是出题人给的最短路径。

🧭 第三步备选——方法② 作差造函数法（拉格朗日忘了用它）

还是从 \(\displaystyle e^{x_{n+1}} = \frac{e^{x_{n}} - 1}{x_{n}}\) 出发：

作差：exn − exn+1 = exn − (exn − 1) / xn

通分：exn − exn+1 = (xnexn − exn + 1) / xn

现在看分子部分。令 g(t) = t et − et + 1，t ≥ 0：

  • g(0) = 0·1 − 1 + 1 = 0
  • g'(t) = et + t et − et = t et ≥ 0（t ≥ 0）

因为 g'(t) ≥ 0，g(t) 在 [0, +∞) 上单调递增。所以当 t > 0 时 g(t) > g(0) = 0。

令 t = xn > 0，则 g(xn) > 0，即 xnexn − exn + 1 > 0。

所以 exn − exn+1 > 0 → exn > exn+1 → xn > xn+1

{xn} 严格单调递减。✓

🧭 第三步备选——方法③ 寻导法（母函数法）

把递推式 \(\displaystyle e^{x_{n+1}} = \frac{e^{x_{n}} - 1}{x_{n}}\) 两边取自然对数：

xn+1 = ln(exn − 1) − ln xn

所以可以写 xn+1 = f(xn)，其中 f(t) = ln(et − 1) − ln t，t > 0。

求导：

f'(t) = et/(et − 1) − 1/t = (t et − et + 1) / t(et − 1)

分母 t(et − 1) > 0（t>0），分子我们已经证明过 g(t) = t et − et + 1 > 0，所以 f'(t) > 0，f(t) 单调递增。

又因为直接从已知条件可算出 x₂ < x₁（比如用拉格朗日法，或 ex₂ = (ex₁−1)/x₁ < ex₁ → x₂ < x₁），所以：

{xn} 由 x₂ < x₁ 且 f 递增 → x₃ = f(x₂) < f(x₁) = x₂ → ... 依次有 xn+1 < xn，递减。✓

🧭 第四步：由单调有界原理得收敛

数列 {xn} 单调递减且有下界 0 → 由单调有界原理，数列 {xn} 收敛。

设 limn→∞ xn = A。

🧭 第五步：求极限值 A

对递推式 xn exn+1 = exn − 1 两边取极限（利用极限保号性）：

A · eA = eA − 1  ⇒  eA(A − 1) = −1  ⇒  eA(1 − A) = 1

解这个方程：令 h(A) = eA(1 − A) − 1

  • 先试 A=0：h(0) = 1·1 − 1 = 0 ✓
  • 看单调性：h'(A) = eA(1 − A) − eA = −A eA
  • 当 A ≥ 0 时 h'(A) ≤ 0（等号仅 A=0），所以 h(A) 严格递减
  • 因此唯一解为 A = 0
∴ limn→∞ xn = 0

回顾整个流程：

① 变形递推式显化 exn+1 → ② 经典不等式 e^x−1≥x + 数学归纳法证有界 → ③ 拉格朗日中值定理（或作差/寻导）证单调递减 → ④ 单调有界原理 → ⑤ 取极限解方程 → ⑥ A=0。六步全。

🎯 出题人视角：这道题同时出现在 2018 年数一、数二、数三的卷子上，说明它是当年所有人都必须会的标准题。出题人想看你三件事：① 给递推式之后能不能自己变形成「可拉格朗日」的形式；② 会用 e^x−1≥x 这个基本不等式证正有界；③ 单调递减的判断路径——他给了你至少三条路（拉格朗日/作差/寻导），你能走通任意一条都给分。
如果你想在考场上快速得分：走方法①（拉格朗日法）。它最短、最省笔、最不容易出错。拉格朗日忘了 → 方法②（作差造函数法），它的思路是「通分 → 令函数 → 求导 → 判大小」。
🧠 小白记忆类比：把这道题比作「玩滑梯」。x₁>0 是你站在滑梯顶。数学归纳法证明你每一步都能踩到滑梯面（有下界 0）。拉格朗日法证明滑梯的坡度是往下的（单调递减）。单调有界原理说 "滑梯有底有坡 → 一定能滑到底"。最后取极限解方程算出底就在地面 zero。

[解] 由题设 xn exn+1 = exn − 1，变形为 exn+1 = (exn − 1) / xn。

步骤①：证明 xn > 0（有下界）[给分点 1]
由 x₁ > 0 及 e^x − 1 > x (x > 0)：
  ex₂ = (ex₁ − 1)/x₁ > x₁/x₁ = 1 → x₂ > 0。

步骤②：证明 {xn} 严格单调递减 [给分点 2]
（推荐拉格朗日法，最快）
对 f(t)=et 在 [0, xn] 上用拉格朗日中值定理：
  ∃ ξn ∈ (0, xn)，使 (exn − 1)/xn = eξn。
故 exn+1 = eξn。由 ξn < xn 及 et 的单调性：
  eξn < exn → exn+1 < exn → xn+1 < xn。

步骤③：单调有界原理 [给分点 3]
{xn} 单调递减有下界 ⇒ 收敛。

递推式两边取极限：A·eA = eA − 1 ⇒ eA(1 − A) = 1。
令 h(A) = eA(1 − A) − 1，A ≥ 0。
h(0) = 0，h'(A) = −A eA ≤ 0 ⇒ h(A) 在 [0, +∞) 上严格递减，

  ∴ limn→∞ xn = 0。 [证毕]

递推式数列极限的万能三步走
拿到任何递推式 xn+1 = f(xn) 或 xnexn+1=…：
① 先定有界：变形后看能不能用不等式/数学归纳法找到一个界（这题 xn > 0 是天然下界）；
② 再判单调：优先看看有没有 (ex−1)/x 这种"差÷长"结构 → 有就用拉格朗日；没有就作差/作商/寻导；
③ 取极限：收敛后对递推式两边取极限，解方程得 A。
小白口诀：有界 \(\to\) 单调 \(\to\) 取极限。
4 种证明单调性方法 + 1 张决策树
① 拉格朗日法（首选）：看到差÷长结构→一步到位 e^{ξ}；
② 作差造函数法（备胎万能）：xn+1 − xn 通分 → 令 g(x)→导数→判符号；
③ 作商法：xn 全正、比能化简时优先，否则不如作差；
④ 寻导法（母函数法）：递推式能写成 xn+1=f(xn)，且 f 可导时备用；
决策树：差÷长结构？→ 拉格朗日 ✓ | 差÷长不存在？→ 能通分？→ 作差 | 能直接看？→ 作商
单调有界题型 6 大踩坑点
  • 只证有界忘了证单调：有界 + 不单调 ≠ 收敛（摆动数列也有界但不收敛）。
  • 下界讲反了：我们证的是"下界 0"不是"上界"> —— 递减数列有下界才收敛。
  • 数学归纳法开始少写了 n=1 检验：必须先算 x₂ > 0，再假设 xk → 推 xk+1。
  • 取极限后解方程漏解：eA(1−A)=1 不是多项式方程，先试 A=0，再用单调性证唯一。
  • 拉格朗日中值定理用反了区间：是 [0, xn] 不是 [xn, 0]，和为零的顺序一致。
  • 用到不等式 \(e^{x} - 1 \ge x\) 时忘了提条件：只有在 \(x>0\) 时严格 >；\(x<0\) 时方向可能相反，要小心。

类型左右极限函数值例子记忆标签
可去间断点 左右极限存在且相等 不等于极限（或无定义） \(y=\dfrac{\sin x}{x}\) 在 \(x=0\) 「补一个值就能连续」

📝 真题 3 | 2022 数一 | 选择题 · 5分
设 \(f(x)\) 在 \((-\infty,+\infty)\) 内连续，\(\lim\limits_{x\to 0}\dfrac{f(x)-1}{x}=2\)，则 \(x=0\) 是 \(f(x)\ln|x+1|\) 的（ ）。A. 间断点 B. 连续但不可导 C. 可导不连续 D. 连续且可导

🎯 出题人考察点（这是高频考点！）

① 已知极限反推函数值；② 连续/可导的复合判定；③ \(\ln|x+1|/x\to 1\) 等价替换

🧭 详细解析（小白逐步版）

由条件得 \(f(0)=1\)。设 \(g(x)=\ln|x+1|\)，\(g(0)=0\)。\(g(x)\) 在 \(x=0\) 不可导但连续。\(f(x)g(x)\) 极限：\(\lim_{x\to 0}f(x)g(x)=1\cdot 0=0\)，故 \(f(0)g(0)=0\) 连续。但导数极限：\(\lim_{x\to 0}\frac{f(x)g(x)}{x}=\lim_{x\to 0}f(x)\cdot\frac{\ln|x+1|}{x}=1\cdot 1=1\)，故可导。选 D。
