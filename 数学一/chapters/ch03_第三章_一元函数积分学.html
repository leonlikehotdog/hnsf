🌌 诞生背景 · 积分是导数的"亲子关系"
这一章的故事，是从古希腊"算面积"的死磕开始的，最后由牛顿和莱布尼茨用一个公式统一了 2300 年的争吵。
🎬 ch02 学的导数 vs ch03 要学的积分

ch02 我们学的是"正着来"：知道函数 → 求它"瞬时变化率"。

ch03 要学的积分是"反着来"：知道瞬时变化率 → 反推出"原来长了多少"。

比如：

📍 ch02 → 一辆车正在以 60 km/h 跑（瞬时速度），问"这一秒速度多少"？

📍 ch03 → 一辆车一直以 60 km/h 跑，问"过去 1 小时跑了多少 km"？

看起来是同一件事的两种看法，但数学上要"反着算"绝对没那么简单 —— 这就是本章要解决的核心问题：

导数 = 微分；积分 = 求"原" —— 互为逆运算

⏳ 2300 年积分史 · 面积问题从未消退
📖 阿基米德抛物线面积
-250 阿基米德："我算出了曲边形的面积！"

🎯它是为了解决什么问题？

古希腊人不会算"曲边图形的面积"。比如抛物线和一条直线围成的形状，套不上矩形公式。

公元前 250 年，西西里岛的阿基米德冥思苦想出一个"穷竭法"：

把抛物线内部切成无数个三角形 → 求和 → 正好等于 4/3 × 底 × 高，精确的常数比。

"我敢打赌：曲边形面积也能精确算出来 — 这就是数学的力量。" —— 阿基米德

这就是最早的积分。比牛顿莱布尼茨早 2000 年。方法对的，只是没发明"极限"那套代数符号。

⚠️小八卦：罗马军队攻破叙拉古城时，一个士兵冲进阿基米德家要抓他。阿基米德正埋头画几何图形，头也不抬说"别踩我的圆"。士兵一怒之下拔剑把他杀了。

📱今天有什么用？你手机里所有的"圆角矩形按钮"面积、设计师画的所有弧线字体的外框、桥梁弧形部分的面积 —— 全部都靠阿基米德这招算。

📖 卡瓦列里不可分量原理
1629 卡瓦列里："把图形切成无数细线"

🎯它是为了解决什么问题？

阿基米德死后 1900 年，没有人能写出通用的"曲边形面积 = 多少"的公式。

1629 年意大利修道士 卡瓦列里 (Bonaventura Cavalieri) 提出不可分量原理 (Indivisibles)：

"任何平面图形，都可以想成无数条平行线堆出来的。把这些线都加在一起，就是它的面积。"

这是第一次用"连续累加"的思路代替"穷举细分"。听起来理所当然，但在 1629 年这是一个超大革命 —— 当时大家还在信奉"只有有限的数能加总"。

"把一根 1 米长的线切成无穷多份 — 它们加在一起还是 1 米。这不是悖论，这是真理。" —— 卡瓦列里

📱今天有什么用？核磁共振 (MRI) 扫描就是用"切片 → 累加"的思路，把一张 3D 人体模型从无数 2D 切片拼回来。这就是卡瓦列里的"不可分量"。

📖 牛顿-莱布尼茨公式
1670 两个不认识的数学家，独立发明了同一件事

🎯它是为了解决什么问题？

1660 年代末，两件事让微积分行得通了：

① 算瞬时速度（导数）—— 我们在 ch02 学过

② 算曲线面积（积分）—— 一直没人搞定

1669 年 牛顿（在剑桥大学）发现：算瞬时速度的"反"操作，正好就是算面积！也就是：

如果 f 的导数是 F（瞬时变化率 = F），那么 F 累积起来就是 f → 这俩互为"父子"

几乎是同一时间，莱布尼茨也独立发现同一件事。并且他在 1675 年 10 月 29 日的笔记本里第一次写下了 ∫ 这个符号。他选了一个"拉长的 S"，代表拉丁文 Summa（总和）。

从此"算面积"不再是难题：你只要找到 f 函数的原函数 F(x)，那 ∫ f dx = F(b) - F(a)，一步出答案 —— 因为导数 + 积分互为逆运算。

"微积分基本定理：积分是导数的逆运算 — 2000 年的死磕到此为止。" —— 牛顿《流数法》1671

📱今天有什么用？你手机电量从 100% 掉到 20%，手机能估计还要多久充满 —— 因为它每秒在用"瞬时电流对时间的积分"算剩余电量。物理里几乎所有"瞬时量 → 总量"的转换都是这个公式。

📖 凑微分法
1670s 一个"看到函数和它的导数凑一起就化简"的窍门

🎯它是为了解决什么问题？

牛顿发现了"积分 = 导数的逆运算"。但接下来一个新问题：

很多函数没有原函数公式。比如 \(\int e^{x^2} dx\)，永远写不出"基本函数"形式。

数学家们灵机一动：如果被积函数本身就是某个"复合函数"的导数，那直接令 u = 内层，化成简单积分即可。

∫ 2x · e^(x²) dx → 看到 2x 就是 x² 的导数 → 令 u = x² → 原式 = ∫ e^u du = e^(x²) + C

这是凑微分法的灵魂：原函数隐藏在"复合函数"的结构里。你只需要把它揪出来。

"积分就像开锁 — 你要找到那个对的钥匙 (u)。" —— 现代微积分老师口头禅

📱今天有什么用？机器学习里的"激活函数"导数求解、概率论里的"分布函数积分"、经济学里的"边际积分" —— 全部都是这个套路的现代版本。

📖 分部积分法
1690s 莱布尼茨：「乘起来的积分，要拆开算」

🎯它是为了解决什么问题？

凑微分只能解"复合函数"型的积分。但有些被积函数是两个函数相乘（比如 x · sin x, x · ln x）—— 凑微分无能为力。

1690 年代，莱布尼茨推导出一个公式：

∫ u dv = uv - ∫ v du

意思是：把 uv 的积分转化为 u·v 减去 v·du 的积分。通常选"反对幂三指"中靠前的作 u —— 这样 u' 求导会变简单（指数 → 二次方 → ... → 0）。

举个例子：∫ x · e^x dx

  • 取 u = x, dv = e^x dx
  • 则 du = dx, v = e^x
  • 原式 = x · e^x - ∫ e^x dx = x · e^x - e^x + C = e^x(x - 1) + C
"反对幂三指 — 一个 5 字口诀解决 80% 的分部积分题。" —— 莱布尼茨

📱今天有什么用？工程里"电流 × 电压 = 功率，但功率对时间的积分 = 能量" —— 这种"乘积的积分"在物理里满地都是，分部积分就是它的统一钥匙。

📖 反常积分（广义积分）
1821 柯西：「无穷大也可以积分！」

🎯它是为了解决什么问题？

正常积分只能算有限区间内的有限函数。但物理里大量现象是"无穷"的：

• 一根无限长的导线，磁场是多少？

• 一个无限大的电阻，电流是多少？

• 从有文明到永远的宇宙，总能量是多少？

1821 年，柯西 (Cauchy) 第一个敢在积分上限里写 +∞：

∫₁^+∞ 1/x² dx = lim (M→+∞) ∫₁^M 1/x² dx = lim (1 - 1/M) = 1 ✓

但有些无穷积分 不收敛：

∫₁^+∞ 1/x dx = lim (M→+∞) ln M = +∞ ✗ 发散

所以有了判断判别式 ∫₁^+∞ 1/x^p dx：p > 1 收敛，p ≤ 1 发散。

"有限可以积，无穷也可以积 — 只要它'收敛'。" —— 柯西《分析教程》1821

📱今天有什么用？概率论里"随机变量取遍全宇宙会得到什么概率"就是反常积分 —— 这就是你在 ch15-ch19 学概率的前提。

🎁 这一章的"灵魂主线"
一句话讲完 ch03 所有故事的共同主线

🧵这一章所有公式，归根结底就为了解决一个 2000 年的死磕：怎么算"曲边形的面积"。

积分的核心 = 把"无穷多个无穷小片段"加起来的极限。

它有以下几种"伪装"：

📍 几何上的伪装：曲边梯形面积、旋转体体积、弧长

📍 物理上的伪装：做功 = 力 × 位移的积分、功率 × 时间 = 能量

📍 概率上的伪装：分布函数的积分 = 概率（ch15-ch19 会用到）

📍 经济上的伪装：现金流的总现值 = 利率 × 时间的积分

💪学完 ch03 + ch02 你的"微积分工具箱"已经有刃有背了：

• ch01 → 学了"极限"是基础工具

• ch02 → 学了"导数"是各种变化率的代名词

• ch03 → 学了"积分"是导数的逆运算（也就是"累积")

接下来 ch04（向量）、ch05/ch06（多元）只是把它们推广到更高维的版本。骨架没变。

📖 必背：凑微分法

看到被积函数里既有 f(x) 又有 f'(x)，把 f'(x)dx 凑成 d(f(x))，整个式子就化为「一个函数的积分」。

∫ f'(x) · g(f(x)) dx → 令 u = f(x)，du = f'(x) dx → ∫ g(u) du

口诀：「看到导数，乘进去；凑出 du，直接积分」。

原式凑微分结果
∫ 2x · ex² dx令 u=x²，du=2x dx∫ eu du = ex² + C
∫ cos x · sin3x dx令 u=sin x，du=cos x dx∫ u³ du = sin⁴x/4 + C
∫ 1/(x ln x) dx令 u=ln x，du=1/x dx∫ 1/u du = ln|ln x| + C
📖 必背：分部积分 ∫u dv = uv − ∫v du

口诀（反对幂三指）：遇到「指数 × 正弦/余弦」「幂函数 × 三角」「幂函数 × 对数」「幂函数 × 指数」时，优先选「反对幂三指」中靠前的作 u。

题型u（先求导的）dv（先积分的）
∫ x · ex dxxex dx
∫ x · sin x dxxsin x dx
∫ x · ln x dxln xx dx
∫ ex · sin x dxsin xex dx（要循环两次）
⚠️ 必背：定积分换元必须同时换上下限

不定积分换元后要回代；定积分换元后要同时把上下限换成新变量的上下限，否则答案差 10 倍甚至正负号反了。

∫ab f(x) dx = ∫αβ f(φ(u)) φ'(u) du，且 α = φ(a)，β = φ(b)

小白技巧：换元前先在小本子上写「x = φ(u)，当 x=a 时 u=?；当 x=b 时 u=?」，再动笔。

📝 例题 3（定积分换元）
求 ∫04 √(16 − x²) dx。
令 x = 4sin t，dx = 4cos t dt。当 x=0 时 t=0；当 x=4 时 t=π/2。原式 = ∫0π/2 4cos t · 4cos t dt = 16 ∫0π/2 cos²t dt = 16 · π/4 = 4π。
📖 必背：反常积分（区间或被积函数无穷）
类型判别式结论
∫1+∞ 1/xp dxp > 1 收敛p ≤ 1 发散
∫01 1/xq dxq < 1 收敛q ≥ 1 发散
∫−∞+∞ e−x² dx—= √π（必背）

对比法：已知 ∫1+∞ 1/x² dx 收敛，对 0 < f(x) ≤ 1/x² 时，∫f 必收敛（夹逼）。

📖 必背：求面积与旋转体体积公式
S = ∫ab |f(x) − g(x)| dx（两曲线之间）
Vx = π ∫ab f²(x) dx（绕 x 轴旋转，圆盘法）
Vy = 2π ∫ab x · |f(x)| dx（绕 y 轴旋转，柱壳法）

小白口诀：「绕谁轴，谁不动；半径平方积进去」。

📝 例题 4（面积）
求曲线 y = x² 与 y = 2x 所围成的图形面积。
先求交点：x² = 2x → x=0 或 x=2。在 [0,2] 上 2x ≥ x²，所以面积 = ∫02 (2x − x²) dx = (x² − x³/3)|02 = 4 − 8/3 = 4/3。
📖 小白扫盲：不定积分 · 定积分 · 反常积分（一表分清）

🎯 抓住本质：不定积分是找原函数，定积分是算数值，反常积分是正常积分的极限推广

积分类型形式结果核心公式注意事项
不定积分 \(\int f(x)\,dx\) 函数族 \(F(x) + C\) \(\int f(x)\,dx = F(x) + C\) 必须加 \(C\)；换元后要回代
定积分 \(\int_a^b f(x)\,dx\) 数值（一个数） \(\int_a^b f(x)\,dx = F(b) - F(a)\) 换元必须换上下限；被积函数在 \([a,b]\) 上须连续
反常积分 \(\int_a^\infty f(x)\,dx\)
或 \(\int_a^b f(x)\,dx\)（瑕点）		 数值（或发散） \(\int_a^\infty f(x)\,dx = \lim_{t\to\infty}\int_a^t f(x)\,dx\) 必须先判别敛散性，收敛才能算数值

⚠️ 反常积分的两种不同「无穷」

  • 区间无穷无穷限：\(\int_a^\infty\) 或 \(\int_{-\infty}^b\) — 上限或下限趋于无穷
  • 被积函数无穷（瑕积分）：在积分区间内有瑕点（分母为 0、对数取 0 等）— 如 \(\int_0^1 \dfrac{1}{\sqrt{x}}\,dx\)
  • 判断口诀：「区间无限用极限，瑕点先拆再求极限；两无穷不能直接代，拆成两段各自算」

📝 真题 1 | 2018 数一/二/三 | 解答题 · 10分
求 \(\int \dfrac{\arctan\sqrt{x}}{\sqrt{x}(1+x)}\,dx\)。

🎯 出题人考察点（这是高频考点！）

① 凑微分识别（\(d(\arctan\sqrt{x})\)）；② 换元法的灵活运用；③ 反函数积分技巧

🧭 详细解析（小白逐步版）

**思路**：令 \(t=\sqrt{x}\)，\(x=t^2,dx=2t\,dt\)，原式 \(= 2\int\dfrac{\arctan t}{1+t^2}\,dt = (\arctan t)^2 + C = (\arctan\sqrt{x})^2 + C\)。

💯 得分点（阅卷老师按点给分）

① 换元过程 (4分)；② 凑微分 (3分)；③ 最终结果 (2分)；④ +C (1分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 忘了 +C；⚠️ 换元后上下限未对应更新；⚠️ 直接分部积分会非常复杂

📝 真题 2 | 2020 数二 | 解答题 · 12分
设 \(f(x)\) 在 \([0,1]\) 上连续可微，\(f(0)=0, f(1)=1\)，\(\int_0^1 f(x)\,dx=1\)。证明：存在 \(\xi\in(0,1)\) 使 \(f\prime(\xi)=2\)。

🎯 出题人考察点（这是高频考点！）

① 积分中值定理 + 罗尔定理联合使用；② 辅助函数构造；③ 中间值定理

🧭 详细解析（小白逐步版）

**思路**：构造辅助函数 \(F(x)=f(x)-x^2\)。\(F(0)=0,F(1)=0\)，由罗尔定理存在 \(\eta\) 使 \(F\prime(\eta)=0\)，即 \(f\prime(\eta)=2\eta\)。但要 \(f\prime=2\)，需再调整。**正确构造**：设 \(G(x)=f(x)-2x\)，则 \(G(0)=0,G(1)=-1\)，由积分中值定理 \(\int_0^1 f\,dx=1\) 知存在 \(c\) 使 \(f(c)=1\)，于是 \(G(c)=1-2c\)。当 \(c=0\) 时 \(G=1\) 与 \(G(1)=-1\) 反号。

💯 得分点（阅卷老师按点给分）

① 构造辅助函数 (5分)；② 应用积分中值定理 (3分)；③ 应用罗尔定理 (3分)；④ 结论 (1分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 辅助函数不对是头号丢分点；⚠️ 三个定理的适用条件混淆
