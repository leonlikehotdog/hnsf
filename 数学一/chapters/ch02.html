📖 必背：洛必达法则使用条件

当 lim f(x)/g(x) 为 "0/0" 或 "∞/∞" 型未定式，且 f'(x)、g'(x) 存在，g'(x) ≠ 0 时：

lim f(x)/g(x) = lim f'(x)/g'(x)
📝 例题 1
求 y = x³ + 2x² - 3x + 1 在 x = 1 处的切线方程
先求导 y' = 3x² + 4x - 3，代入 x = 1 得斜率 k = 4；再求 y(1) = 1，所以切线为 y - 1 = 4(x - 1)
📝 例题 2
求函数 f(x) = x³ - 3x + 2 的极值
f'(x) = 3x² - 3 = 0 得 x = ±1；f''(x) = 6x；f''(1) = 6 > 0 为极小值 f(1) = 0；f''(-1) = -6 < 0 为极大值 f(-1) = 4
📖 必背：隐函数求导三步法

题目给的是 F(x, y) = 0，没法直接写成 y = f(x) 时，按下面三步走：

  1. 等式两边同时对 x 求导，y 看成 x 的函数 → 出现 dy/dx 或 y'。
  2. 把含 y' 的项留在等号左边，其他项移到右边。
  3. 解出 y'，得到 dy/dx 的显式表达式。
例：x² + y² = 1 两边求导得 2x + 2y·y' = 0 → y' = −x/y。注意：y 出现在分母，不要写成 −x。
📖 必背：反函数求导公式
ddxf−1(y)=1f′(x)=1f′(f−1(y))

小白理解：正函数求导是「顺水」，反函数求导是「逆水」，二者速度互为倒数。记忆口诀：「反函数的导 = 1 ÷ 原函数在对应点的导」。

常见陷阱：题目问的是 f−1(x) 的导数，要把 f−1(x) 的导数表达成 1 / f'(f−1(x))，不能直接当成 1/f'(x)。

📝 例题 3（隐函数）
设 y 由方程 ey + xy = e 确定，求 dy/dx。
两边对 x 求导：ey·y' + y + x·y' = 0 → y'(ey + x) = −y → y' = −y / (ey + x)。
📖 必背：麦克劳林公式（x=0 处的泰勒展开）

如果 f(x) 在 x=0 处 n 阶可导，则：

f(x) = f(0) + f'(0)x + f''(0)x²/2! + f'''(0)x³/3! + ... + f(n)(0)xn/n! + o(xn)

记忆口诀：「n 阶导除以 n!，乘 x 的 n 次方」一直加到 n 阶。

为什么重要：泰勒是洛必达的「降维打击」。当分子分母都是复杂函数时，直接洛必达要展开 3~4 次求导，容易写错；改用泰勒，只需代入「f(n)(0)」即可，结果更稳。

📝 例题 4（泰勒）
求 limx→0 (ex² − 1 − x²) / (x4) 的极限。
把 ex² 展开到 x⁴ 项：ex² = 1 + x² + x⁴/2 + o(x⁴)。代入分子 = (1 + x² + x⁴/2) − 1 − x² = x⁴/2。除以分母 x⁴ 得 1/2。
📖 必背：偏导与全微分
概念公式小白理解
对 x 偏导 ∂f/∂x把 y 当常数，只对 x 求导只看 x 这一条方向上 f 怎么变化
对 y 偏导 ∂f/∂y把 x 当常数，只对 y 求导只看 y 这一条方向上 f 怎么变化
全微分 df∂f/∂x · dx + ∂f/∂y · dyx 和 y 同时小变化时，f 的总增量

生活例子：f = 工资 = 底薪 + 时薪 × 工时。底薪、时薪、工时 3 个变量互不影响，∂工资/∂工时 = 时薪，∂工资/∂时薪 = 工时。

📝 例题 5（偏导）
设 f(x, y) = x² + xy + y³，求 ∂f/∂x、∂f/∂y。
∂f/∂x = 2x + y（y 当常数）；∂f/∂y = x + 3y²（x 当常数）。全微分 df = (2x+y)dx + (x+3y²)dy。
📖 必背：二元无条件极值
  1. 求 ∂f/∂x = 0，∂f/∂y = 0，解出所有驻点。
  2. 求二阶偏导：fxx、fyy、fxy。
  3. 计算判别式 AC − B²（A=fxx，C=fyy，B=fxy）：
    • AC − B² > 0 且 A > 0 → 极小值
    • AC − B² > 0 且 A < 0 → 极大值
    • AC − B² < 0 → 不是极值
📝 例题 6（条件极值）
在 x² + y² = 1 的条件下，求 f = xy 的最大值。
用拉格朗日乘数法：L = xy + λ(1 − x² − y²)。∂L/∂x = y − 2λx = 0，∂L/∂y = x − 2λy = 0。解得 x = ±y，代入约束条件得 x = ±√2/2，y = ±√2/2，fmax = 1/2。
