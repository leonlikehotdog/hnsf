第十二章：线性方程组 | 6分
⭐⭐⭐⭐⭐

🎯 核心考点

  • 齐次线性方程组的解的结构与求解
  • 非齐次线性方程组的解的结构与求解
📖 必背：解的结构定理

齐次方程组 Ax = 0

  • 一定有零解（x = 0）
  • 有非零解 ⟺ r(A) < n（未知数个数）
  • 通解 = 基础解系的线性组合：k₁ξ₁ + k₂ξ₂ + ...

非齐次方程组 Ax = b

  • 有解 ⟺ r(A) = r(A|b)
  • 通解 = 齐次通解 + 非齐次特解
📝 例题
求解线性方程组：x₁ + 2x₂ = 5，3x₁ + 4x₂ = 11
写成矩阵形式 [1 2; 3 4][x₁; x₂] = [5; 11]，|A| = 1·4 - 2·3 = -2 ≠ 0，有唯一解。
用克拉默法则或消元：由①得 x₁ = 5-2x₂，代入②得 3(5-2x₂)+4x₂=11 ⟹ 15-6x₂+4x₂=11 ⟹ -2x₂=-4 ⟹ x₂=2，x₁=1。
所以解为 x₁ = 1，x₂ = 2
