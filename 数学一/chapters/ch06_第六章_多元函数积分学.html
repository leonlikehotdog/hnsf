🎯 重点掌握：二重积分

⚠️ 三重积分、曲线曲面积分建议放弃

📖 二重积分（直角坐标）
∫∫D f(x,y) dσ = ∫ab dx ∫y₁(x)y₂(x) f(x,y) dy

先对 y 积分，再对 x 积分

📖 二重积分（极坐标）
∫∫D f(x,y) dσ = ∫αβ dθ ∫r₁(θ)r₂(θ) f(r cos θ, r sin θ) r dr

适用于圆形、扇形区域，别忘了多乘一个 r

📝 例题：计算二重积分
计算 ∫∫D (x+y) dxdy，其中区域 D：0 ≤ x ≤ 1，0 ≤ y ≤ 1
原式 = ∫01dx ∫01(x+y)dy = ∫01[xy + y²/2]01dx
= ∫01(x + 1/2)dx = [x²/2 + x/2]01 = 1
📖 小白扫盲：三大类多元积分（一表分清考什么）

🎯 二重积分、三重积分、曲线/曲面积分 — 区别在"在哪积"和"积什么"

积分类型积分区域被积对象计算方法标记
二重积分 平面区域 \(D\) 二元函数 \(f(x,y)\) 直角坐标 / 极坐标
先定限再求积		 \(\iint_D f(x,y)\,d\sigma\)
三重积分 空间区域 \(\Omega\) 三元函数 \(f(x,y,z)\) 直角坐标 / 柱坐标 / 球坐标 \(\iiint_\Omega f(x,y,z)\,dV\)
第一类曲线积分
（对弧长）		 平面曲线 \(L\) \(f(x,y)\) 参数化：\(\int_L f\,ds = \int_a^b f(\phi(t))\,|\phi'(t)|\,dt\) \(\int_L f(x,y)\,ds\)
第二类曲线积分
（对坐标）		 有向曲线 \(L\) \(P\,dx + Q\,dy\) 参数化 / 格林公式 \(\int_L P\,dx + Q\,dy\)
第一类曲面积分
（对面积）		 曲面 \(S\) \(f(x,y,z)\) 投影法：将曲面投影到坐标平面 \(\iint_S f(x,y,z)\,dS\)
第二类曲面积分
（对坐标）		 有向曲面 \(S\) \(P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy\) 高斯公式 / 投影法 \(\iint_S \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S}\)

🧠 三大公式的关系

  • 格林公式：通第二类曲线积分通平面二重积分 — \( \oint_L P\,dx+Q\,dy = \iint_D (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})\,d\sigma\)
  • 高斯公式：通第二类曲面积分通三重积分 — \( \oiint_S \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S} = \iiint_\Omega \nabla\cdot\boldsymbol{F}\,dV\)
  • 斯托克斯公式：通空间曲线积分通曲面积分 — \( \oint_L \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{r} = \iint_S \nabla\times\boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S}\)
  • 一句话：格林 + 高斯 + 斯托克斯 = 由简到繁的「降维」工具
⚠️ 第一类 vs 第二类（曲线/曲面）的区分
第一类（对弧长/对面积）：被积函数是数值，结果不依赖方向，物理意义是质量/面积
第二类（对坐标）：被积函数是向量，结果依赖方向，物理意义是功/通量
判断方法：看 \(\mathrm{d}s\) → 第一类；看 \(\mathrm{d}x\) 或 \(\mathrm{d}\boldsymbol{S}\) → 第二类

📝 真题 1 | 2020 数一 | 解答题 · 12分
计算 \(\iint\limits_{D}\dfrac{x^{2}}{y^{2}}\,d\sigma\)，其中 \(D\) 由 \(y=x, y=2, x=1\) 围成。

🎯 出题人考察点（这是高频考点！）

① 二重积分的积分次序选择；② 积分区域画图；③ 累次积分计算

🧭 详细解析（小白逐步版）

**画图**：D 是梯形。\(\int_1^2 dx\int_x^2 \dfrac{x^2}{y^2}\,dy = \int_1^2 x^2(\frac{1}{x}-\frac{1}{2})\,dx = \int_1^2 (x-\frac{x^2}{2})\,dx = [\frac{x^2}{2}-\frac{x^3}{6}]_1^2 = (2-\frac{8}{6})-(\frac{1}{2}-\frac{1}{6}) = \frac{2}{3}-\frac{1}{3}=\frac{1}{3}\)。

💯 得分点（阅卷老师按点给分）

① 画图确定区域 (3分)；② 化为累次积分 (3分)；③ 内层积分 (3分)；④ 外层积分 (3分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 上下限搞反；⚠️ 先 y 后 x 还是先 x 后 y 选错；⚠️ 被积函数在 y 内容易算错

📝 真题 2 | 2023 数一 | 解答题 · 12分
计算三重积分 \(\iiint\limits_{\Omega}(x^{2}+y^{2})\,dV\)，其中 \(\Omega\) 是由 \(z=\sqrt{x^2+y^2}\) 与 \(z=1\) 围成的区域。

🎯 出题人考察点（这是高频考点！）

① 球坐标变换；② 锥面方程转换；③ 三重积分计算

🧭 详细解析（小白逐步版）

**球坐标**：\(x^2+y^2=r^2, dV=r^2\sin\varphi\,dr\,d\varphi\,d\theta\)。\(\Omega: 0\le\theta\le 2\pi, 0\le\varphi\le \pi/4, 0\le r\le 1/\cos\varphi\)。原式 \(= \int_0^{2\pi}d\theta\int_0^{\pi/4}\sin\varphi\,d\varphi\int_0^{1/\cos\varphi} r^4\,dr\)。

💯 得分点（阅卷老师按点给分）

① 球坐标变换 (3分)；② 确定积分限 (4分)；③ 累次积分 (4分)；④ 最终答案 (1分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 球坐标体积元公式记错（\(r^2\sin\varphi\)）；⚠️ 锥面方程转极坐标不熟练；⚠️ 积分限方向
