ch03 一元积分做的事情很单纯：把无限多个竖细条加起来，求出"面积"。

本章要把这件事从 1 维平面搬到 2 维平面上：先切细条（沿 y 方向），再把无数根竖细条"摞"成一片，从而求出一片曲面的"体积"。

三个核心问题：

  • 🥇 二重积分（在平面区域上求体积）—— 牛顿 1676 → 高斯 1814

工具 公式 小白翻译
直角坐标 · 先 y 后 x \(\displaystyle\iint_D f(x,y)\,d\sigma=\int_a^b\!dx\int_{y_1(x)}^{y_2(x)}\!f\,dy\) 先看每条「竖细条」的高（沿 y），再把每根条的高度加起来（沿 x）
直角坐标 · 先 x 后 y \(\displaystyle\iint_D f\,d\sigma=\int_c^d\!dy\int_{x_1(y)}^{x_2(y)}\!f\,dx\) 改为看「横细条」——有时候这样更方便
极坐标 \(\displaystyle\iint_D f\,d\sigma=\int_\alpha^\beta\!d\theta\int_{r_1(\theta)}^{r_2(\theta)}\!f(r\cos\theta,r\sin\theta)\,r\,dr\) 圆形 / 扇形用极坐标；别忘了 多乘一个 r（面积元变了）
对称性 · 偶 若 D 关于 y 轴对称，f(x,y) 关于 x 为偶 → \(\iint_D f=2\iint_{D_{右}}f\) 图形对折后一半积分，再乘 2
对称性 · 奇 若 D 关于 y 轴对称，f(x,y) 关于 x 为奇 → \(\iint_D f=0\) 正负抵消，不用算
轮换对称性 把 \(x,y\) 互换，区域 D 不变 → \(\iint_D f(x,y)d\sigma=\iint_D f(y,x)d\sigma\) 尤其是 \(x^2+y^2\)、\(x+y\) 这种对称函数

识别口诀：

  • 区域出现 圆 / 圆弧 / 扇形 → 选极坐标
  • 区域是 矩形 / 梯形 / 三角 / 函数曲线 → 直角坐标
  • 被积函数是 x² + y² / x² − y² / y/x → 极坐标
  • 被积函数是 x / y / x+y → 看区域，区域规则用直角，区域圆用极
📖 公式推理 1 · 为什么 \(\iint_D f\,d\sigma=\int_a^b\!dx\int f\,dy\)？

核心问题："在 D 上求和"能不能拆成"先沿 y 求和，再沿 x 求和"？

第 1 步（画一根竖细条）：固定 x，把 y 从下边界 \(y_1(x)\) 加到上边界 \(y_2(x)\)，得到这根条对体积的贡献：

\(A(x)=\displaystyle\int_{y_1(x)}^{y_2(x)} f(x,y)\,dy\)

这就是固定 x 时，整根竖细条的"截面"高度。

第 2 步（把每根竖条摞起来）：再沿 x 从 a 加到 b：

\(V=\displaystyle\int_a^b A(x)\,dx=\int_a^b\!dx\int_{y_1(x)}^{y_2(x)} f(x,y)\,dy\)

这一段叫累次积分（iterate integral），也叫"二次积分"。整个推导的关键在于"把一片切成条，再把条摞成体积"的几何直觉。

严格条件：如果 D 是"好的"区域（连通、无缝隙、边界不太病态），且 f 在 D 上连续，那累次积分 = 二重积分（这就是 Fubini 定理 的简化版）。考研范围内的区域都满足。

小白速记："先 y 后 x" = 「先看一根条里有多少东西」，「再把所有条加起来」。

📖 公式推理 2 · 为什么极坐标要再多乘一个 r？

核心问题：把 (x,y) 换成 \((r\cos\theta, r\sin\theta)\) 之后，为什么积分里要冒出来一个 r？

第 1 步（极坐标变换）： \(x=r\cos\theta,\ y=r\sin\theta\)。一对 \((r,\theta)\) 对应一个 \((x,y)\)。

第 2 步（算面积元）：在极坐标里取一小块：r 方向变 dr，θ 方向变 dθ。它的形状像一段弯曲的小矩形。

这一小块的「长」= \(r\,d\theta\)（沿弧线走的真实距离，半径 r × 弧度 dθ），「宽」= \(dr\)（沿半径的真实距离）。

所以面积 = 长 × 宽 = \((r\,d\theta)\cdot(dr)=r\,dr\,d\theta\)。

\(d\sigma = dx\,dy = r\,dr\,d\theta\)

第 3 步（代入积分）：

\(\displaystyle\iint_D f(x,y)\,dx\,dy=\int_\alpha^\beta\!d\theta\int_{r_1(\theta)}^{r_2(\theta)}\!f(r\cos\theta,r\sin\theta)\,r\,dr\)

关键陷阱：那个 r 是在 dr 前面的乘子，不是 f 里的 r。漏乘就直接扣 2 分。

小白速记：「极坐标必有 r，位置在 dr 的系数里」。

📖 公式推理 3 · 对称性为什么能省力（奇偶验证）

问题：一个 D 关于 y 轴对称，能不能只用右半边算？

分两块：把 D 拆成左右两块 \(D_+\) 和 \(D_-\)，这两块形状完全一样，只是 x 一个正一个负。

当 f 关于 x 是偶函数（f(-x,y)=f(x,y)）：

\(\displaystyle\iint_{D_+}\!f(x,y)\,d\sigma = \iint_{D_-}\!f(x,y)\,d\sigma\)（因为只是把 x 取负，值不变）

所以 \(\iint_D f = 2\iint_{D_+}f\)。

当 f 关于 x 是奇函数（f(-x,y)=-f(x,y)）：

\(\displaystyle\iint_{D_+}\!f(x,y)\,d\sigma = -\iint_{D_-}\!f(x,y)\,d\sigma\)

两块相加 = 0。这就是「奇零偶倍」的口诀。

对应表格：

函数关于 x 的性质D 关于 y 轴对称结果
偶✓\(2\iint_{右}f\)
奇✓\(0\)

对 y 轴同理。

📖 公式推理 4 · 格林公式的物理直觉（"沿边的流量 = 内部总散度"）

公式：

\(\displaystyle\oint_L P\,dx+Q\,dy=\iint_D\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)d\sigma\)

理解思路一：物理图像（流体场）

把 \((P,Q)\) 想成一个 2D 流体在每一点的速度向量，曲线积分 \(\oint P\,dx+Q\,dy\) 就是「沿这条闭合曲线走一圈，流体顺着我走的方向推进了多少」—— 它等于区域 D 的「流体净流出」总量。

而 \(\dfrac{\partial Q}{\partial x}-\dfrac{\partial P}{\partial y}\) 的本质是向量 \((P,Q)\) 的「旋度」的一种表示。格林公式告诉我们：

沿边界的总流量 = 内部每个小区域的旋度积分

理解思路二：拼贴法（小白版本）

  • 把 D 切成很多很多小矩形；每个矩形自己算"沿边的"贡献，等于"内部的"贡献
  • 把每个小矩形贡献加起来；内部的项相消，只剩下最外面一圈
  • 最外面那一圈恰好就是曲线积分 \(\oint_L\)

几何意义（旋度）： \(\dfrac{\partial Q}{\partial x}-\dfrac{\partial P}{\partial y}\) 在某点 > 0 表示向量场 \((P,Q)\) 在那个点呈「逆时针旋转」的倾向；= 0 表示「无旋」；< 0 表示「顺时针旋转」。

所以格林公式说：「沿边界绕一圈的总功」=「内部每个小区域旋转倾向的总和」。

小白速记：「积外变积内」—— 把难算的曲线积分，用格林公式化成更好算的二重积分。

📝 例题：从基础到综合

📝 例题 1（直角）· 矩形区域上的简单积分
计算 \(\displaystyle\iint_D (x+y)\,d\sigma\)，其中 D = \{0 ≤ x ≤ 1,\ 0 ≤ y ≤ 1\}。

🎯 考点：矩形区域 + 多项式被积函数 = 直角坐标最快

🧭 第一步：定积分次序

先 y 后 x 最直观——每根「竖细条」都是 y 从 0 到 1。

🧭 第二步：先对 y 积分

\(\displaystyle\int_0^1 (x+y)\,dy = \left[xy+\dfrac{y^2}{2}\right]_0^1 = x+\dfrac{1}{2}\)

🧭 第三步：再对 x 积分

\(\displaystyle\int_0^1 \left(x+\dfrac{1}{2}\right)dx = \left[\dfrac{x^2}{2}+\dfrac{x}{2}\right]_0^1 = \dfrac{1}{2}+\dfrac{1}{2}=1\)
小白记忆类比：算这个就像夹汉堡——先把每片肉 (y) 叠上去，再把整条面包 (x) 拍扁。
🎯 出题人视角：这是入门题，真正考的是「累次积分的基本动作」。别颠倒上下限。
📝 例题 2（极坐标）· 圆盘上的 x² + y²
计算 \(\displaystyle\iint_D (x^2+y^2)\,d\sigma\)，其中 D 是圆 \(x^2+y^2\le 1\)。

🎯 考点：识别"圆形区域 + x²+y²" = 极坐标必杀

🧭 第 1 步：判坐标系

圆 + 含 x²+y² → 极坐标。三件套：

  • \(x^2+y^2=r^2\)
  • \(d\sigma=r\,dr\,d\theta\)
  • 边界：从 r=0 到 r=1，θ 从 0 到 2π

🧭 第 2 步：套公式

\(\displaystyle\iint_D (x^2+y^2)\,d\sigma = \int_0^{2\pi}\!d\theta\int_0^1 r^2\cdot r\,dr = \int_0^{2\pi}\!d\theta\cdot\int_0^1 r^3\,dr\)

🧭 第 3 步：算两次

\(\displaystyle\int_0^{2\pi}d\theta = 2\pi,\quad \int_0^1 r^3\,dr = \dfrac{1}{4}\)

🧭 第 4 步：相乘

\(\displaystyle\iint_D (x^2+y^2)\,d\sigma = 2\pi\cdot\dfrac{1}{4}=\dfrac{\pi}{2}\)
小白记忆类比：圆盘上积 x²+y² 就像给一个煎饼"测总质量"——每个圈上 r² 都一样，先求一根圈的质量，再把所有圈叠起来。
🎯 出题人视角：极坐标题最易错的不是积分，而是忘了那一个 r（或写错位置）。
📝 例题 3（换序）· 把"先 y 后 x"换成"先 x 后 y"
计算 \(\displaystyle\int_0^1\!dx\int_x^1 e^{y^2}\,dy\)。提示：先 y 后 x 积分不出，需要换序。

🎯 考点：遇到 \(\int e^{y^2}\,dy\) 这种"原函数不存在"的，立刻想换序

🧭 第 1 步：画区域

积分区域 D：x 从 0 到 1，y 从 x 到 1。也就是「直线 y=x 上方、y=1 下方、x=0 右方」的那个三角形。

\(D: 0 \le y \le 1,\ 0 \le x \le y\)

🧭 第 2 步：改写为「先 x 后 y」

先 y 后 x 的话，每个 x 对应 y ∈ [x, 1]；改写成先 x 后 y：每个 y 对应 x ∈ [0, y]。

🧭 第 3 步：套公式

\(\displaystyle\int_0^1\!dx\int_x^1 e^{y^2}\,dy = \int_0^1\!dy\int_0^y e^{y^2}\,dx = \int_0^1 ye^{y^2}\,dy\)

🧭 第 4 步：算最后一步

令 \(u=y^2\)，\(du=2y\,dy\)，原式 = \(\dfrac{1}{2}\int_0^1 e^u\,du = \dfrac{1}{2}(e-1)\)。

小白记忆类比：换序就像把"刷漆"从「先刷每根柱子的左右」改成「先刷每一行的前后」——两件事做完后墙的颜色一样。
🎯 出题人视角：换序题关键有两点：① 正确画区域；② 上下限换错 0 分。
📝 例题 4（对称性）· 奇函数在对称区域上为 0
计算 \(\displaystyle\iint_D x\,y^2\,d\sigma\)，其中 D 由 \(x=-1, x=1, y=0, y=1\) 围成（一个矩形）。

🎯 考点：识别对称性 → 0 分拿满

🧭 第 1 步：看区域

D 是 \(-1 \le x \le 1,\ 0 \le y \le 1\)。关于 y 轴对称（x 取正负都有）。

🧭 第 2 步：看被积函数

\(f(x,y) = xy^2\)。关于 x 是奇函数（因为 x 换成 -x，函数变号）。

🧭 第 3 步：下结论

"奇函数在关于 y 轴对称的区域上 = 0"。不用算，直接结束。

🎯 最终答案：\(\iint_D xy^2\,d\sigma = 0\)。
小白记忆类比：左右对称的图形上，奇函数就像一只"会抵消"的队伍——左边一群人朝东冲，右边同样多的人朝西冲，总数 0。
🎯 出题人视角：这种题就是送分。识别出对称性后，1 行解决；不识别，硬算 5 分钟还在那积分。
📖 小白扫盲：三大类多元积分（一表分清考什么）

🎯 二重积分、三重积分、曲线/曲面积分 — 区别在"在哪积"和"积什么"

积分类型积分区域被积对象计算方法标记
二重积分 平面区域 \(D\) 二元函数 \(f(x,y)\) 直角坐标 / 极坐标
先定限再求积		 \(\iint_D f(x,y)\,d\sigma\)
三重积分 空间区域 \(\Omega\) 三元函数 \(f(x,y,z)\) 直角坐标 / 柱坐标 / 球坐标 \(\iiint_\Omega f(x,y,z)\,dV\)
第一类曲线积分
（对弧长）		 平面曲线 \(L\) \(f(x,y)\) 参数化：\(\int_L f\,ds = \int_a^b f(\phi(t))\,|\phi'(t)|\,dt\) \(\int_L f(x,y)\,ds\)
第二类曲线积分
（对坐标）		 有向曲线 \(L\) \(P\,dx + Q\,dy\) 参数化 / 格林公式 \(\int_L P\,dx + Q\,dy\)
第一类曲面积分
（对面积）		 曲面 \(S\) \(f(x,y,z)\) 投影法：将曲面投影到坐标平面 \(\iint_S f(x,y,z)\,dS\)
第二类曲面积分
（对坐标）		 有向曲面 \(S\) \(P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy\) 高斯公式 / 投影法 \(\iint_S \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S}\)

🧠 三大公式的关系

  • 格林公式： 第二类曲线积分 ↔ 平面二重积分 — \(\displaystyle\oint_L P\,dx+Q\,dy = \iint_D (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})\,d\sigma\)
  • 高斯公式： 第二类曲面积分 ↔ 三重积分 — \(\displaystyle\oiint_S \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S} = \iiint_\Omega \nabla\cdot\boldsymbol{F}\,dV\)
  • 斯托克斯公式： 空间曲线积分 ↔ 曲面积分 — \(\displaystyle\oint_L \boldsymbol{F}\cdot\mathrm{d}\boldsymbol{r} = \iint_S \nabla\times\boldsymbol{F}\cdot\mathrm{d}\boldsymbol{S}\)
  • 一句话：格林 + 高斯 + 斯托克斯 = 由简到繁的「降维」工具
⚠️ 第一类 vs 第二类（曲线/曲面）的区分
第一类（对弧长/对面积）：被积函数是数值，结果不依赖方向，物理意义是质量/面积
第二类（对坐标）：被积函数是向量，结果依赖方向，物理意义是功/通量
判断方法：看 \(d\sigma\) 或 \(ds\) → 第一类；看 \(dx\) 或 \(d\boldsymbol{S}\) → 第二类
看图定坐标系（二选一）
① 见圆或圆弧 → 直接极坐标
看区域：是不是有圆、圆弧、扇形、心脏线
看被积函数：有没有 \(x^2+y^2\)、\(\dfrac{y}{x}\)、\(\arctan(y/x)\) 这类难直接积的

② 否则 → 直角坐标
看区域：矩形 / 三角 / 梯形 / 由函数曲线围成
看被积函数：多项式 / 简单三角 / 指数（不含 \(x^2+y^2\)）

实战三连问：
① 区域轴对称吗？能用对称性吗？
② 区域能不能"先 y 后 x"清晰定限？不能则换序
③ 被积函数有没有"积分不出来"的部分？考虑分区域或换系
5 种武器 · 一题一用
① 直角坐标 · 先 y 后 x：最常见的入门套路
② 直角坐标 · 先 x 后 y：当被积函数难对 y 积、对 x 好积时
③ 极坐标：圆 / 圆弧 / 含 \(x^2+y^2\) 时
④ 对称性：奇零偶倍，省一半时间
⑤ 分区域积分：区域不连通或被积函数分段时拆开算
小白十大易错（错一个扣 2~3 分）
  • ❌ 极坐标 忘了那个 r（写错位置最常扣分）
  • ❌ 上下限写反（大的在右下，小的在左上）
  • ❌ 对称性用反（奇函数当成偶函数处理）
  • ❌ 换序忘画图（直接写错上下限）
  • ❌ 区域拆错（一半漏算或重算）
  • ❌ 极坐标限 θ 和 r 写错（角从 0 到 2π 还是 0 到 π/2）
  • ❌ 内层积分上限是函数（如 \(y_2(x) = 1-x^2\)）时被忽略
  • ❌ 用了极坐标但函数没换成 \((r\cos\theta, r\sin\theta)\)
  • ❌ 单位或量纲写错（如面积单位应是 m²，写成 m）
  • ❌ 最后不写结果，或忘了带单位
本章应试策略（卷面四步）
① 画图（30 秒）：草图中标出区域的边界、交点，确定对称性
② 选坐标系（10 秒）：圆 / 圆弧 / 含 \(x^2+y^2\) → 极坐标；否则直角坐标
③ 定上下限（30 秒）：先 y 后 x 还是先 x 后 y，写清上下限
④ 算两次一元积分（2~3 分钟）：先内层后外层，每一步标清楚
⑤ 写最终答案：数值 + 单位 + 必要的判定

📝 真题 1 | 2017 数一 | 解答题 · 10分
计算 \(\displaystyle\int_0^1\!dy\int_{\sqrt{y}}^1 e^{\frac{x^2}{2}}\,dx\)。提示：原积分次序算不出，需要换序。

🎯 出题人考察点

① 二重积分换序；② 区域画图；③ 对称性积分技巧。这是高频考点！

🧭 详细解析（小白逐步版）

第 1 步：识别不可积信号

看到 \(e^{x^2/2}\) 对 x 积分——这个函数原函数不是初等函数，所以要先 y 后 x 走不通。

第 2 步：画出区域 D

原式是「先 y 后 x 还是先 x 后 y」？看 dxdy 的顺序，写在前面的积分变量是内层，写在后面的外层。原题 \(\int dy\int dx\) = 「先 x 后 y」。

区域描述：y 从 0 到 1，x 从 \(\sqrt{y}\) 到 1。

把 \(\sqrt{y}\le x\le 1\) 改成 y 的范围：\(\sqrt{y}\le x \Rightarrow y\le x^2\)。再与 \(0\le y\le 1\) 取交集，得到 D：

\(D: 0 \le x \le 1,\ 0 \le y \le x^2\)

第 3 步：改写积分（先 y 后 x）

\(\displaystyle\int_0^1\!dx\int_0^{x^2} e^{x^2/2}\,dy\)

第 4 步：先对 y 积分（y 直接积）

\(\displaystyle\int_0^{x^2} e^{x^2/2}\,dy = x^2 e^{x^2/2}\)

第 5 步：再对 x 积分

观察到 \(\dfrac{d}{dx}(e^{x^2/2})=x e^{x^2/2}\)，而现在有 \(x^2 e^{x^2/2}=x\cdot (xe^{x^2/2})\)。

用分部积分（u = x, dv = x e^{x^2/2} dx）：

\(\displaystyle\int_0^1 x^2 e^{x^2/2}\,dx = \left[x e^{x^2/2}\right]_0^1 - \int_0^1 e^{x^2/2}\,dx = e^{1/2} - \int_0^1 e^{x^2/2}\,dx\)

最后那个 \(\int_0^1 e^{x^2/2}\,dx\) 不是初等积分，留作答案或近似。

🎯 最终答案：\(\displaystyle\int_0^1 x^2 e^{x^2/2}\,dx = e^{1/2} - \int_0^1 e^{x^2/2}\,dx\)

💯 得分点（阅卷老师按点给分）

① 正确识别不可积 (2分)；② 画图定区域 (3分)；③ 写出交换后的积分 (3分)；④ 计算结果 (2分)。

⚠️ 小白三大踩坑

⚠️ 误判内层外层：原题 \(\int dy \int dx\) 是「先 x 后 y」，不是「先 y 后 x」；

⚠️ 换序上下限写反：内层 [0, x²] 不要写成 [x², 0]；

⚠️ 对 e^{x²} 绝望：原函数不存在，告诉你只能分部积分或夹逼。

🔗 知识点桥接

这是一道标准的"换序题"，在 ch03 的反常积分里也出现过"积不出的函数"。掌握了"识别不可积 → 改换积分变量"思路，二者就通了。

📝 真题 2 | 2020 数一 | 解答题 · 12分
计算 \(\displaystyle\iint\limits_{D}\dfrac{x^{2}}{y^{2}}\,d\sigma\)，其中 D 由 \(y=x, y=2, x=1\) 围成。

🎯 出题人考察点（高频考点）

① 积分次序选择；② 积分区域画图；③ 累次积分计算。

🧭 详细解析（小白逐步版）

第 1 步：画图

三条线 \(y=x, y=2, x=1\) 围成一个上宽下窄的梯形，范围：x ∈ [1, 2]，y ∈ [x, 2]。

第 2 步：选择积分次序

被积函数 \(\frac{x^2}{y^2}\) 含 \(1/y^2\)。如果先 y 积分，\(\int 1/y^2\,dy = -1/y\)，很干净。所以先 y 后 x。

第 3 步：化为累次积分

\(\displaystyle I = \int_1^2\!dx\int_x^2 \dfrac{x^2}{y^2}\,dy\)

第 4 步：内层积分（对 y 积分时 x 当常数）

\(\displaystyle\int_x^2 \dfrac{x^2}{y^2}\,dy = x^2\cdot\left[-\dfrac{1}{y}\right]_x^2 = x^2\left(-\dfrac{1}{2}+\dfrac{1}{x}\right) = x-\dfrac{x^2}{2}\)

第 5 步：外层积分

\(\displaystyle I = \int_1^2 \left(x-\dfrac{x^2}{2}\right)dx = \left[\dfrac{x^2}{2}-\dfrac{x^3}{6}\right]_1^2 = \left(2-\dfrac{8}{6}\right)-\left(\dfrac{1}{2}-\dfrac{1}{6}\right) = \dfrac{2}{3}-\dfrac{1}{3}=\dfrac{1}{3}\)
🎯 最终答案：\(I = \dfrac{1}{3}\)

💯 得分点（阅卷老师按点给分）

① 画图确定区域 (3分)；② 化为累次积分 (3分)；③ 内层积分 (3分)；④ 外层积分 (3分)。

⚠️ 小白踩坑预警

⚠️ 上下限搞反；⚠️ 先 y 后 x 还是先 x 后 y 选错；⚠️ 被积函数在 y 内容易算错（特别是 \(-1/y\) 的符号）。

🔗 知识点桥接

这种"先 y 后 x"的基础题是每年必考的套路，掌握它基本就能拿到 6~8 分。

📝 真题 3 | 2019 数一 | 解答题 · 12分
计算 \(\displaystyle\iint\limits_{D} (x^2+y^2)\,d\sigma\)，其中 D 由 \(x^2+y^2=2x\) 围成。

🎯 出题人考察点（极坐标经典）

① 圆方程化为极坐标；② 极坐标下定积分限；③ 别忘了那个 r。

🧭 详细解析（小白逐步版）

第 1 步：把圆方程转极坐标

原方程 \(x^2+y^2=2x\)，代 \(x^2+y^2=r^2, x=r\cos\theta\)：

\(r^2 = 2r\cos\theta \Rightarrow r = 2\cos\theta\)

这是一个圆心在 (1,0)、半径为 1 的圆。

第 2 步：定 θ 的范围

r ≥ 0 意味着 \(\cos\theta \ge 0\)，即 \(\theta \in [-\pi/2, \pi/2]\)。

第 3 步：化为极坐标累次积分

\(\displaystyle I = \int_{-\pi/2}^{\pi/2}\!d\theta\int_0^{2\cos\theta} r^2\cdot r\,dr = \int_{-\pi/2}^{\pi/2}\!d\theta\int_0^{2\cos\theta} r^3\,dr\)

第 4 步：内层积分（对 r）

\(\displaystyle\int_0^{2\cos\theta} r^3\,dr = \left[\dfrac{r^4}{4}\right]_0^{2\cos\theta} = \dfrac{(2\cos\theta)^4}{4} = 4\cos^4\theta\)

第 5 步：外层积分

用对称性 \(\cos^4\theta\) 在 \([-\pi/2, \pi/2]\) 上是偶函数，只算 \([0, \pi/2]\) 乘 2：

\(\displaystyle I = 2\int_0^{\pi/2} 4\cos^4\theta\,d\theta = 8\int_0^{\pi/2}\cos^4\theta\,d\theta\)

用降幂公式 \(\cos^4\theta = \dfrac{3+4\cos 2\theta+\cos 4\theta}{8}\)：

\(\displaystyle\int_0^{\pi/2}\cos^4\theta\,d\theta = \dfrac{3}{8}\cdot\dfrac{\pi}{2} = \dfrac{3\pi}{16}\)

所以

\(I = 8\cdot\dfrac{3\pi}{16}=\dfrac{3\pi}{2}\)
🎯 最终答案：\(I = \dfrac{3\pi}{2}\)

💯 得分点

① 极坐标转换 (3分)；② 定积分限 (4分)；③ 内层积分 (2分)；④ 外层积分 (3分)。

⚠️ 小白踩坑

⚠️ 漏乘那个 r 直接扣 2 分；⚠️ θ 的范围算反（圆心不在原点）；⚠️ 降幂公式用错。

🔗 知识点桥接

这种"圆方程转极坐标后定 r"是 ch06 的进阶套路。会了这个，"圆环、扇形、心脏线"都通用。

📝 真题 4 | 2022 数一 | 解答题 · 12分
计算 \(\displaystyle\iint\limits_{D} |x-y|\,d\sigma\)，其中 D：\(0\le x\le 1, 0\le y\le 1\)。

🎯 出题人考察点（绝对值 + 拆分）

① 绝对值拆分；② 用 \(y=x\) 把方形分成两个三角；③ 分段积分。

🧭 详细解析

第 1 步：识别不可积信号

\(|x-y|\) 含绝对值，要先去掉再积。把方块分成两个三角形：\(D_1: x\ge y\) 和 \(D_2: y>x\)。

第 2 步：去掉绝对值

\(|x-y| = \begin{cases}x-y, & x\ge y\\ y-x, & y>x\end{cases}\)

第 3 步：分两区积分（利用轮换对称性）

由于方块 D 关于直线 y=x 对称，把 \(D_1\) 的结果算两遍即可：

\(\displaystyle I = 2\iint_{D_1}(x-y)\,d\sigma = 2\int_0^1\!dx\int_0^x (x-y)\,dy\)

第 4 步：内层积分

\(\displaystyle\int_0^x (x-y)\,dy = \left[xy-\dfrac{y^2}{2}\right]_0^x = x^2 - \dfrac{x^2}{2}=\dfrac{x^2}{2}\)

第 5 步：外层积分

\(\displaystyle I = 2\int_0^1 \dfrac{x^2}{2}\,dx = \int_0^1 x^2\,dx = \dfrac{1}{3}\)
🎯 最终答案：\(I = \dfrac{1}{3}\)

💯 得分点

① 识别绝对值 (2分)；② 区域拆分 (3分)；③ 分段积分 (4分)；④ 用对称性简化 (3分)。

⚠️ 小白踩坑

⚠️ 忘了用对称性，左右两边算两遍（浪费时间）；⚠️ 绝对值没拆直接积分（极常见错误）；⚠️ 上下限写错：内层 y 从 0 到 x（不是从 x 到 0）。

🔗 知识点桥接

这是一道"绝对值 + 对称性 + 分区域"的综合题，整合了 ch06 的三件套：识别不可积、画图拆分、用对称性。
