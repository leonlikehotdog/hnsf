🎯 核心考点

  • 期望、方差、协方差、相关系数的计算
📖 必背：期望和方差的性质

期望性质：

  • E(C) = C
  • E(aX + b) = aE(X) + b
  • E(X ± Y) = E(X) ± E(Y)
  • X,Y 独立时 E(XY) = E(X)E(Y)

方差性质：

  • D(C) = 0
  • D(aX + b) = a²D(X)
  • D(X) = E(X²) - [E(X)]²
  • X,Y 独立时 D(X ± Y) = D(X) + D(Y)
📝 例题
设随机变量 X 的分布律为 P(X=0)=0.2，P(X=1)=0.3，P(X=2)=0.5，求 E(X) 和 D(X)
E(X) = 0×0.2 + 1×0.3 + 2×0.5 = 0 + 0.3 + 1.0 = 1.3
E(X²) = 0²×0.2 + 1²×0.3 + 2²×0.5 = 0 + 0.3 + 2.0 = 2.3
D(X) = E(X²) - [E(X)]² = 2.3 - 1.69 = 0.61

📝 真题 1 | 2021 数一 | 解答题 · 10分
设 \(X\) 服从参数为 1 的指数分布，\(Y=X^{2}\)，求 \(EY\) 和 \(\text{Cov}(X,Y)\)。

🎯 出题人考察点（这是高频考点！）

① 指数分布；② Gamma 函数；③ 协方差计算

🧭 详细解析（小白逐步版）

\(X\sim E(1)\)，\(f(x)=e^{-x},x>0\)。\(EY=E(X^2)=\int_0^\infty x^2 e^{-x}\,dx=2!=2\)。\(E(XY)=E(X^3)=3!=6\)。\(\text{Cov}(X,Y)=E(XY)-E(X)E(Y)=6-1\cdot 2=4\)。

💯 得分点（阅卷老师按点给分）

① 写出 X 分布 (2分)；② EY (3分)；③ EXY (3分)；④ 协方差 (2分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ \(\int x^n e^{-x}dx=n!\) 记错；⚠️ Cov 公式记反
