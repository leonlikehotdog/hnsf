🎯 核心考点

  • 一阶线性微分方程
  • 二阶常系数线性微分方程
📖 必背：一阶线性微分方程通解公式

标准形式：y' + P(x)y = Q(x)

y = e-∫Pdx[ ∫Q·e∫Pdxdx + C ]
📖 二阶常系数齐次微分方程

y'' + py' + qy = 0，写特征方程 r² + pr + q = 0：

  • 两个不等实根 r₁ ≠ r₂：y = C₁er₁x + C₂er₂x
  • 重根 r₁ = r₂ = r：y = (C₁ + C₂x)erx
  • 共轭复根 r = α ± βi：y = eαx(C₁cos βx + C₂sin βx)
📝 例题
求微分方程 y' + 2y = ex 的通解
这是一阶线性微分方程，P(x) = 2，Q(x) = ex。
∫Pdx = 2x，e∫Pdx = e2x，e-∫Pdx = e-2x。
y = e-2x[∫ex·e2xdx + C] = e-2x[∫e3xdx + C] = e-2x[e3x/3 + C] = ex/3 + Ce-2x
