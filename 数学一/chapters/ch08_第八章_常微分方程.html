🎬 什么是微分方程？你能在 30 秒内意会

你学了导数（ch02）：知道 y 就能算 y'

你学了积分（ch03）：知道 y' 就能反推 y

但物理学家扔给你一个问题：

"我不知道 y，但我知道 y 和 y' 的关系。你能反推 y 是什么吗？"

这就是微分方程：把"函数 + 它的导数"混在一起的方程。世界是个变化的世界，所以世界几乎所有规律都是微分方程。

⏳ 350 年常微分方程史 · 让宇宙"会算"
📖 牛顿第二定律 F = ma
1687 牛顿："给我力，我能算一切"

🎯它是为了解决什么问题？

1665 年伦敦大瘟疫，牛顿躲到乡下。看着苹果落地，他在想一个只有 1 岁的孩子才能问的"笨"问题：

"为什么苹果往下掉而不往上飞？"

其实更复杂：苹果从树上掉下来，下落速度越来越快 —— 这是"加速度"。加速度来自力。这条链子就是：

F = m · a ↔ F = m · a'' ↔ m·d²x/dt² = F

1687 年牛顿在《自然哲学的数学原理》(Principia) 中把这个方程写出来，并用它解释了行星运动、潮汐、苹果落地三大看似无关的现象。

"我不用假设 — 我用方程式能告诉你月球为什么绕地球转、苹果为什么落。" —— 牛顿《原理》1687

📱今天有什么用？

① 手机计步：加速度传感器每秒钟测三轴的 a 变化，用 F = ma 反推你走了多少步

② 游戏人物移动：跳跃曲线 = 二次积分 m·d²y/dt² = -mg

③ SpaceX 火箭回收：实时求解 6 自由度的牛顿方程，让火箭精确落地

📖 指数增长 dy/dt = ky
1690 雅各布·伯努利："人口 = e 的人间化身"

🎯它是为了解决什么问题？

1690 年巴黎在闹瘟疫，法国主教问数学家："巴黎人口会无限制增长吗？"

雅各布·伯努利的回答是：会（在没有资源限制的情况下）。他写下：

dy/dt = k·y → y(t) = y₀·e^(kt)

这就是指数增长方程 —— 跟你在 ch01 看到的复利问题是一回事（连续利滚利 + 指数增长 = 同一件事）。

"如果人口不设上限，100 年后人类都要把地球挤爆。" —— 雅各布（有点夸张，但意思到了）

📱今天有什么用？

① 疫情预测：每天感染人数的指数增长期，用这个微分方程拟合

② 放射性碳-14 测年：m(t) = m₀·e^(-λt)，λ 是衰变常数

③ 贷款复利：你信用卡欠款按月结，一年后你欠的就是本金 × e^(年利率)

📖 弹簧振动 y'' = -ω²y
1740 欧拉："吊灯的摆动是 sin 函数"

🎯它是为了解决什么问题？

伽利略年轻时，注意到礼拜堂里的吊灯总是左右晃一下又回来。他敏锐地提出：摆动周期只跟"绳长 + 重力加速度"有关，跟"摆幅多大"无关（这就是"等时性"）。

170 年后，欧拉把这个直觉变成方程：

m·y'' + k·y = 0 → y = C·cos(√(k/m)·t + φ)

解完后的形式竟然是个三角函数。这解释了为什么"单摆永远以同样的频率摆" —— 即便把它晃得很厉害，周期还是几乎不变（小幅时）。

"几乎所有的'周期性'自然界，都是这个方程的近亲。" —— 欧拉

📱今天有什么用？

① 机械手表：里面的摆轮 = 弹簧振动方程 y'' = -ω²y

② 电子电路：LC 振荡电路 = 数学形式一样的方程

③ 地震预警：建筑物振动分析 = 高阶版弹簧方程

📖 一阶线性微分方程
1700s 莱布尼茨：「加法扰动也能解」

🎯它是为了解决什么问题？

最朴素的微分方程：y' + P(x)y = Q(x)。这是"y 的增长率 = 某个外力 Q - 某个内部阻力 P·y"。

莱布尼茨给出一个"长但万能"的解法：

y = e^(−∫P dx) · [∫Q·e^(∫P dx) dx + C]

本质上是两步法：① 乘上一个"积分因子"让左边凑成 (y·I)' 形式；② 直接积分。

"这个公式像瑞士军刀 — 看上去复杂，但任何一阶线性方程都能用它拆。" —— 莱布尼茨

📱今天有什么用？

① RLC 电路：电流随时间的变化 = 经典一阶线性方程

② 人口增长 + 食物限制：y' = a·y − b·y² 看起来非线，但 +− 可以分开算

📖 二阶常系数 + 特征方程
1740 欧拉的"代数替代微分"绝技

🎯它是为了解决什么问题？

想解 y'' + p·y' + q·y = 0 这种二阶方程。但求导操作太麻烦。

欧拉的绝招：假设解就是 e^(rx)。代进去，求导 → e^(rx) 自动求导就是 r·e^(rx)。

代入得：e^(rx) · (r² + p·r + q) = 0 → 特征方程：r² + p·r + q = 0

微分方程竟然变成了代数方程 ！这就是特征方程法。然后根据根的 3 种情况（两实根 / 重根 / 共轭复根），写出 3 种不同的解。

"把'微分'替换成'代数' — 这才是最深的数学把戏。" —— 欧拉

📱今天有什么用？几乎所有"会晃的东西"（桥梁、机械臂、芯片减震器）都靠这个方程做稳定性分析。

🎯 核心考点

  • 一阶线性微分方程
  • 二阶常系数线性微分方程
📖 必背：一阶线性微分方程通解公式

标准形式：y' + P(x)y = Q(x)

y = e-∫Pdx[ ∫Q·e∫Pdxdx + C ]
📖 二阶常系数齐次微分方程

y'' + py' + qy = 0，写特征方程 r² + pr + q = 0：

  • 两个不等实根 r₁ ≠ r₂：y = C₁er₁x + C₂er₂x
  • 重根 r₁ = r₂ = r：y = (C₁ + C₂x)erx
  • 共轭复根 r = α ± βi：y = eαx(C₁cos βx + C₂sin βx)
📝 例题
求微分方程 y' + 2y = ex 的通解
这是一阶线性微分方程，P(x) = 2，Q(x) = ex。
∫Pdx = 2x，e∫Pdx = e2x，e-∫Pdx = e-2x。
y = e-2x[∫ex·e2xdx + C] = e-2x[∫e3xdx + C] = e-2x[e3x/3 + C] = ex/3 + Ce-2x

📝 真题 1 | 2020 数一/二 | 解答题 · 12分
求微分方程 \(y\prime\prime+2y\prime+y=xe^{x}\) 的通解。

🎯 出题人考察点（这是高频考点！）

① 二阶常系数非齐次线性方程求解；② 重根情形；③ 特解形式判断

🧭 详细解析（小白逐步版）

**齐次解**：特征方程 \(r^2+2r+1=0\)，\(r=-1\)（重根），\(y_h=(C_1+C_2 x)e^{-x}\)。**特解**：\(f(x)=xe^x\)，\(\lambda=1\) 不是特征根，设 \(y^*=e^x(Ax+B)\)。代入解得 \(A=1/4, B=-1/4\)。\(y^*=\dfrac{1}{4}(x-1)e^x\)。通解 \(y=(C_1+C_2 x)e^{-x}+\dfrac{1}{4}(x-1)e^x\)。

💯 得分点（阅卷老师按点给分）

① 齐次解 (4分)；② 特解形式 (2分)；③ 代入求系数 (4分)；④ 通解 (2分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 重根时齐次解忘了 \(x\) 因子；⚠️ 特解形式判断错（\(\lambda\) 与特征根的关系）
