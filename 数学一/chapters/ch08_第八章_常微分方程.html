🎯 核心考点

  • 一阶线性微分方程
  • 二阶常系数线性微分方程
📖 必背：一阶线性微分方程通解公式

标准形式：y' + P(x)y = Q(x)

y = e-∫Pdx[ ∫Q·e∫Pdxdx + C ]
📖 二阶常系数齐次微分方程

y'' + py' + qy = 0，写特征方程 r² + pr + q = 0：

  • 两个不等实根 r₁ ≠ r₂：y = C₁er₁x + C₂er₂x
  • 重根 r₁ = r₂ = r：y = (C₁ + C₂x)erx
  • 共轭复根 r = α ± βi：y = eαx(C₁cos βx + C₂sin βx)
📝 例题
求微分方程 y' + 2y = ex 的通解
这是一阶线性微分方程，P(x) = 2，Q(x) = ex。
∫Pdx = 2x，e∫Pdx = e2x，e-∫Pdx = e-2x。
y = e-2x[∫ex·e2xdx + C] = e-2x[∫e3xdx + C] = e-2x[e3x/3 + C] = ex/3 + Ce-2x

📝 真题 1 | 2020 数一/二 | 解答题 · 12分
求微分方程 \(y\prime\prime+2y\prime+y=xe^{x}\) 的通解。

🎯 出题人考察点（这是高频考点！）

① 二阶常系数非齐次线性方程求解；② 重根情形；③ 特解形式判断

🧭 详细解析（小白逐步版）

**齐次解**：特征方程 \(r^2+2r+1=0\)，\(r=-1\)（重根），\(y_h=(C_1+C_2 x)e^{-x}\)。**特解**：\(f(x)=xe^x\)，\(\lambda=1\) 不是特征根，设 \(y^*=e^x(Ax+B)\)。代入解得 \(A=1/4, B=-1/4\)。\(y^*=\dfrac{1}{4}(x-1)e^x\)。通解 \(y=(C_1+C_2 x)e^{-x}+\dfrac{1}{4}(x-1)e^x\)。

💯 得分点（阅卷老师按点给分）

① 齐次解 (4分)；② 特解形式 (2分)；③ 代入求系数 (4分)；④ 通解 (2分)

⚠️ 小白踩坑预警（错一个扣 2~4 分）

⚠️ 重根时齐次解忘了 \(x\) 因子；⚠️ 特解形式判断错（\(\lambda\) 与特征根的关系）
