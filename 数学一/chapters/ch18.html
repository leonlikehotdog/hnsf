🎯 核心考点

  • 期望、方差、协方差、相关系数的计算
📖 必背：期望和方差的性质

期望性质：

  • E(C) = C
  • E(aX + b) = aE(X) + b
  • E(X ± Y) = E(X) ± E(Y)
  • X,Y 独立时 E(XY) = E(X)E(Y)

方差性质：

  • D(C) = 0
  • D(aX + b) = a²D(X)
  • D(X) = E(X²) - [E(X)]²
  • X,Y 独立时 D(X ± Y) = D(X) + D(Y)
📝 例题
设随机变量 X 的分布律为 P(X=0)=0.2，P(X=1)=0.3，P(X=2)=0.5，求 E(X) 和 D(X)
E(X) = 0×0.2 + 1×0.3 + 2×0.5 = 0 + 0.3 + 1.0 = 1.3
E(X²) = 0²×0.2 + 1²×0.3 + 2²×0.5 = 0 + 0.3 + 2.0 = 2.3
D(X) = E(X²) - [E(X)]² = 2.3 - 1.69 = 0.61
