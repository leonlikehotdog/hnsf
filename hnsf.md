## 一、数学一「战略放弃清单」（极其重要！必看！）

数学一是你上岸最大的障碍，5个月薄弱基础不可能面面俱到。以下是**经过精确计算的扔/留方案**：

### 必须掌握的知识点（覆盖约90-100分，拿到80即可）
```
【高等数学】（约50分）
  ✓ 函数极限计算（洛必达、等价无穷小、泰勒公式前3项）—— 必考，基础
  ✓ 一元函数导数与微分（求导公式、切线法线、单调性极值）—— 必考，基础
  ✓ 不定积分与定积分计算（凑微分、分部积分、换元法）—— 必考，基础
  ✓ 定积分应用（面积、旋转体体积）—— 常考，计算量大但套路固定
  ✓ 多元函数微分学（偏导、全微分、无条件极值）—— 必考，基础

【线性代数】（约20分）
  ✓ 行列式计算（2阶/3阶/递推法）
  ✓ 矩阵运算（乘法、逆、秩）
  ✓ 线性方程组解的结构与求解
  ✓ 特征值与特征向量（求特征值、对角化判断）
  ✓ 二次型标准化（配方法、正交变换法）

【概率论】（约10分，能拿多少算多少）
  ✓ 随机事件与概率（古典概型、条件概率、全概率公式）—— 基础不牢也能快速上手
  ✓ 常见分布（0-1分布、二项分布、正态分布、均匀分布）
  ✓ 数字特征（期望、方差的计算公式）
```

### 建议完全放弃的知识点（投入产出比极低）
```
✗ 曲线积分与曲面积分（高数最难章节之一，考试最多1道选填，直接蒙C）
✗ 级数（敛散性判断+幂级数展开+傅里叶级数，题型复杂，放弃）
✗ 三重积分（计算量大且复杂，即使学会考场也大概率算错）
✗ 多元函数条件极值（拉格朗日乘数法可以掌握，但复杂应用题放弃）
✗ 证明题（数学一证明题通常极难，直接放弃，把时间留给计算题）
✗ 区间估计与假设检验（概率论最末章节，大概率不考，不浪费时间）
```

### 数学一真题目标分数拆解
```
选择题 10题 × 5分 = 50分 → 目标对 7题 = 35分 ← 只做简单题，不会就跳过
填空题 6题 × 5分 = 30分 → 目标对 4题 = 20分 ← 计算能力是关键
大题     6题 × 分值不等 → 目标只做前2道简单大题 ≈ 25分
                                 总分 ≈ 80分 ✓
```

## 二、408专业课「精确踩分指南」（你的核心拉分项）

> 408统考150分，数据结构45分 + 计算机组成原理45分 + 操作系统35分 + 计算机网络25分

### 数据结构（45分 — 目标拿下40分）
```
这是你的本行（软件工程背景），必须拿满。
✓ 线性表（顺序表+链表的基本操作，必考代码题）
✓ 栈与队列（出入栈、循环队列、表达式求值）
✓ 树与二叉树（遍历、线索二叉树、哈夫曼树、BST）
✓ 图（邻接矩阵/表、BFS/DFS、最小生成树、最短路径、拓扑排序）
✓ 排序与查找（7大排序对比、二分查找、哈希表）
```

### 计算机组成原理（45分 — 目标拿下30分）
```
这是408最难的部分，放弃深奥硬件内容，只抓高频考点。
✓ 数的表示与运算（原反补、溢出的判断、浮点数IEEE754）
✓ 存储器层次（Cache映射方式、主存扩展、虚拟存储器）
✓ 指令系统（寻址方式、指令格式、RISC/CISC对比）
✓ CPU结构（数据通路、流水线冒险与解决）—— 常考大题
✓ 总线与I/O（中断、DMA、通道方式）
✗ 放弃：微程序控制器细节、各类外设具体工作原理
```

### 操作系统（35分 — 目标拿下30分）
```
偏记忆性，性价比极高。
✓ 进程管理（PV操作、死锁判定与银行家算法）—— 必考大题
✓ 内存管理（分页/分段/段页式、页面置换算法）
✓ 文件系统（FCB、目录结构、磁盘调度算法）
✓ I/O管理（SPOOLing、缓冲技术、设备独立性）
✓ 操作系统概述（基本特征、分类）—— 用于选填
```

### 计算机网络（25分 — 目标拿下20分）
```
偏记忆性，性价比极高。
✓ 物理层（奈氏准则、香农定理）
✓ 数据链路层（CSMA/CD、交换机、VLAN）
✓ 网络层（IP协议、子网划分、路由算法、NAT）
✓ 传输层（TCP/UDP、三次握手、流量/拥塞控制）
✓ 应用层（DNS、HTTP、FTP、SMTP）—— 主要是记忆
```

## 三、碎片化时间利用方案（在职备考的核心打法）

> 在职备考最怕「等有整块时间再学」。本方案把学习拆成5分钟-30分钟-2小时三档，任何场景都能学。

### 碎片场景×时间档位对应表

| 时间段 | 可占用时间 | 适合做什么 | 工具/方法 |
|-------|-----------|-----------|----------|
| 起床洗漱+早餐 | **15min** | 听数学公式音频 / 政治带背音频 | 提前录好或找B站音频 |
| 通勤（地铁/公交） | **20-40min** | 用Anki刷单词 / 刷408选择题 | 手机App（墨墨背单词 + Anki） |
| 工作摸鱼空隙 | **5-10min** | 看一道错题 / 记一个碎片知识点 | 纸质小卡片或手机备忘录 |
| 午休前30分钟 | **20-30min** | 做10道408选择题 / 背英语作文模板句 | 手机刷题App（考研帮） |
| 下班通勤 | **20-40min** | 复盘早上背的单词 / 听408知识点串讲 | Anki复习 + B站视频 |
| 晚饭后-睡前 | **1.5-2h** | **核心整块学习时间** | 集中攻克数学/408难点 |
| 周末上午 | **3h** | 数学真题模考 or 408套卷 | 完整限时做题 |
| 周末下午 | **2h** | 英语真题 + 政治刷题 | 保持手感 |
| 周末晚上 | **1-2h** | 本周错题复盘 + 下周计划 | 整理笔记 |

### 每日时间预算（工作日）

```
07:00-07:15  起床洗漱 → 听数学公式音频
07:15-07:45  通勤 → Anki刷单词（30个新词+50个复习）
08:00-12:00  工作 → 如有摸鱼时间，看2-3道错题
12:00-12:30  午休 → 刷15道408选择题（考研帮App）
12:30-13:30  午睡 + 摸鱼处理
13:30-17:30  工作 → 间隙看知识点卡片
17:30-18:00  下班通勤 → 复习早上的单词
18:00-19:00  晚饭 + 休息
19:00-19:30  政治刷题 or 英语阅读1篇（20min）
19:30-21:30  数学（2h）—— 只做基础题
21:30-22:00  休息/洗澡
22:00-23:00  408专业课（1h）
23:00-23:30  复盘今天学了啥 + 定明天计划
23:30        睡觉

工作日总学习时长 ≈ 3.5-4h/天 ✓
```

### 周末时间预算

```
08:00-08:30  起床 + 复习本周错题
08:30-11:30  数学真题模考（限时3h）
11:30-13:00  午饭 + 午休

17:00-19:00  晚饭 + 休息
```

## 五、各题型的"投机"答题策略

### 数学一
| 题型 | 投机策略 |
|------|---------|
| 选填（80分） | 会做的认真算，不会的5秒内凭直觉选一个，绝不纠结。目标：拿满基础题，蒙对1-2道难题 |
| 大题前2道 | 务必完整作答，步骤写详细（步骤有分）。通常是极限计算+导数应用 |
| 大题中间2道 | 尝试第一问，第二问写个开头就放弃，不浪费30分钟 |
| 大题最后2道 | 看都不看，直接写个"解"字，省下时间检查选填 |

### 英语一
| 题型 | 投机策略 |
|------|---------|
| 阅读（40分） | 精做4篇，每篇20min。先看题干划关键词，再读文章找定位 |
| 作文（30分） | 背熟2套模板（积极话题+消极话题），考场上直接套。大作文25min，小作文15min |
| 新题型（10分） | 排序题花15min做，不会就按逻辑猜 |
| 翻译（10分） | 写满中文，字迹工整，句子不通顺也写，给同情分 |
| 完型（10分） | **直接全选A**，能得约3分。节省20分钟给阅读 |

### 408专业课
| 题型 | 投机策略 |
|------|---------|
| 选择题（80分） | 40题×2分。数据结构部分的题必须全对。计组难题靠排除法 |
| 大题（70分） | 数据结构算法题必写（伪代码也得分）。PV操作题必拿满分。计组大题写步骤 |

### 政治
| 题型 | 投机策略 |
|------|---------|
| 选择题（50分） | 肖八+肖四刷3遍，考试时不会的用排除法 |
| 大题（50分） | 肖四全部背熟。抄材料+写满+分点作答。字迹工整极重要 |
